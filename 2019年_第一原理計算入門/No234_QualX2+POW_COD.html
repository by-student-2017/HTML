■ QualX2 (Windows 10 (64 bit))
(http://www.ba.ic.cnr.it/softwareic/qualx/)

□ Install (+ Registration)
(http://www.ba.ic.cnr.it/softwareic/qualx/i)
1. Register now for free
  Public research institution? [check] Yes
  Terms & Conditions [check]
  REGISTER
  ※ 必要事項を記入していけばよい。非営利の機関や大学では無料で利用可能
2. QualX2
3. download here the latest version 
4. QualX2 2.24 64 bit version (22.11.2018) 
  WINDOWS 64 BIT
5. QualX2-221118_install64.exe
  (詳細情報 > 実行 > はい)
6. All [Next>] > [Install] > [Finish]
◇ POW_COD
3. POW_COD 
4. PowCod 1901
  POW_COD SQLITE3 DATABASE(FULL)
5. cod1901.zip を解凍
※ COD (+AMCSD)

□ database setting
1. QualX
2. Search > Database Indexation
3. New > POW_COD database file (e.g. C:\cod1901\cod1901.sq) > Open
4. (click) column (e.g. COD-1901)
5. Select POW_COD_0
6. Close

□ Usage
1. QualX
2. File > Import diffraction data > examples > example1.dat > Open
3. OK (Set wavelength radiation)
4. Search > Search-Match
5. (click) column (e.g B6 La) > [Accept selected phase](check mark on the right-hand side) 
  (click) column (e.g Si) > [Accept selected phase](check mark on the right-hand side) 
  (click) column (e.g Al2 O3) > [Accept selected phase](check mark on the right-hand side) 
  (In my case, B6 La 11.5%, Si 31.7%, Al2 O3 56.9%)
----------
◇ SPring-8, Poton factory, AichiSR, etc
3. select wavelength
  User Defined
  Lambda: (input wavelength [Å])
  OK
----------
◇ Normal XRD [D8 ADVANCE(Bruker), SmartLab (Rigaku), etc]
3. select wavelength
  Cu
  (Lambda: 1.540560 [Å])
  OK
----------

□ Usage (Restraints mode)
1. QualX
2. File > Import diffraction data > examples > example1.dat > Open
3. OK (Set wavelength radiation)
4. Search > Restraints
  Or
  Search with restraints
5. (click) column (e.g B6 La) > [Accept selected phase](check mark on the right-hand side) 
  (click) column (e.g Si) > [Accept selected phase](check mark on the right-hand side) 
  (click) column (e.g Al2 O3) > [Accept selected phase](check mark on the right-hand side) 
  (In my case, B6 La 11.5%, Si 31.7%, Al2 O3 56.9%)
6. (copy) CARD > COD (http://www.crystallography.net/cod/search.html) 
  > Search by COD ID: [(past (CARD)] > search > CIF

□ XRD input data
◇ type 1
  2theta intensity
◇ type 2
  1st line: start(2theta) step(2theta) end(2theta)
  2nd or later: intensity
(※ C:\Program Files\QualX\examples
  CPD2.dat, MIXT_5.dat, example1.dat)

□ asc2dat
(http://www.misasa.okayama-u.ac.jp/~masami/asc2dat.zip)
  *.asc > asc2dat.exe > *.dat

□ 2θ Zero Correction
(http://www.ba.ic.cnr.it/softwareic/qualx/data-reduction/2θ-zero-correction/)
※ 上手く動作しないときはこれを試してみると良いとのこと

※ [academic and non-profit research institutions]と非営利の機関も利用できるのでありがたい。
※ バックグラウンドやピークサーチは自動で行われる。データの質が悪くてノイズをピークと誤ってしまうような場合は、peak searchでピークの閾値を変えて対処できる
※ Restrain: 元素の制限
※ 複数の相が検出された場合は各相も計算される
※ card number = ID (COD)
※ POW_CODのアドレスの中に日本語フォントがあると上手く動作しない（私の場合）
