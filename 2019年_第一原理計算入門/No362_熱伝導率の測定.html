◆ レーザーフラッシュ法での測定のポイント
・標準試料と測定する試料の厚みを合わせる
・表面を平坦にする
　[熱拡散率 = 0.1388 * 厚み2 / ハーフタイム]であるため、厚み計測の不確かさが厚みの二乗で影響する
・レーザーのショットの回数は3から5回（標準偏差による評価ができる）
・昇温速度
　100℃以下は5 K/min
　100℃以上は10 K/min
　※ 温度制御でのオーバーシュートは防げない
・Ar流量は150 ml/min程度

◇ カーボンスプレー
・Graphit33 / Graphiteを利用の場合
　しわにならない程度に厚く塗布する（缶をよく振って、表2回+裏2回、塗布する）
・黒化剤420 mLを利用の場合
　ハーフタイムが短い（薄い試料など）場合はまばらに黒化するのが良い。黒化剤420mLを利用するとそれが行い易い。
※ 熱拡散率については、熱拡散率が小さかったり、試料の厚みが厚いものはカーボンスプレーによる結果の依存性は小さい
※ 900℃程度の高温でデータが悪くなるのはカーボンが揮発している為。Ar流量を増やして対処する。

◇ 黒化処理（カーボンスプレーを塗布）する理由
・照射面側
　パルス光を吸収して熱に変換するため
　パルス光の反射を防ぐため
　可視光および近赤外光の透過の防止
・検出器面側
　照射面側から伝わった熱の輻射率を上げるため

◇ 酸化を防ぎたい場合は下記のように行う
・Arでのフラッシング
　[真空引き10分 > Arを入れる > 栓を開ける > 栓を閉める]を2回行う
・Arの流量
　100 ~ 200 ml/min程度。多く流すほど酸化しにくい

◇ 計算のモデル
□ Cowanモデル
・試料上下からの熱損失を考慮に入れたモデル
・ハーフタイムと5*ハーフタイムの信号比を用いて熱損失の効果を評価
・樹脂、金属、セラミックスなど多様なサンプルの解析に使用可能
（最も頻繁に使用されるモデル）
  ※私の場合は[Cowan+puls correction]と[線形ベースライン補正]を使っている。「puls correction」は、入射するレーザーがパルス幅（光が入射したときから入射の終わりまでの時間）を持つので、それを補正するためのもの。この補正を加えておけば問題はない。
□ Cape-Lehman(Standard)モデル
・試料の上下面だけでなく、側面からの熱損失も考慮したモデルで最も精密なモデルとなる
・耐火材などといった厚く（2~3 mm）、サイズの大きなサンプルを高温で測定するときなどに使用
（300℃以下でそれ程厚くない試料の場合はCowanモデルを使用する場合が多い）
□ Radiation (Transparent)モデル
・試料内および照射面に塗布したカーボン層からの熱輻射による熱伝達の効果を考慮したモデル
・透光性のある試料では輻射による一部の熱が速く伝わるため、パルス照射後にステップ状のシグナル変化が観測される
（このようなスペクトルの場合、Cowanモデルで解析すると熱拡散率を実際より高めに評価してしまうことがある）
※ Cowan（コウアン）法とCape-Lehman（ケープリーマン）法は常温では同じ結果になる。全体としてもほぼ同じ結果。
※ 計算モデルは測定前に指定しているが、測定後でも計算モデルを変更できる。Resultsなどの画面を見て、フィットが1に近いものを採用する。

◇ 測定可否の判断
・パルス幅 =< ハーフタイム / 5 = (0.1388 * 厚み2 / 熱拡散率) / 5

◇ スペクトルが負の値から立ち上がる場合
・透光性がない試料の場合
　光が漏れているので、試料のセッティングを確認する
・透光性がある試料の場合
　試料両面に金属コート（Pt, Ag, Auなどを100 nm以上）をしてから黒化処理をする
　照射両面：パルス光の透過を防ぐため
　検出器面側：試料内部からの輻射を防ぎ、試料裏面からのみの温度上昇をみる
　金属コートが必要な試料：透明の試料、パルス光の波長範囲で透光性のある試料
　（Ndガラスレーザー： 1060 nm、Xeランプ：約200-2000 nm）

◇ フラッシュ法による比熱容量測定
・測定試料と比熱容量が既知の参照試料に、同じエネルギーのフラッシュ光を照射し、外挿温度上昇の比から比熱容量を算出する
・測定試料と参照試料で、加熱パルス光の吸収率が同一かつ放射温度波長における分光放射率が同一である必要がある
・（JIS R 1611:2010）温度応答を測定する面の放射率が等しいと仮定

◇理論
熱拡散率 α α = λ / (ρ*Cp) = d2 / τ
= 0.1388 * d2 / t1/2		 単位：m2 / s
熱伝導率 λ λ = α * ρ * Cp  単位: W / (m*K)
熱浸透率 ε ε = √(λ * ρ * Cp) 単位: J  / (m2*K) * s(-1/2)
体積比熱 ρ*Cp ρ*Cp 単位: J / (m3*K)
ここで、ρは密度、Cpは比熱、dは試料厚さ、τは熱拡散の特性時間、 t1/2はハーフタイム（パルスを入射してから最大の温度の1/2の温度になる時間）。

◆ サーモリフレクタンス法
・薄膜の熱拡散率の測定に用いられる
・（レーザー）フラッシュ法が赤外線検出器で試料背面の温度上昇を測定するのに対し、サーモリフレクタンス法では、温度が上昇すると反射率が変化することを利用
・基本原理はレーザーフラッシュ法と同じ

◇多層試料の評価（測定可否の判定方法）
・２層試料の樹脂層の熱拡散率を測定する場合を考える
1. 各サンプルの厚みと物性値を確認する
2. 各層の熱抵抗およびハーフタイムを求める
3. 各層の熱抵抗とハーフタイムの寄与率を求める
　各層の熱抵抗の寄与度 =< 20%（既知のものがか？）、各層のハーフタイムが10%以上
4. 測定後、熱拡散率から熱抵抗とハーフタイムの寄与率を確認する

・以下の3層の場合の全体の熱抵抗R
--------------------
1層目：λ1, d1
--------------------R12
2層目：λ2, d2
--------------------R23
3層目：λ3, d3
--------------------
・全体の熱抵抗R
　R = d1/λ1 + d2/λ2 + d3/λ3 + R12 + R23
・全体のみかけの熱伝導率λ
　λ = (d1 + d2 + d3) / (R1 + R2 + R3 + R12 + R23)

・未知層 + 既知層試料の接触熱抵抗の算出
　既知層の熱拡散率が未知の場合、未知層の厚みの異なる試料を用意し、未知層の厚みを変えた試料を複数用意して、未知層の厚みと熱抵抗を直線（熱抵抗 = 接触熱抵抗 + 厚み/熱伝導率）でプロットする。その切片（厚み=0）を正味の接触熱抵抗とし、傾きを[1/正味の熱伝導率]とする。

◆ トラブルシューティング（LFA 457 MicroFlas）

◇ スペクトルの強度が弱いとき
・原因となる理由：測定している試料に対して温度が高すぎるために、測定している試料から金属蒸気が発生して窓を汚している可能性がある。DSCやTG-DTA、文献などから金属蒸気が発生しない試料の温度を調べて、測定する温度の上限を決めるべきである。
・検出器側の窓（Ge）やレーザーの入射側の窓が汚れている可能性がある。
・対処1：検出器側の窓は（エタノールで湿らせた綿棒で軽く浮かせて拭く｛特に中心部分にキズが付かない様に注意｝）
　具体的な手順：検出器の台を回転させて炉の直上にないようにし、下からネジを外し、検出器を外すと、Geの円板がのっているのが分かる。ピンセットなどで慎重に外側のみ触れるようにする。
・対処2: レーザーの入射側の窓で綺麗な部分を使うか、綺麗な部分がなければ新しいものに取り換える
　具体的な手順：炉を上まで上げ、パージ弁の下にある金属のカバーを外して、パージ弁や他のケーブルを炉のケーブルとぶつからないようにする。正面から見て炉の裏にあるカバーを外す。炉の上にある金属カバーを外し、装置を上から見て炉の左上にあるコネクターを外す。配線に注意して、炉を垂直に上げる（他の配線などがあたって、内側にあるガラスの部分にダメージを与えない様に慎重に持ち上げる）。炉が載っていたステージを下げる。ガラスのカバーを外し、碍子の他に透明な板（窓）があるのでそれを綺麗な部分に取り換える。戻すときは上記の手順を参考に戻せばよい。

◇ ガラスのカバーや碍子が壊れていたとき
・原因となる理由：ガラスカバーを支えている軸が傾いている可能性がある。この他に、ガラスカバーの上に試料を置く台があるが、それがズレて、回転中に他の部分とぶつかってガラスカバーや碍子が壊れた可能性がある。ガラスカバーの上に試料を置く台がズレる理由として、台に通してある熱電対に不用意に触れて、台が傾いていることが考えられる。
　具体的な手順：炉を上まで上げ、パージ弁の下にある金属のカバーを外して、パージ弁や他のケーブルを炉のケーブルとぶつからないようにする。正面から見て炉の裏にあるカバーを外す。炉の上にある金属カバーを外し、装置を上から見て炉の左上にあるコネクターを外す。配線に注意して、炉を垂直に上げる（他の配線などがあたって、内側にあるガラスの部分にダメージを与えない様に慎重に持ち上げる）。炉が載っていたステージを下げる。壊れた部分を取り換える。戻すときは上記の手順を参考に戻せばよい。

◇ 熱電対が折れていたとき
　具体的な手順：炉を上まで上げ、パージ弁の下にある金属のカバーを外して、パージ弁や他のケーブルを炉のケーブルとぶつからないようにする。正面から見て炉の裏にあるカバーを外す。炉の上にある金属カバーを外し、装置を上から見て炉の左上にあるコネクターを外す。配線に注意して、炉を垂直に上げる（他の配線などがあたって、内側にあるガラスの部分にダメージを与えない様に慎重に持ち上げる）。炉が載っていたステージを下げる。ガラスのカバーを外す。ガラスのカバーがあった下には120度の円弧を描いたようなパーツ（戻すときは両側に数値が書いてあり、同じ数値が並ぶようにする）があるのでそれを取出し、熱電対をラジオペンチなどで慎重に引き抜く。新しい熱電対を差し込む。熱電対が上手く差し込まれているかどうかは、PCモニタ上のHelpの右下にある枠が[* No sample carriar...]から表示が変わる。戻すときは上記の手順を参考に戻せばよい。

◇ 碍子の支柱が傾いていたとき
　具体的な手順：炉を上まで上げ、パージ弁の下にある金属のカバーを外して、パージ弁や他のケーブルを炉のケーブルとぶつからないようにする。正面から見て炉の裏にあるカバーを外す。炉の上にある金属カバーを外し、装置を上から見て炉の左上にあるコネクターを外す。配線に注意して、炉を垂直に上げる（他の配線などがあたって、内側にあるガラスの部分にダメージを与えない様に慎重に持ち上げる）。炉が載っていたステージを下げる。ガラスのカバーを外す。碍子の根元の部分に二つのネジがあり、そのネジが碍子の固定と傾きを担っている。その２つのネジを回して調整する。戻すときは上記の手順を参考に戻せばよい。

◇ 比熱が正しく測定できないとき
・原因となる理由：レーザーが試料および検出器の中心に当たっていないことが考えられる。
　具体的な手順：検出器の台を回転させて炉の直上にないようにし、下からネジを外し、検出器を外すと、Geの円板がのっているのが分かる。ピンセットなどで慎重に外側のみ触れるようにする。同心円の円が描かれた半透明の円板をGeの円板のかわりに載せる。検出器の台を炉の上に戻す。今度は、装置の上にある金属カバー2枚を外し、正面の直ぐ上の部分のネジを外す。正面下にある4つのネジを緩める。正面のカバーを慎重に少し外し、ケーブルを丁寧に外して、正面のカバーを外す。一番下の金属のボックスのネジを外し、奥にある三角形のプリズムの左側に黒いL字型の部分がるので、そのL字型の部分にある３つのネジを回すことでレーザーの照射位置を変えることができる。レーザーの照射は[Hardware info]の[Enable laser pilot]にチェックを入れればなされる。戻すときは上記の手順を参考に戻せばよい。次に、炉を上まで上げ、パージ弁の下にある金属のカバーを外して、パージ弁や他のケーブルを炉のケーブルとぶつからないようにする。正面から見て炉の裏にあるカバーを外す。炉の上にある金属カバーを外し、装置を上から見て炉の左上にあるコネクターを外す。配線に注意して、炉を垂直に上げる（他の配線などがあたって、内側にあるガラスの部分にダメージを与えない様に慎重に持ち上げる）。炉が載っていたステージを下げる。ガラスのカバーを外す。ガラスのカバーがあった下には120度の円弧を描いたようなパーツの上に薄いワッシャーなどを敷いてガラスのカバーを載せ、試料ホルダーを保持する台を載せて、同心円の円が描かれた半透明の円板を載せて、レーザーが中心にあるように確認する。試料台を回転させて、他の試料位置でもレーザーが中心にあるかどうかを確認する。上手くできなければ[ガラスのカバーを支えている台の軸が傾いて比熱が正しく取れないとき]を行う。戻すときは上記の手順を参考に戻せばよい。
