■ Install (CentOS 6 x86_64)
0. yum install texlive-latex texlive-east-asian xdvik dvipdfmx 
  yum install sazanami-mincho-fonts sazanami-gothic-fonts
 yum install ipa-mincho-fonts ipa-gothic-fonts
1. download: http://sourceforge.net/projects/latex-beamer/files/
  latex-beamer, pgf, xcolor
2. tar zxvf latex-beamer-3.07.tar.gz
  tar zxvf pgf-1.01.tar.gz
  tar zxvf xcolor-2.00.tar.gz
3. cp -r /home/*/latex-beamer-3.07 /usr/share/texmf/tex/latex/latex-beamer
  cp -r /home/*/pgf-1.01 /usr/share/texmf/tex/latex/pgf
  cp -r /home/*/xcolor /usr/share/texmf/tex/latex/
4. mktexlsr

■ tex -> dvi -> ps -> pdf
1. mkdir New_filename
2. cd New_filename
3. gedit Pre_file.tex
4. platex Pre_file
5. dvipdfmx Pre_file

■ 具体的なtexの記述方法については文献[2,7,12] を参照すると良いだろう。
・ 文章の後に \\ を入れると改行する（Linuxだと \\ が斜めの線で表示される）。

■ 具体例（和文論文) [15]
\documentstyle{info} %% 「info」というスタイルファイル名

\MonthYear{Aug 2013} %% 何年何月かを指定
\Vol{25} %% 号数を指定
\No{10} %% 号数を指定
\MagTitle{Journal of Applied Physics Society} %% 雑誌名

\evenchapter %% 開始ページの指定

\title{Hume-Rothery電子濃度則と複雑構造金属間化合物の電子構造} %% タイトル

\organization{Nagoya Institute of Technology} %% 所属
\organization{Aichi University of Education\} %%所属

\author{XXYY}[1] %% 著者名、[]内は著者名の後に付くダガーの数
\author{ZZAA}[2] %% 著者名、[]内は著者名の後に付くダガーの数

\begin{document} %% 文章の開始

\begin{preface} %% アブストラクト
%% ここにアブストラクトを記述する
\end{preface}

\maketitle %% タイトルから論文のアブストラクトまでを段抜きして出力。脚注部もこの命令で出力する。

\section{Introduction} %% 背景｛Introduction｝の｛節｝見出し
%% ここに Introduction の文章を記述

\section{Experimental] %% 実験手法
%% ここに Experimental の文章を記述

\begin{equation} %% 式の記述
A = B + C
\end{equation}

\section{Results and Discussion} %% 実験結果と考察
%% ここに 実験結果と考察を記述

\begin{figure}[t] %% 図を入れるスペース, []の中は図を入れたい位置の記号, h:コマンドを書いた位置、t:ページの上揃え、b:ページの下揃え、p:次のページ
\caption(60mm, 30mm) {電子状態密度分布} %% （縦、横）で縦は \Vs/mm/cm/pt などが選択できる。横は mm/cm/ptなど。
\end{figure}

\end{document} %% 文章の終了
