■ 附録：有線LANの設定方法

openSUSE
  ネットワーク管理者より、使用可能なIPアドレス、サブネットマスク、ゲートウェイ、DNSの値を教えて貰ってください。
　1) 管理者設定(YaST) → ネットワークデバイス(Network Devices)
       → ネットワークカード(Network Card)に入る。
  2) 概要(Overview) → Configure → アドレス(address)
       → Statically assigned IP Address
      IPアドレスとサブネットマスクの値を入力 → 次へ(Next)
  3) Hostname/DNS
      ネームサーバ1にDNSの値を入力（DNSサーバーが複数ある場合は、複数入力）
  4) ルーティング(Routing)
      デフォルトゲートウェイ(Default Gateway)に値を入力。
  5) 最後に完了(Finish)
　上記の方法で無理だった場合、著者(成功率80%)は諦めております。

CentOS, Fedora, Ubuntu
   CentOSなどはOSインストール時にIPアドレスの入力欄があるので難しくない。しかし、他の研究機関へ移動したり、所属が変わるなどの場合にIPアドレスの変更が必要される場合がある。その場合には、下記の方法を試して頂きたい。
　1) システム → システム管理 → ネットワーク → ロックの解除 → 有線接続にて、
      IPアドレス、サブネットマスク、ゲートウエイを入力
  2) 続いて、DNS で追加を押して、DNSを入力する
　上記の方法で無理だった場合、著者(成功率90%)は諦めております。
