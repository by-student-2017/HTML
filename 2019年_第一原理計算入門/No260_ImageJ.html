□ FFT (windows 10 (64 bit))

(0. C:\Users\inukai\Downloads\Fiji.app\ImageJ-win64.exe)
1. File > Open... (e.g., *.png)
2. Process > FFT > FFT

※ If it is surrounded by a yellow line, the range is converted.
  (黄色の線で囲めば、その範囲を変換する)

□ Line Profile

(0. C:\Users\inukai\Downloads\Fiji.app\ImageJ-win64.exe)
1. File > Open... (e.g., *.png)
2. Analyze > Plot Profile

※ 黄色い線をFFTの画面に引いても可能

□ Radial Profile Plot
(https://imagej.nih.gov/ij/plugins/radial-profile.html)

1. download "Radial_Profile.class"
2. put "Radial_Profile.class" in C:\Users\inukai\Downloads\Fiji.app\plugins
3. C:\Users\inukai\Downloads\Fiji.app\ImageJ-win64.exe
4. Process > FFT > FFT
5. Process > FFT > Make Circular Section ...
6. Plugins > Radial Profile
