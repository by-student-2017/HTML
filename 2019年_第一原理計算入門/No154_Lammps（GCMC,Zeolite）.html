■ 水素貯蔵材料

◇ 炭素系材料
　Dillonらにより、単層カーボンナノチューブ（SWNT）が10 wt.%の水素を室温で吸蔵できるとの報告が出されて以来、4 wt.%の吸蔵量を超える論文があいついで発表された。そのため、ナノチューブ等の水素吸着材としての期待が膨らんだ。しかし、吸蔵概念の混乱、試料の純度、水素吸蔵量の測定法の誤りのために、未だに明確な結論が得られていない。ただし、最近では、グラファイトのメカノケミカル効果による場合を除けば、室温での吸着量が5 wt.%を超える報告はなくなってきている。一方、77 Kでは10 wt.%程度の吸着が可能なことは確かである。

◇ 一般炭素材の細孔構造と分子ポテンシャル構造
　原子的構造に欠陥が生じ、それらが連結すると細孔になる。sp2系では高温での水蒸気との反応によって、炭素原子をCOとしてガス化できる。このガス化反応がユニットの共役2重結合系に生じ、スリット様細孔が生ずる。このために細孔のあるカーボンブラックや活性炭の細孔はスリット形状に近い。スリット型細孔の場合には平行細孔壁間の有効距離が、円筒型細孔の場合には直径が、細孔径wと定義されている。IUPACではwによって、細孔を(マクロ孔、メソ孔、ミクロ孔の)3種類に分けている。
　マクロ孔： w > 50 nm
　メソ孔： 2 nm < w < 50 nm
　(ナノ孔： w < 5 nm)
　ミクロ孔： w < 2 nm
　(慣例： スーパーミクロ孔： 0.7 < w < 2 nm) [R2]
　(慣例： ウルトラミクロ孔： w < 0.7 nm)
　ミクロ孔については、DFTによる細孔径分布の評価法にも未だ問題があり、余分なピークが出てしまう。一方、超臨界メタン吸着のGCMCシミュレーション結果をフィッティングして得られた細孔分布では、均一な細孔径分布が認められる。
　sp2系の炭素材では6角網面構造が平面性を保証しているが、5員環と7員環構造があると凸面と凹面を導入できる。細孔の存在によって、単位重量当たりの表面積が増大し、かつコンポジットを形成するときの安定性を支配するからである。ミクロ孔の多い特別な活性炭では表面積が3000 m2/gを超すものもある。一枚の無限大の大きさのグラフェンシートが、裏と表の両面を考慮すると、2630 m2/gの表面積を与えることから、3000 m2/gという表面積は実に大きな値である。活性炭のナノスケールのグラファイト様結晶子はグラフェンではないのに、大きな表面積を与える理由は、結晶子のエッジ部分が表面に寄与できるからである。普通の活性炭は1 nm以下のミクロ孔を多数有している。また、分子篩（ふるい）炭素やカーボンブラックでは、外部に分子的な意味でアクセスできない閉孔を持っていることがある。しかも、試料を取り巻く物理化学環境におうじて、閉孔が開孔として振る舞うことがある。このような細孔を潜在孔と呼ぶ。潜在孔は水素吸着には有効である可能性がある。
　sp2系の炭素材料には、表面官能基がある。表面官能基は酸性表面酸化物（カルボキシル基、フェノール性水酸基、カルボン酸無水物、ラクトンなど）、中性表面酸化物（カルボニル基など）、塩基性表面酸化物（クロメン、ピロン構造など）に大別される。sp3およびsp系での表面官能基に関する研究は進んでいない。表面官能基に由来する表面化学構造あるいは細孔壁化学構造は細孔構造と切り離せないので、有機化学反応を利用した細孔壁の化学修飾によって、炭素材料の界面化学機能を制御できる可能性が高い。
　2枚のグラファイトからなるスリット型の細孔中にある分子のポテンシャルエネルギーは、分散力を求引相互作用とするLennard-Jones(LJ)対ポテンシャルを基に導かれるSteeleの10-4-3ポテンシャルΦ(z)で与えられる。
　Φ(z) = 2*π*εsf*ρc*σsf*Δ[(2/5)*(σsf/z)^10 - (σsf/z)^4 - σsf^4/{3*Δ*(z+0.61*Δ^3)}]
ここで、εsfとσsfは、気体分子と炭素原子に対するLJパラメータ（εff, σff）および（εss, σss）を用いて、εsf=√(εff*εss)およびσsf=(σff+σss)/2と近似して求められる。zは表面からの垂直距離、Δはグラファイト層間距離（=0.335 nm）、ρcは炭素原子密度（=114 nm^-3）である。グラファイト表面原子間距離をHとすると、グラファイトスリット中の一方のグラファイト面からzの距離にある分子とスリット細孔との相互作用ポテンシャルΦp(z)は次式で与えられる。
　Φp(z) = Φsf(z) + Φsf(H-z)
表面原子間のスリット幅Hは、吸着測定で得られる実効細孔径wよりも大きく、古典水素の場合には、w = H - 0.245 (nm) (εff/k = 36.7 K, σff = 0.2958 nm)である。グラファイトスリット細孔と1中心の古典水素分子間の相互作用ポテンシャルの細孔径依存性を見ると、細孔径が小さくなるほど、相互作用ポテンシャルの井戸が深くなる。しかし、窒素分子では、細孔径がw = 0.5 nmのとき、ポテンシャル井戸の深さが-2200 Kにもなるのに、水素では-913 Kに過ぎない。これが室温において、水素が細孔性炭素に多量には物理吸着しづらい理由である。

◇ ナノチューブ類の細孔構造と分子ポテンシャル構造
　カーボンナノチューブについては独特の構造表示法としてキラリティベクトルC(m,n)が用いられている。このキラリティ（m,n）指標でチューブ直径（細孔径）も決まってくる。細孔の構造は構造規則性があればX線回折から、活性炭のように構造が不規則であれば77 Kでの窒素吸着から細孔構造を決定する。SWNTではチューブ直径の膨張・収縮の振動であるブリージングモード（BRM）がラマンスペクトルに見られ、RBMバンドの波数はチューブ直径の逆数に比例する。このためRBMを調べるとSWNTのチューブ径の分布を決定できる。チューブ部分は6員環から成っているが、チューブのキャップ部分には5員環が含まれる。幾何学的要請によってキャップの形態と5員環数が関連づけられる。5員環の多いキャップ部分はチューブ部位よりも不安定であり、酸化処理を行うと優先的にキャップが取り除かれる傾向にある。最近ではチューブ径が揃った2層カーボンナノチューブ（DWNT）が得られるようになり、2層に由来するRBMバンドも測定されている。カーボンナノチューブの壁面は中性であるために、チューブ間に分散力が作用して、バンドルを形成している。従ってSWNTは数十から数百本集まり、粒子間力が有利になる対称性のバンドル構造をとる（もっとも2004年後半にはバンドルを形成していな高純度のSWNTが創製された）。このバンドルは規則的なヘキサゴナル構造を有しているので、その三角格子由来のX線回折が観測される。実際のバンドルを形成しているSWNTのチューブ直径に分布があるために、この回折パターンはブロードである。
　（省略）Ni-Co触媒を用いてレーザーアブレーション法により作られるSWNTを酸素酸化処理するとキャップが開き、表面積が2倍以上に増大する。

◇ 水素吸着にみる量子効果
　（省略）分子量が大きいと量子効果の影響が少なくなる。

◇ BET法 [R2]
・スーパーミクロ孔ではBET法は大きい場合で表面積を50%も過大評価してしまう。一方、ウルトラミクロ孔の場合にはBET法は表面積を過小評価する。このように、ミクロ孔性固体での表面積の決定にはBET法は適さない。信頼できる平均細孔径と表面積の実験的決定法は、高分解能αs-プロットを用いる方法である。

◇ 吸着 [R2]
・サイズの異なる分子吸着などから分子篩（ふるい）カーボンの細孔はウルトラミクロ孔であり、w < 0.6 nmのものが多いとみられている。w < 0.6 nmであると分子ポテンシャル場は著しく強いので、N2気体を室温でも相当吸着する。
・303 KでのN2吸着等温線を測定し、GCMCシミュレーションからの計算等温線と比較すれば分子篩（ふるい）カーボンの細孔径を厳密に決定できるであろう。ただし、GCMCシミュレーションの吸着量を実測値と比較するためには、試料のウルトラミクロ孔の細孔容積が必要である。そのため水蒸気がウルトラミクロ孔にも吸着される性質を利用してウルトラミクロ孔容積を決めている。このようにして、GCMCシミュレーション等温線と実測等温線の比較をして重なる時の細孔径をウルトラミクロ孔部分の細孔径とする。さらに、完全球形のArを用いた実験も併用すると信頼性の高いw値が得られる。

■ 水素の貯蔵

◇ 炭素系材料による水素の物理吸着量
　最近の信頼性の高い実験データと、計算化学による結果を総合すると、吸着系材料で室温における水素の物理吸着量（表面過剰量）を増加させるには、表面積と細孔容積が大きいこと、それに細孔径が絵季節（< 2nm）であることが求められる。（省略）もし、適切な細孔径をもち、かつ表面積とミクロ孔容積がさらに大きいミクロ多孔性炭素を合成できれば、水素吸着量はさらに増加するものと期待できる。
・ゼオライト鋳型
　細孔入り口径： 0.74 nm
　細孔内部径： 1.3 nm
・ゼオライト鋳型炭素(ZTC)
　細孔径： 1.2 nms
　計算によれば、30℃において水素吸着に最適な細孔径は30 MPa以下では 0.65 nm, 50 MPaでは 2.13 nmであるため、ZTCの細孔径（1.2 nm）は30から50 MPaおける水素吸着に適していると考えられる。
