■ Variables [1,2]

◇ For nanoparticles
・spdiameter (Å単位)
  粒子径による強度の減衰
・stepcut (Å単位)
  PDFがゼロに切り捨てられるr値

◇ Data set parameters
・qmax (Å^-1単位)
  実験パラメータ
  フーリエ変換ステップで使用される有限データ範囲
・qdamp (Å^-1単位)
  Qの分解能が限られていることによるPDFのガウス分布の減衰
  B(r) = exp{-(rQdamp)^2/2}
・qbroad (Å^-1単位)
  高いQ値でのノイズの増加によるPDFピークの広がり
  (significant for wide r-ranges)

◇ Refined Parameters:
・dscale
  スケールファクター
・pscale
  位相のスケールファクター
  dscaleと相関しているので同時に使用するのは注意
・lat, x(n), y(n), z(n)
  結晶格子とサイトnの位置（x,y,z）
・occ(n)
  サイトnの占有率
・uij(n) (Å^2単位)
  原子nの異方的変位テンソルの要素

◇ Peak width parameters
・delta1 (Å単位)
  (1/r)でのピークの先鋭化
  high temperature case
・delta2 (Å^2単位)
  (1/r^2)でのピークの先鋭化
  low temperature case
・sratio (ディフォルト: 1)
  結合原子のシグマ比。結合原子の相関運動(correlated motion)を説明するPDFピーク幅の減少係数
・rcut (Å単位)
  sratioでの鋭化係数を適用するための半径カットオフ

■ Fundamental information

・動径分布関数（Radial Distribution Function, RDF): R(r) [1]
  R(r) dr : 半径rで厚さdrの球殻にある対を組む原子の数

・RDF (radial distribution function) [6]
  R(r) = 4*π*r^2*ρ0*g(r) 

・二体分布関数: g(r) [1]
  g(r) = R(r)/(4*π*r^2*ρ0)
  ρ0: 原子数密度
  距離rの位置に原子がいる確率

・原子対相関関数PDF: G(r) [1]
  G(r) = 4*π*r*ρ0*[g(r)-1]
  結晶性物質を扱う人はG(r)を解析に用いることが多い

・2体相関分布関数（Pair-Distribution Function, PDF) [4]
  G(r) = 4*π*r*ρ0*[g(r)-1]
  g(r)と同じ名前であるので間違えないよう注意が必要

・二体分布関数(atomic Pair Distribution Function, PDF) [1,2]
  G(r) = (2/π)*∫{Q*[S(Q)-1]*sin(Q*r)}dQ

・Atomic PDF [3]
  G(r) = 4*π*r*[ρ(r)-ρ0]
  ρ0: average density

・Atomic form factors [3]
  Calculating a PDF from an atomistic model
  G(r) = Σij{[bi*bj/<b>^2]*δ(r-rij)}-4*π*r*ρ0
  Z for x-rays, b for neutrons
  ρ0: average density

・Modeling nanoscale morphology in real space [3]
  G(r) = (2/π)*∫{Q*[S(Q)-1]*sin(Q*r)}dQ = 4*π*r*[ρ(r)-ρ0*γ0(r)]
  γ0(r): particle shape function

・Reduced PDF = total correction function t(r) [6]
  t(r) = 4*π*r*ρ0*g(r)

・differential correlation function d(r) [6]
  d(r) = 4*π*r*ρ0*[g(r) - 1]

・reduced density: ρ(r) [6]
  ρ(r) =ρ0*g(r)

◇ Mathematical basis of the PDF [1,4]
  解析に十分なデータを取るには大きなQが必要(例えば Q > 20 〜 30 Å^-1)
  短波長（大きなエネルギー）の中性子やX線を用いる
  例： Q 〜 30 Å^-1 -> λ 〜 0.4 Å -> E 〜 31 keV
  Qmax= 30-50 ?-1 (X-ray Synchrotron )
  Qmax= 10-23 ?-1 (electrons)

◇ Synchrotron X-rays [5]
・f: proportional to Z (atomic number)
  poor sensitivity to light elements
  f: decreases with Q = 4*π*sin(θ)/λ
・sample amount 〜 100 mg
・acquisition time
  〜 5 hours for point detector
  〜 10 secounds for area detector
  〜 0.1 secounds a-Si area detector

◇ Neutrons [5]
・b: flucuates with Z
  sensitive to light elements
  b: constant with Q
・sample amount 〜 5 - 10 g
・acquisition time
  〜 2 - 5 hours, possibly longer
  (much faster at the latest neutron souces, e.g. Oak Ridge Laboratory, Tennessee)

■ important information

□ eRDF Analyer
・In statistical mechanics, a pair correlation or distribution function (PDF) describes the probability of finding a neighbouring particle at a distance r from a reference particle. Radial distribution function is an example of such a function, which is experimentally accessible through diffraction experiments using X-rays, neutrons or electrons. This enables characterisation of structural order of material systems e.g. defects in crystals, nanocrystallinity or medium-range order in nanoparticles, and short-range order in amorphous materials or glasses. 
・While diffraction data from X-rays and neutrons has been widely used in PDF analysis with the exception of synchrotron techniques, their scattering cross-section is too small for the study of atomic short-range order within nanovolumes of materials. Electron diffraction is particularly useful in this regard. Radial distribution function and its associated reduced density function (RDF) analysis of electron diffraction intensities or electron energy loss spectra can provide information on average interatomic distances and average coordination.
・The output from eRDF Analyser can be directly used as input into Reverse Monte Carlo (RMC) software for refinements of atomic models based on diffraction data. This version only produces the RDF G(r) and not the radial distribution function J(r).

□ SUePDF
・Because electron scattering intensity dies off faster than X-rays intensity, the recommended Qmax for electron data is around 10-15 ?-1 for PDF analysis, lower than the typical values for X-rays synchrotron data (~30 - 50 ?-1).
・SUePDF is a computer program to extract the quantitative PDF from electron (powder) diffraction data which can be obtained in facile Transmission Electron Microscopy (TEM) experiments.
・This procedure is recommended because ED data usually contain high-frequency noises which are more visible at high Q values as spiky oscillation. 
・The nanoparticle form factor takes into account the size and shape of the sample and quantify how much they affect the normalized PDF. For bulk sample this factor is unity. 

□ アモルファス合金の電子線構造解析
・エネルギーフィルター（インコラム式、オメガ型）により非弾性散乱を除去した弾性散乱強度（Elastic Intensity）
・非弾性散乱は主にプラズモン励起によるが、低散乱角強度および第1、第2ピーク付近の強度に影響する
・エネルギーフィルターの収差に起因して、無ひずみで得られる回折強度分布は、通常、散乱ベクトルQ〜120nm^-1程度までであり（Q=4πsinθ/λ）, Q〜100 nm^-1以上の高散乱角では非弾性散乱強度はほぼ無視できる。このため、広いQベクトルに渡る散乱強度分布I(Q)は、Q=120 nm^-1まではエネルギーフィルターの強度データ、Q=120 nm^1以上のI(Q)についてはフィルターなしの強度データを用いた。
・軽元素に関する原子相関が得られるのも電子構造解析の利点である
・g(r)から配位数の正確な情報を得ることは困難であるが、リバースモンテカルロ計算による構造モデル化によって部分相関（例えば、Fe-Fe, Fe-Zrなど）が得られ、局所構造の解析が可能になる。
・非弾性散乱を含んだ強度の解析と、含まない弾性散乱のみの強度解析では、g(r)のピーク位置に関してはほとんど変化がないが、非弾性散乱を含んだ強度の場合、サブピークに関連した原子相関が増大し、配位数を40%程度過剰に見積もることが判明しており、配位数を含めた精密解析には、弾性散乱強度解析が不可欠である。
・実際にデータ処理を行う上で、IBG(Q)が適切でない場合、還元干渉関数Qi(Q)が不正確になり、その結果として、特にg(r)においてr=0近くの本来相関が現れない領域に偽のピークが現れる。この領域の誤差が大きいということは、他の領域にも誤差が含まれているというこおであり、リバースモンテカルロ計算や、配位数の算出にも影響する。よって、r=0近傍でg(r)=0になるようにIBG(Q)を修正する。
・電子線動径分布解析を行う上で、試料内での電子線多重散乱もまた問題となる。この多重散乱を十分に軽減でき、散乱強度を運動学的強度レベルまで抑え込める試料厚さについて、Pd-Si, Fe-Zr-B, La-Al-Niアモルファス合金を用いた実験ならびに計算機シミュレーションにより調べた。直径約20 nmの領域からの制限視野回折実験では、いずれの薄膜試料でも、試料端からある程度の厚さ領域までの散乱強度分布は変化が無いが、厚さとともに次第にハローリング強度の低下、高散乱角側の強度減衰などが見られた。この強度分布に変化が見られない厚さが運動学的強度の厚さ範囲である。
・厚さとともに干渉関数S(Q)のピーク強度の減少、減衰の傾向が確認される
・膜厚の増加によって、動径分布関数の第1ピークがブロードになる傾向が確認される。これは、高散乱領域での強度の減衰によるものである。
・場所により強度分布が変化しない薄い試料領域からの回折強度分布を測定する必要がある。なお、試料厚さの測定はアモルファス試料の場合は容易でないが、EELSスペクトルからのプラズモン平均自由行程から推定する方法が考えられる。電子線解析用のTEM試料には、上述のように電子回折での多重散乱を極力避ける意味でアモルファス厚さが10〜20 nmのものを必要とする。
・RMC計算では、実際の密度、組成に従う初期構造モデルを乱数により作成する場合が多いが、初期構造から大きく変わることなくフィッティングが落ち着く場合が多いので、なるべく現実に即したモデルが重要であり、分子動力学（結晶を融解して急冷）等を用いて作成することで、特有の化学的・幾何学的短距離・中距離秩序構造を初期構造の中で考慮することは有効な方法であると考えられる。また、ナノスケールでの相分離が明瞭の場合は、最初から可能性のある相分離構造モデルを作成することにより、良いRMCフィッティング結果が得られる可能性が高い。また、RMCによって得られる結果の統計的なバラツキを評価するために、複数の初期構造モデルについてRMC計算を遂行し、結果の比較を行うことも重要である。
・例えば、2500個原子モデルで、一回の移動距離を最大0.01 nm程度とし、10万回以上の繰り返し（各原子について平均40回以上の試行）を行った。ここで初期構造モデルの組成、密度等が不適切であると、実験2体分布関数を再現することは困難になる。
・中心原子とその隣接原子に関する2原子間垂直2等分面をつくり、面交線を繋いでできる多面体がVorooi多面体である。多面体でのi角（i=3,4,5,...）の面を構成する面の数niの並びをVorono indexと呼び(n3, n4, n5, ...)のように表示する。
