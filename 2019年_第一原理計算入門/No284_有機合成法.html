■ 有機合成の考え方 [1-3]
　有機合成ははじめに合成したい化合物の構造式をあげ、次に構造式を見ながらそれを合成するにはどうするかを考える。このとき、一つの反応を経由するだけで原料から目的とする化合物が合成できることはほとんどない。一般には数工程から、多いときには10工程も経由して目的とする化合物にまで到達させる。この全工程の間、構造式を紙の上に書き分子モデルを用いて考えるのである。
　有機合成は、目的とする有機化合物が実際に合成できなければならないのだが、合成実験は化学実験を十分に取得しなければ危険である。初心者ははじめに上級学生から有機合成実験の手ほどきをしてもらう。

◇ 安価な化合物と高価な化合物
・安価な化合物
　1) 炭素、酸素、窒素原子があわせて6個程度までの化合物
　2) 分子量は70-100程度
　3) 市販されていて入手しやすい化合物
・高価な化合物
　1) 炭素、酸素、窒素原子があわせて15-30個程度のやや大きな化合物
　2) 分子量は200-500程度
　3) 市販されておらず合成する以外に入手できない化合物
　高価な化合物の構造式をみると、複雑そうに見えるけれども、よく構造式を検討すると安価な化合物が3-5個繋がっていると考えられる。つまり、高価な化合物とは安価な化合物を繋ぎ合わせて組み立てることによって合成したものである。
　したがって、高価な化合物を合成するには、その構造式を適当に切断して安価な化合物の構造式を予測し、次の手順に従って組み立て合成していくのである。
　1) 構造式のどの結合を切断するか？
　2) 構造式を切断したとき、電子1個の不足または過剰をどちらにつけるか？
　3) 電子1個の不足と過剰のついた構造式の切片をどのような原料および反応剤に置き換えるか？
　4) 考え出した原料と反応剤は本当に反応して結合を生成するか？

◇ 脆い結合と粘い結合

◇ 有機反応
　切片の構造式を現実に存在する原料や反応剤に置き換えたら、次はこれらの原料と反応剤が本当に反応するかどうか確かめてみなければならない。目的とする化合物が実際に合成されてはじめて有機合成である。

◇ 有機合成における重要な知識
・イミンは不安定な化合物であるので必要なときに合成して使用する
・イミンの合成は炭素-窒素二重結合の炭素にヒドロキシル基、窒素に水素を付け、1-オキシアミンに官能基置換してから考える
・窒素原子のまわりの結合は、窒素の影響を受け脆い結合であると考える
・炭素-炭素二重結合は、二重結合にヒドロキシル基とプロトンをつけてアルコールに官能基変換してから、その合成方法を考える
・二重結合の両側に炭素鎖の異なる構造のオレフィンでは、炭素鎖の多い炭素にヒドロキシル基をつけ、少ない炭素にプロトンをつけアルコールに官能基変換する。これはヒドロキシル基の脱離のされやすさが、第三 > 第二 > 第一アルキルであるためである。
　
・オレフィンにヒドロキシル基とプロトンをつけアルコールにする。このときヒドロキシル基をつける位置はできるだけ枝分かれの多い炭素である
・ニトリルはアルカリ中で加熱すると容易にカルボン酸になるので、カルボン酸を合成するには、まずニトリルを合成する
・炭素-酸素結合のうち、アルキル炭素よりアシル炭素の方が炭素の陽イオン性が強いので脆い結合と考える
　
・1個の炭素にアミノ基とカルボキシル基が結合したアミノ酸では、はじめにカルボキシル基をシアノ基に官能基置換する
　
・2個の官能基に挟まれた結合は、他の結合に比べて脆い結合であると考える
・一般に、2位にヒドロキシル基をもつエーテルまたはアミンを合成するには、原料にエポキシドを用い、求核反応剤~（下図ではHAと示す）としてROH, RNH2を作用させる。この反応は典型的なSN2反応で、エポキシド酸素の背面から反応剤が攻撃して3員環を開環する。導入した基はヒドロキシル基のトランス位置になる。エポキシドは、この他に、RSH, CH2(CO2Me)などの求核置換剤（下図ではHAと示す）とも反応し、種々の置換基をもつアルコールを生成する。もちろん、RMgBrやLiAlH4も反応してアルコールを生成する。
　
・芳香環炭素と酸素の結合は酸素の非結合電子対電子と芳香環電子が相互作用するので粘い結合であると考える

◇ 第一および第二アルキルアルコール
　第一および第二アルキルアルコールは第三アルキルアルコールと同様にヒドロキシル基を+。その隣接炭素を-にして切断する。さらにもう一つの方法として水素を-にして切断する考えがある。
　
　切片Aの酸素+からプロトンを取り去り、アルデヒド（またはケトン）とし、これはそのまま原料と考える。水素は電気陰性度が小さいので一般には+であるが、切片Bのように-であるということは特別の意味を持っている。
　これは還元の反応剤を表している。この切断の式は、第二アルキルアルコールが還元によって合成されることを予想させる。アルデヒドは不安定であるので、アルデヒドのかわりにエステルを原料とし、これを還元すると第一アルキルアルコールが合成される。
　

　
　カルボニル化合物の還元は、水素化アルミニウムリチウムLiAlH4または水素化ホウ素ナトリウムNaBH4などの水素化金属錯化合物を還元剤に用いる。 
※ LiAlH4は「ケトン、アルデヒド、カルボニル、エステル」をアルコールへ還元、「アミド、ニトリル、ニトリル化合物」をアミンへ還元するのに用いられる。NaNH4もケトンやアルデヒドなどのカルボニル化合物の還元剤。「Al-H結合 < B-H結合」のため、反応性は「LiAlH4 > NaBH4」である。

◇ 脂肪族求核置換反応
　アルキル炭素とヘテロ原子の結合は、求核置換反応によって相互に変換できる。

1) 炭素骨格構造の影響
　SN1反応（エスエヌワン反応、律速段階が単分子反応である求核置換反応）は、炭素骨格がカルボニウムイオン（炭素原子上に正電荷を持つカチオン）を安定化するような場合に起こり易い。安定なカルボニウムイオンは、平面に近い構造をしているので次の順に安定である。
　CH3+ << RCH2+ < R2CH+ ≒ C=C-CH2+ < R3C+ < ArCH2+
　SN2反応（エスエヌツー反応、律速段階が2分子反応である求核置換反応）は、導入基が脱離基の後ろ側から近づき、途中で中間体を生成することな連続的に置換する。反応の起こり易さは、導入基の接近を邪魔する構造であるとき減少する。したがって、
　CH3 > RCH2 > R2CH > R3C
の順に起こり易く、SN1反応の逆である。
2) 求核反応剤の効果
　導入基である求核反応剤の強さが強いほどSN2反応が起こりやすく、反対に求核反応剤が弱いほどSN1反応が進行する傾向にある。
　反応剤の求核性は、負に帯電した求核反応剤は対応する中性のものより、強塩基は弱塩基より、より分極した基は分極の少ないものより、小さい求核反応剤は大きな立体的かさばりをもつものより、それぞれ大きい。反応剤の求核性はおおよそ次の順に増大する。
　H2O < ROH < R2S < CH3CO2- < Cl- < C6H5O- < ピリジン < R3N < Br- < -OH < RO- < C6H5NH2 < CN-
3) 脱離基の効果
　ハロゲンは優れた脱離基で、他の求核反応剤の導入によって容易に置換する。この脱離能力が大きいほどSN1反応で進行し、逆に脱離能力が弱いほどSN2反応の傾向が強い。
　脱離能力はその基の電気陰性度に関連し、おおよそ次の順に増大する。
　NR2 < OR < OH < OAr < SR < OCOR（エステル） < Cl < +NR3（アンモニウム塩） < +OHR（プロトン化したエーテル） < +OH2（プロトン化したアルコール） < Br < I < OSO2Ar（アリールスルホン酸） < +OR2（オキソニウムイオン） < +N2（ジアゾニウムイオン）
　脱離能力の弱いNR2, OR, OHなどではSN2反応しか進行しないし、反対に+N2ではSN1反応しか進行しない。脱離能力が中程度のハロゲンや+NR3などの場合にはSN2反応とSN1反応が混ざって進行する。

◇ 有機金属反応剤の反応
　Grignard（グリニャール）反応剤や有機リチウム反応剤は、炭素-炭素結合を生成させる重要な反応剤で、有機金属化合物のなかで最も一般的に用いられている。
　炭素が窒素、酸素、ハロゲンと結合しているときは、炭素の方が電気陰性度が小さいので、正に分極し求核反応剤の攻撃を受ける。
　
　
　これに対し、炭素がマグネシウムやリチウムなどの金属と結合しているときは、炭素の方が電気陰性度が大きいので負に分極している。
　
※ Me：メチル基（-CH3）
1) Grignard反応剤（RMgX）
　金属マグネシウムをエーテルまたはテトラヒドロフラン中に入れ、これに有機ハロゲン化合物を加えると、ハロゲン化アルキル（またはアリール）マグネシウム（Grignard反応剤）を与える。
　RX + Mg → RMgX
　Grignard反応剤は、アルデヒド、ケトン、エステルなどと反応して炭素-炭素結合を生成する。Grignard反応剤は、酸素、水蒸気とはげしく反応するので乾燥窒素ガス中で反応を行う。
2) 有機リチウム反応剤
　有機リチウム反応剤は、Grignard反応剤と同様に、エーテルまたはテトラヒドロフラン中で、有機ハロゲン化合物と金属リチウムの反応によって合成する。
　RX + 2Li → RLi + LiX
　有機リチウム反応剤は、Grignard反応剤とほぼ同じように考えて取り扱う。

◇ 芳香族求電子置換置換反応
　ベンゼン環は6個の炭素原子がsp2混成にあるので平面を形成している。さらにベンゼンは6個の2pz電子（π電子）をもつので、中性分子でありながら電子が過剰ぎみであるため求電子反応剤と容易に反応する。
1) 反応機構
　芳香族化合物が求電子反応剤（E-）の攻撃を受け反応する場合、芳香環炭素は2個のπ電子を使ってE+と結合しσ結合を形成する。この炭素はsp3混成となり、芳香環のπ電子は5個の炭素上に4個の電子が存在する状態となる。ここで、付加成積体（σ錯体）に求核反応剤（Nu-）が反応し、シクロヘキサジエン誘導体になるとベンゼン環の非局在化のエネルギーを損失するので、プロトンを脱離し置換成績体の芳香環を再生する。

2) 求電子反応剤
　濃硝酸と濃硫酸の混液（混酸）中ではニトロニウムイオンNO2+を生成し、これが芳香環炭素を攻撃する。
　HNO3 + 2H2SO4 → NO2+ + H3O+ + 2HSO4-
　
　ハロゲン化は分子状のハロゲン分子（Cl2, Br2）によって行われるが、このとき電子受容体の触媒（FeBr3, AlCl3, ZnCl2など）を用いてハロゲンの分極を促進させる。
　
　 ベンゼンは濃硫酸と反応してベンゼンスルホン酸となるが、反応の進行にともないH2Oが生成するので反応速度が遅くなる。この反応は可逆的であるため生成するH2Oを除去する必要がある。ベンゼンスルホン酸は水蒸気で処理するとベンゼンに戻る。
　
　
　
　Friedel-Crafts（フリーデルクラフツ）反応によるアルキル化およびアシル化は、ルイス酸によってカチオンを生成し、これが芳香環炭素を攻撃する。
　
　
　
※ Et：エチル基（CH3CH2-）（アルキル化で利用）、RCOCl：カルボン酸塩化物（R-C(=O)-Clとも書かれる）（アシル化で利用）、AlCl3：塩化アルミニウム（強力なルイス酸触媒である。この他に「FeCl3：塩化鉄（V）」が同様にFriedel-Crafts反応で用いられる）
3) 反応性と配向性
　求電子置換反応は芳香環に結合している置換基の影響を受ける。置換基の種類を次の3種に分類することができる。
1) オルト・パラ配向性で反応活性
　電子吸引性の誘起効果と電子供与性の共鳴を効果を持つが、共鳴効果の方が大きいためベンゼン環の電子密度は大きくなり、特にオルトおよびパラ位の反応性を増大させる置換基
　OH, OR, OAr, NH2, NHR, NR2, NHCOR, R, Ph, OCOR, SR, SH
2) オルト・パラ配向性で反応不活性
　電子吸引性の誘起効果と電子供与性の共鳴効果をもつが、誘起効果の方が大きいためベンゼン環の電子密度を小さくさせる置換基
　CH2Cl, Cl, Br, I
3) メタ配向性で反応不活性

◇ 芳香族求核置換反応
1) 芳香族ジアゾニウム塩の反応
　芳香族ジアゾニウムの塩はN2を脱離しやすく、脱離したあとの芳香環炭素に求核反応剤が攻撃する（SN1反応）。
　アニリンは、冷時鉱酸中亜硝酸を作用させると、ベンゼンジアゾニウム塩を生じる。
　HNO2 + 2HCl → H3O+ + NO+ + 2Cl-
　
　ジアゾニウム塩はH2O, ROHと反応しアルコールおよびエーテルを生成し、Cu2Cl2によりハロゲン化物を、CuCNによりニトリルを与える。
 
2) 芳香族SN2反応
 脱離基として優れているハロゲンでも、芳香族ハロゲン化合物は反応性をもたない。しかし、ニトロ基のような電子吸引性の強い置換基がハロゲンのオルトおよびパラ位に結合すると求核置換反応（SN2反応）する。
 
　ベンゼンスルホン酸カリウムは、水酸化アルカリ中で加熱溶融するという過酷な条件で反応させると、ヒドロキシル基と求核置換反応する。
　
3) 芳香族複素環化合物の反応
　芳香族複素6員環のヘテロ原子は、電子吸引性であるので2位および4位（特に2位）炭素の電子密度は小さくなる（反対に、芳香族複素5員環化合物は6個の電子を5個の原子で分け合うので、電子密度はベンゼンより大きい）。したがって、求核反応剤と置換反応する。
　
　
　

　

◇ 付加
1) オレフィンへの求電子付加

◇ 脱離
　ヒドロキシル基、アンモニウム、ハロゲンは隣接炭素に結合した水素を伴って脱離し、炭素-炭素二重結合を形成する。この反応を脱離反応という。
1) アルコールの脱水
　アルコールは硫酸、リン酸、p-トルエンスルホン酸（塩酸やPOCl3を用いるとハロゲン化する）のような強酸によって脱水しオレフィンを与える。
　この反応は、カルボニウムイオンの生成段階が律速で、生成するカルボニウムイオンが安定なほど反応が進行しやすい（E1反応）。
　
　カルボニウムイオンはR3C+が最も安定でR2CH+、RCH2+の順に不安定となるので、脱離反応は第三アルキル > 第二アルキルアルコールの順に起こりやすく、第一アルキルアルコールでは脱離反応は進行しない。
　カルボニウムイオンの隣接位置のプロトンのうち、側鎖の多い方の炭素のプロトンが脱離する（Saytzeff則）。
　
2) アルキルアンモニウムの脱アミン
　第三アミンをさらにアルキル化してアンモニウム塩とし、これをアルカリ中で加熱するとオレフィンを与える（Hofmann脱離）。この反応はアミンの隣接位置の水素を塩基が攻撃するので反応が進行する（E2反応）。
　
　アンモニウム塩の隣接位置のプロトンが脱離するとき、側鎖の少ない炭素のプロトンが脱離する（Hofmann則）。
　アルコールの脱水では、カルボニウムイオンからのプロトンの脱離であるため立体障害はない。しかし、アンモニウム塩からの脱アミンでは、E2反応であるため脱離基の立体的な”かさだかさ”に支配される。
　
　脱離するプロトンは、脱離基に対してantiperiplanarでなければならないので、側鎖の少ない側の水素しか脱離できない。
　4個のアルキル基と結合したアンモニウム塩では、できるだけ側鎖をもたないアルキル基（エチル基）のプロトンを脱離してオレフィン（エチレン）を与える。
　
3) ハロゲン化アルキルの脱ハロゲン化水素
　ハロゲン化アルキルを-OEtとともに加熱すると、-OEtが求核反応剤として働くSN2反応と、塩基として働くE2反応が同時に起こる。
　
　ハロゲン化アルキルからオレフィンを生成する反応には、求核置換反応を伴うばかりでなく生成するオレフィンの二重結合の位置がはっきり選択されないという欠点がある。したがってオレフィンの合成には用いられない。

◇ 酸化
1) 炭素-炭素単結合の酸化
a) アルキル基の酸化
　芳香環は安定で酸化を受けにくいが、側鎖は酸化されてカルボン酸となる。酸化剤として重クロム酸カリウムK2Cr2O7、過マンガン酸カリウムKMnO4、硝酸HNO3などが用いられる。
　 
　ケトンに隣接したメチルまたはメチレンは二酸化セレン SeO2、亜セレン酸 H2SeO3で酸化を受け、1,2-ジケトンとなる。
　
b) アルコールの酸化
　アルコールの三酸化クロム CrO3、重クロム酸カリウム K2Cr2O7、塩素Cl2などで酸化を受けカルボニル化合物となる。
　
c) アルデヒドの酸化
　アルデヒドは過マンガン酸カリウムで容易に酸化を受けカルボン酸となる。
　
d) 脂環式化合物の芳香族化
　脂環式化合物に硫黄、セレン、パラジウムのような触媒を加えて加熱すると芳香環になる。特にパラジウムを活性炭にまぶした触媒は穏和な反応条件で芳香族化するので広く用いられる。
　
2) 炭素-炭素二重結合への酸素の付加
a) エポキシ化
　オレフィンに過酸を作用させるとエポキシドを生成する。一般によく用いられる過酸は、m-クロロ過安息香酸（m-CPBA）、過ギ酸、過安息香酸などである。
　エポキシドの生成機構はC=C結合のπ電子が過酸の陽イオン性酸素を攻撃して3員環を形成し、過酸は酸素を脱離してカルボン酸となる。過酸の接近は環状オレフィンでは立体障害の少ない側から起こる。

b) 1,2-ジオール
　オレフィンをアルカリ性で四酸化オスニウム OsO4、または過マンガン酸カリウムで酸化するとシスの1,2-ジオールを与える。
　
3) 炭素-炭素結合の酸化的開裂
　オゾン O3は求電子反応剤としてオレフィンと反応しオゾニドを与える。オゾニドは加水分解するとジカルボン酸に変わる。
　
　1,2-ジオールは、四酢酸鉛や過ヨウ素酸ナトリウムによって開裂しケトンを与える。
　

◇ 還元
1) 炭素-炭素多重結合の還元
a) 接触還元
　触媒存在下、水素ガスを用いて還元する方法で、操作が簡便なため炭素-炭素二重結合にかぎらずカルボニル化合物の還元、窒素多重結合の還元、および水素化分解など広く用いられる合成法である。
　接触還元は目的によって、水素圧、触媒、反応温度などが選ばれる。水素圧は、常圧、中圧（2 - 10 kg/cm2）、高圧（50 - 300 kg/cm2）があり、それに適した装置がある。
　触媒は、白金、パラジウム、ニッケル、ロジウムなどが多く用いられる。白金触媒は微細なコロイド状にしたもの（Adams触媒）、ニッケル触媒はアルミニウム合金とし多孔質ニッケルとしたもの（Raney触媒）を用いる。パラジウムやロジウム触媒は活性炭、アルミナ、硫酸バリウム、炭酸カルシウムなどの担体の表面に吸着させたものを用いる。
　反応温度および溶媒は、一般に室温でアルコール溶媒が用いられる。加温したり液性を変えることにより目的にかなった還元を行う。
　接触還元は、立体障害の少ない側から接触吸着して水素化するので水素はシスに付加する。
b) 電子移動による還元
　この還元はアルコール中金属ナトリウム、アルカリ溶液中ナトリウムアマルガム Na-Mg、または液体アンモニア中金属ナトリウムやリチウム（Birch還元）によって行われる。
　この反応は、金属ナトリウムがイオン化するときに電子を放出し、これが化合物に付加して陰イオンとなる。この陰イオンにプロトンが結合する。
　
2) カルボニル化合物の還元
　アルデヒド、ケトン、エステルなどのカルボニル化合物を還元してアルコールに導くには、水素化アルミニウムリチウム LiAlH4、水素化ホウ素ナトリウム（またはリチウム）、NaBH4（またはLiBH4）を用いる。カルボン酸はエステルに誘導してから還元する。イミン（>C=N）は還元するとアミンになる。
　
　LiAlH4の水素を一部分アルコキシル基に変換し、LiAlH(OR)3とすると緩和な還元剤になる。これら水素化金属錯化合物による還元、分子中にオレフィン、ニトロ基、ハロゲンの結合があっても選択的にカルボニル基のみに進行する。
　
3) 窒素多重結合の還元
　ニトロ基をアミノ基にまで還元するには、接触還元または酸性中Zn, Fe, Snなどの金属を用いる。この金属による還元を中性溶液中で行うと、ヒドロキシルアミンやニトロソ化合物になる。アルカリ中では2分子が結合したアゾやヒドラゾ化合物になる。
　ニトリルはLiAlH4または白金触媒で還元すると第一アミンになる。
　
4) 水素化分解
　ベンジル位に結合したヒドロキシル基またはアミノ基は、酸性条件下で接触還元すると炭素-酸素および炭素-窒素結合が開裂して炭素-水素となる。
　炭素-ハロゲン結合は、ヒドロキシル基やアミノ基よりも容易に接触還元により水素化分解される。ヒドロキシル基を水素化分解したいときには、いったんハロゲンに官能基変換した後に行うと確実に水素化分解される。
　
　
　
◇ 酸触媒縮合
　酸触媒縮合による炭素-炭素結合生成反応は、プロトン酸またはルイス酸を加えることによって、もともと中性であった炭素を陽イオン性とし、これが電荷の多い炭素を求めて反応するものである。
　π電子や非結合電子対をもつ化合物は、プロトンまたはルイス酸と結合してその隣接炭素を陽イオン性にする。
1) オレフィンの反応
　プロトンがオレフィンと結合しカルボニウムイオンとなり、これがオレフィン（π電子をもち電子過剰である）と結合する。
　
2) アルコールの反応
　酸素の非結合電子対にプロトンが結合し、脱水するとカルボニウムイオンとなる。カルボニウムイオンが二重結合と共役するときは比較的安定であり、電荷の多い炭素と反応する。
　
3) カルボニル化合物の反応
　ホルムアルデヒドは酸によってカルボニウムイオンとなる。
　
a) Prins反応
　ホルムアルデヒドに濃硫酸を加え、生成したカルボニウムイオンはオレフィンと反応し、1,3-ジオキサンを与える。
　
b) クロロメチル化
　ホルムアルデヒドに塩化水素とZnCl2を加え生成したカルボニウムイオンは、求電子反応剤として芳香環水素と置換しクロロメチル誘導体を与える。
　
c) Mannich反応（マンニッヒ反応）
　酸触媒によってホルムアルデヒドから生成したカルボニウムイオンが、第一アミンまたは第二アミンと反応してインモニウムイオンを生じ、これが活性メチレン（またはメチル）と反応すると2-アミノケトンを与える。この反応をMannich反応（マンニッヒ反応）という。
　
　Mannich反応は電荷密度の大きい芳香環炭素とでも進行する。
　
　
　
4) カルボン酸の反応
　カルボン酸は酸触媒によってアシルイオンとなり、これが求弟子反応剤として芳香環水素と置換する。
　
　以下の化合物の生成も同様に考えることができる。
　
5) ハロゲン化合物の反応
　ハロゲン化アルキル（R-X）またはハロゲン化アシル（R-C(=O)-X）はルイス酸によってアルキルイオンまたはアシルイオンとなる。これが求電子反応剤として電荷密度の大きい芳香環水素と置換する。この反応はFriedel-Crafts反応という。
　

◇ 塩基触媒縮合
　塩基触媒縮合による炭素-炭素結合生成反応は、塩基を加えることによって水素をプロトンの形で脱離させ、もともと中性であった有機化合物の炭素を陰イオン性とし、これが陽イオン性の炭素を求めて反応するものである。

■ 高分子の合成 [4]
　多くの分子は、分子量が100か200程度の低分子か、分子量1万以上の高分子かのどちらかである（※ 各種医薬品の分子量の多くは1000どまり。ちなみに砂糖の分子量は342）。

・重合
　炭素間に二重結合を持つ低分子が、互いに二重結合部分で付加して高分子になる反応
・重合反応
　連鎖反応によって進行することが多く、開始剤の種類によって、ラジカル重合、イオン重合、（リビング重合）、配位重合など、多種類の反応がある
・官能基を持つ低い分子は、官能基を反応に使って高分子になる場合もある。このような反応として、重付加反応、重縮合反応、付加縮合重合などが知られている

◆ 連鎖反応
　連鎖反応は、反応活性種が低分子モノマーを攻撃してラジカル中間体を生成することによって進行する。したがって、連鎖反応には活性種が必要である。この活性種を生成する分子を反応開始剤という。開始剤が分裂して、活性点を持った活性種となることから反応が開始される。
　一般に重合反応は、連鎖反応によって進行することが多く、開始剤の種類によって、ラジカル重合、イオン重合、（リビング重合）、配位重合など、多種類の反応がある。
◇ ラジカル重合
　反応開始剤から発生した活性種の活性点がラジカル（不対電子を持った原子団）になっているとき、この反応をラジカル重合という。
・開始剤
　過酸無水物を熱分すると、ラジカル中間体、そして、フェニルラジカルになる。
　
・開始反応
　ラジカル中間体とスチレンが反応し、新たな中間体が生成する。
 
・成長反応
　新たに生成された中間体が、次のスチレンと反応する（次のスチレンを攻撃する）。このような反応が次々と進行し、多くのスチレンが重合した中間体ができる。
　
・停止反応
　反応は、再結合または不均化で停止する。再結合では中間体での互いの不対電子（図中の"・"）を使って共有結合し、二量体となる。不均化では（下の図の例では左から右へ）水素原子が移動して、二つの安定な分子になる。実際の反応では、再結合、不均化、両方が起きて反応が停止する。
　
◇ イオン重合

◇ リビング重合
　再結合や不均化による停止反応が起こらないため、反応によって生じた中間体活性種がいつまでも活性であり続ける（生き続ける、Living）重合反応をリビング重合という。
・開始反応
　ナフタレンに金属ナトリウムを作用させると、ナトリウムからナフタレンに電子が移動したナフタレンアニオンラジカルが生成する。
　
　ナフタレンアニオンラジカルにスチレンを作用させると、ナフタレンアニオンラジカルの電子がスチレンに移動して、アニオンラジカルが生成し、電子を与えたナフタレンアニオンラジカルはナフタレンに戻る。
  
・成長反応
　アニオンラジカルは、1分子中にアニオン部分とラジカル部分がある。アニオン部分は互いに静電反発するので結合できないが、ラジカル部分で結合するとジアニオン（ジ=2の意）となる。
　ジアニオンは分子の両端にアニオン部分があるので、ここを使ってアニオン重合が可能になる。その結果、多数のスチレンを重合したリビングポリスチレンとなる。これは両端にアニオン部分があるので活性な中間体となる。
　
　
・特徴
1) 高分子の長さが揃う
　リビング重合では、次アニオンの生成が非常に早く進行する。そのため、進行する重合反応の個数は加えた開始剤（ナフタレン）の濃度に比例する。反応はモノマー（スチレン）がなくなるまで進行する。重合度は下の式で表される。
　重合度 P = [M] / {(1/2) *[C] }
ここで、[M]はモノマー濃度、[C]は開始剤濃度である。この反応での全ての高分子の重合度は等しくなる。
2) コポリマーを作る
　全てのスチレンが反応した後、系にスチレン以外のモノマーを加えれば、その時点で、その新しいモノマーを元にポリマー化が再スタートする。つまり、この方法では、ブロックコポリマーを生成することができる。

◇ メリフィールド合成
・アミノ酸の修飾
　メリフィールド合成を行うために、アミノ酸の活性部分が余計な反応をしないように保護基を用いて余計な反応をブロックする。アミノ酸にHO-Lを作用させると、アミノ基部分を保護されたアミノ酸となる。
　
　（※ Lは保護基）
・ペプチド結合
　側鎖に塩素を持っている高分子にアミノ基部分を保護したアミノ酸を作用させると、側鎖にアミノ酸が結合した高分子となる。
　
　
　このアミノ酸部分の活性を取り戻すため、トリフルオロ酢酸を作用させて保護基Lを脱離する。
　
　これに別のアミノ酸（保護基L付き）を作用させると、アミノ酸部分が結合したものになる。
　
　以上の操作を連続すると、多数のアミノ酸が結合したポリマーとなる。
　
　最後にフッ素（HF）を作用させてアミノ酸部分をポリマーから外すと、多数のアミノ酸が結合したポリペプチドが生成する。
　
　この合成法の利点は、反応が不溶性の高分子上で行われるため、反応によってできた不純物を洗浄で容易に取り除くことができる点にある。この方法によってインシュリン（アミノ酸51個）などの複雑なポリペプチドの合成が可能となった。この方法の発明者メリフィールドはノーベル賞（1984年）を受賞した。

◇ 化学的耐熱性
・化学的耐熱性は分子鎖の結合解離エネルギーに大きく依存する。すなわち、主鎖に芳香族環を有する高分子や、ケイ素などの無機元素を含む高分子は化学的耐熱性が高いことが知られている。
・分子骨格を剛直化するためには、分子主鎖に芳香族環やかさ高い置換基を挿入したり、ポリイミドに見られるようにはしご状構造を導入すると良いことが知られている。また、高分子は結晶化すると分子間力が増大し、分子の運動が抑制されるために、耐熱性が向上する。液晶性高分子は、分子鎖が剛直であり、対称性が高いために、高い耐熱性を示す。
・立体規則性の高い高分子も高い結晶性を有している。

◇ 難燃性
・燃焼とは、化学結合の切断を伴った酸化反応である。したがって難燃性には化学結合が切れないように、結合エネルギーの大きな高分子を用いることが重要である。すなわち、芳香族環を主鎖に有した剛直な構造が重要で、スーパーエンプラの分類に入るポリイミドなどは難燃性の高い樹脂として知られている。
・高分子材料の耐熱性を上げるためには、分子骨格を剛直にすること、結晶化によって分子間力を強化すること、強化繊維などと複合化することなどが有効である。
　分子骨格を剛直化するためには、分子主鎖に芳香族環やかさ高い置換基を挿入したり、ポリイミドに見られるようにはしご状構造を導入するとよいことが知られている。また、高分子は結晶化すると分子間力が増大し、分子の運動が抑制されるために、耐熱性が向上する。液晶性高分子は、分子鎖が剛直であり、対称性が高いために、高い耐熱性を示す。
・高分子材料を燃えにくくするためには、高分子の分子構造を強固にするだけでなく、燃えにくくする薬剤の添加も有効である。ハロゲン化合物は、ポリマー分子の切断に伴って発生する活性ラジカルを効率よく補足する。また、水酸化マグネシウム Mg(OH)2は燃焼時に発生する熱を、分子の脱水反応により吸熱する。

◇ 耐候性
・高分子材料も酸素、オゾン、光（紫外線）、熱、水分の影響で化学的、物理的に分解し劣化する。いちばんの原因は酸素による酸化反応である。光や熱によってラジカルに分解した箇所に酸素が入り込み過酸化物となる。これら活性な結合は、その後の過酸化物のラジカル分解を経てどんどん開裂反応が進んでいく。
・第三級C-H結合や二重結合の隣（アリル位）のC-H結合は、結合切断によってその生成したラジカルが安定であるために切れやすい。そのため、そのような化学構造を有する高分子は耐候性が悪くなる。
・高分子の耐候性はその立体障害も大きく影響する。
・非晶性ポリマーにおいて、Tgの低いポリマーでは分子間相互作用が弱くなり、酸素分子の拡散が起こりやすくなり、耐候性が落ちることになる。逆に結晶性ポリマーの場合は、結晶域が酸素の拡散を防ぐため、その耐候性が高くなる。
・高分子材料の耐候性を上げるために、酸化防止剤や光安定化剤が添加される。酸化防止剤としては、フェノールやアミン類のようなラジカル捕捉剤が用いられ、酸化反応によって生じる活性ラジカルを捕捉することによって分解反応を抑えている。ベンゾフェノン類のような、紫外線を吸収することのできる光安定化剤も添加されている。

◇ 耐薬品性
・耐薬品性を支配する要因は、ポリマー分子と薬品が混ざり易いかどうかであり、溶解度パラメーターが近いと混ざり易くなり、耐薬品性は落ちることになる。もう一つの因子は、薬品分子がどれだけポリマー中を拡散できるかである。ポリマー鎖が水素結合能や結晶化のために動きにくかったり、架橋密度が高かったり、ポリマー分子鎖の剛直性が大きい場合、その運動性は大きく抑制され、薬品のポリマー鎖内での拡散が遅くなり耐薬品性が向上する。
・ポリアミドやポリエステルのような極性ポリマーの場合、結晶性であり、剛直構造のものが多く、非極性ポリマーに比べて耐薬品性は高いが、逆に極性溶媒や酸・アルカリ水溶液に対しては弱くなる。一方、ポリエチレンやポリスチレンなどの無極性ポリマーは、有機溶剤には弱いが酸やアルカリには耐性を示す。
・フッ素樹脂の代表であるポリテトラフルオロエチレン（PTEE）は典型的な無極性ポリマーではあるが、あまりにも極性が低いために、有機薬品とも親和性がなく、さらにはその高い結晶化度（95%）のために、工業樹脂の中では最高の耐薬品性を示すことで知られている。

◇ バリア特性
・気体のバリア特性を決めるものは、耐薬品性と同じく、気体分子の溶解性と拡散性である。液体の溶解性と同様、構造や性質が似たものどうしはよく溶け合う。酸素や二酸化炭素などのような非極性ガスは、非極性ポリマーを透過しやすい。逆に水蒸気は、ポリビニルアルコールやセルロース膜を容易に通過できる。拡散性においても、分子間水素結合、結晶化度、架橋密度など、ポリマー鎖が多きアン凝集エネルギーを持っている場合、さらには主鎖が剛直な構造をしている場合には、気体分子の拡散性が落ちる。
・保護フィルムの重要な役割は、酸化剤である酸素を透過させないことによって内容物の鮮度を守ることである。非極性な酸素分子を透過させないためには、極性が高く分子鎖が剛直で結晶化度の高いものがよい。代表的なものとしてポリビニルアルコールやセルロース、さらにはポリビニリデンクロライドなどの極性高分子フィルムが上げられる。
・水蒸気などの場合、材料の極性を落とすことで溶解度を下げればよい。しかしこのことは、高分子鎖間の凝集エネルギーを下げてしまい、分子鎖の動きが容易になる。そのため気体分子の拡散性が高まってしまい、バリア特性が低下してしまうことにもなる。

◇ 光透過性
・光の透過に一番大きな影響を与えているものは、ポリマー分子が持っている光吸収特性である。ポリマー分子の回転、振動や電子運動などによってそのエネルギー準位に対応した波長の光が吸収される。
・このような吸収の一つは、可視・紫外領域において生じる「電子遷移吸収」であり、ベンゼン環やカルボニル結合などを有する高分子材料では、その吸収が大きい。そのため、可視光域を対象とした光学材料には不向きである。
・もう一つは赤外領域で生じる「分子振動吸収」である。この領域の光は、C-H、O-H結合などの振動エネルギーに対応している。このような結合を必ず含んでいる有機系高分子材料では、その領域の光の透過に対して、大きな障害を及ぼすことが知られている。
・高分子材料による光の散乱も光の透過量を低下させる原因となる。散乱の要因となるものは、高分子中の分子鎖の結晶領域と非晶領域との界面であり、結晶性高分子であるポリエチレン（PE）やPPは、透明性の悪いポリマーとして知られている。
・他の要因として、ポリマー分子のミクロ環境下における密度揺らぎがある。この密度揺らぎは、ポリマー構造の分極率に大きく依存し、分極率が高いベンゼン環などを含有するポリマー（ポリスチレン PS）は散乱損失が大きいことが知られている。さらに、多成分系結晶性高分子は、ミクロな界面を持っていることから、散乱損失が大きくなる。
