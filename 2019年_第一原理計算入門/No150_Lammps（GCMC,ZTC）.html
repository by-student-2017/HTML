■ GCMC (ZTC with H2) calculation (https://lammps.sandia.gov/threads/msg58606.html)

◇ calculation
1. cfgというファイルを作成する
2. 下記に記載の3つのファイル（ztc_gcmc.lmp, H2.txt, ztc_openbabel.dat）を作成
3. mpiexec -np 1 lmp_mpi -in ztc_gcmc.lmp
4. ovito > (drag & drop) run.0.cfg > (click) "|>" (on the bottom right side)

※ 取りあえず動いたという程度で私も入力とその動作にいて不明（理解不足）な点が多い。
※ atom_style full でないと動作しない。atomicでは私の能力がなくて上手くいかなかった。
※ パラメータは温度が徐々に変化するように幾つかの値を試行錯誤している（当然ながら用いるポテンシャルによって変わるので、ポテンシャルが変わると大変）。速度はNISTのデータで√(計算する温度/298(K))倍としている。

variable    p equal press
variable    nugget equal 1.0e-8
variable      lambda equal 1.0

variable    pav equal f_ave[2]
variable    muexav equal f_ave[3]

thermo          1000

# ----- Output setting ----- 
dump            d2 all cfg 1000 cfg/gcmc_run.*.cfg mass type xs ys zs id type

unfix f1
