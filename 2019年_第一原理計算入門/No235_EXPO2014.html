■ Tutorial

□ Tutorial -1-
(http://www.ba.ic.cnr.it/softwareic/expo/tutorials/)
1. EXPO2014
2. File > Load Examples > cime.exp > Go
  (File > Load Examples > cime.exp > Edit > x(close) > Go)
3. [Next >] (on the top right side)
4. [Next >] (on the top right side) > OK
   (automatic peak search procedure)
5. Cell content: (C10H16N6S)4 > OK
6. [Next >] (on the top right side)
  (see background)
7. [Next >] (on the top right side)
  (search space group)
8. [Next >] (on the top right side)
  [P 1 21/a] > OK
  (Input file cime1.exp with new ...) OK
9. All [Next >] (on the top right side)
  (wait for a long time, you can get "end")
10. File > Export Structure > [(*.cif)] > OK
※ 「炭素または窒素原子種の割り当てにおけるいくつかの誤りは、JAVメニューからのグラフィックオプション 'Modify'を使うことによって容易に認識され、修正されることができます」と書かれているがそう簡単ではない気がする
※ *.expにXRDのデータのファイル名とX線の波数が記載されている場合
・cime.exp
-----
%structure cime
%job cimetidine (C10H16N6S) -- Synchrotron data
%initialize
%data
pattern cime.dat
wavelength 1.52904
synchrotron
%ntreor
%continue
-----

□ Tutorial -2-
(http://www.ba.ic.cnr.it/softwareic/expo/tutorials/)
1. EXPO2014
2. File > Load Examples > merca.exp > Go
  (File > Load Examples > merca.exp > Edit > x(close) > Go)
3. All [Next >] (on the top right side)
  (wait for a long time, you can get "end")
4. Solve > Explore Trials
5. [check] Select all new trials > Go
  (10 runs)
6. Quit
※ *.expにXRDのデータのファイル名とX線の波数だけでなく、cellやspace group, contentが記載されている場合
・merca.exp
------
%Structure merca
%Job 2-Mercaptobenzoic acid (C7 H6 O2 S)
%Data
Cell       7.885  5.976   14.949  90.0  100.48  90
SpaceGroup p 21/c
Content   (C7 H6 O2 S)4
Pattern    merca.xy
Wavelength 1.54056
%continue
------

□ Tutorial -3-
(http://www.ba.ic.cnr.it/softwareic/expo/tutorials/)
1. EXPO2014
2. File > Load Examples > paracetamol.exp > Go
  (File > Load Examples > paracetamol.exp > Edit > x(close) > Go)
3. Execute
4. Solutions
5. Quit
※ *.expにcontentの記載がない場合
・paracetamol.exp
------
%Structure paracetamol
%Initialize
%Job Paracetamol (C8H9NO2)
%Data
   Cell  7.100  9.380  11.708  90.0  97.42  90.0
   SpaceGroup p 21/n
   Pattern   paracetamol.xy
   Wavelength 1.54056
%fragment paracetamol.mol
%sannel
------
※ Z値（組成比）が不明な場合や占有率が明確でない場合はContentの記載のないこの方法（simulated annealing法）を選択すればよいように思う

■ Usage

1. Expo2014
2. File > Import Diffraction Pattern > XRDのデータ
  (Kα2をカットしていない場合は [check] Use wavelength2) > OK
3. File > Import Structure > *.cif > OK
4. Solve > Simulated Annealing
5. Execute

※ QualX2で用いたXRDのデータと表示された候補のcifを入力に入れればよい
※ 上記3の手順がなくてもCellやSpace Groupを指定すれば動くような気がするのだが、私が試したところでは上手くいかなかった。理由は不明。

・初期位相の数をうんと増やすと、その最初の位相の値がでたらめでもかなり高い頻度で成果が得られるというものです。古崎さんは位相を乱数で与えてこれにモンテカルロ法と名をつけたので、日本ではその呼び名が使われています。ウールフソンのプログラムはYZARC（逆から読むとcrazy）と呼ばれています。（省略）正しい初期値によるシグナルが悪い初期値によるノイズより大きい時に正解が得られるといってもよいでしょう。
・一般的な制限としては原子の数があります。200以上の独立な原子を含む構造が解けた例はありますが、普通には60位から上になると自動的とは言えないようです。それから原子間ベクトルの重なりの多い場合や、重原子法で偽の対称性が出てくるような場合も少し難しくなります。
・パターソンで解きにくいものは直接法でも解きにくい。例えば、ベンゼン環がたくさん集まった平面分子などでは分子内の原子間ベクトルの重なりが非常に多くなって解きにくくなります。また重原子が特別な位置にあると軽原子もその対称に従ってゴーストが出る場合もあります。
・もし原子数が50以下位でとけないとすれば、それは特別な場合だから、一般的な処方というのはありません。しかし一応整理してみましょう。

・タンパク質の構造解析の標準的な方法は重原子同形置換法ですが、この時の重原子の位置を決めるのに直接法が使われた例があります。またΣ2関係式を使って低分解能のデータから高分解能の構造を求めようとする試みもあります。Σ2式は位相だけでなくEそのものを決めることの出来る関係なのです。従って低分解能のデータしか取れない時でもベクトルの和を測定範囲外にとればそこのEが推定できることになるのです。これはタンパク質だけでなく電子顕微鏡の低分解能の写真から高分解能の像を得るための試みなどにも使われています。

  https://www.jstage.jst.go.jp/article/jcrsj/56/5/56_307/_pdf
・多くの有機化合物では、粉末回折データの空間分解能は、d値で1.5Å（λ=Cu Kα1で2θ≒60度）程度であり、原子位置を原子サイズレベルで特定できるだけの情報がない。その分解能の低さゆえ、回折強度データのフーリエ合成によって得られる電子密度ピークは、それぞれの周辺にフーリエ変換の打ち切り効果によるリップルが存在する。このリップルが隣接した電子密度ピークの位置や強度に誤差を与える。RBM（Resolution Bias Modification）はこの誤差を補正するものである。
・最新のEXPO2014では、ディフォルトで直接法で得られた電子密度マップの補正をRBMによって行うようになっている。分子量300〜400程度の有機化合物であれば、直接法+RBMに要する時間は20秒程度もしくはそれ以下である。
・有機化合物の場合、結合を作る原子同士の距離はおよそ1.0〜2.0Åの範囲内にある。仮に、フーリエ合成によって得られた22つの電子密度ピーク位置の距離が2.5Åだったとすると、これらの2つの原子は結合を作らず、2つの原子を橋渡しするように、中間にもう1つ別の原子が存在することが予想される。
・粉末結晶構造解析の初期構造決定のステップで特に有機化合物を解析する場合に最もよく用いられる手法は「実空間法」である。実空間法とは、測定データと同じ回折パターンが得られるように、単位格子（正しくは非対称単位）中で分子の位置、向き、ねじれ角を、大域最適化アルゴリズムを用いて探索する方法である。
・7%以下（R値）が論文として報告されている結果の7割以上を占める。
・観測されるピークの数が少なければ無理に高分解能の光学系を使用する必要はない。
・格子の長い有機化合物ならば、低角側にピークが密集するため、指数付けのためにピーク分解能を良くする必要がある一方、高角側でピークはほとんど観測されなくなるため、集光ビーム光学系（CB）が推奨される。
・格子の短い無機化合物ならば、低角側のピーク本数は少ないため、さほどピーク分解能を必要としない代わりに、高角側までピークが観測されるため、平行ビーム光学系（PB）が推奨される。
・一般に、材料物性は温度や湿度によって変化することが多く、その原因は結晶構造の変化であることが多い。
