EPW (Electron-phonon Wannier) calculation
--------------------------------------------------------------------------------

■ Installaton

1. cd $HOME/q-e-qe-6.3
2. make epw

Reference
[1] EPW: http://epw.org.uk/Main/DownloadAndInstall
--------------------------------------------------------------------------------
■ Usage (examples)

◇ MgB2
1. cd $HOME/q-e-qe-6.3
2. cd EPW
3. cd examples
4. cd mgb2
5. cd phonons
6. mpirun -np 2 $HOME/q-e-qe-6.3/bin/pw.x -npool 2 < scf.in
7. mpirun -np 2 $HOME/q-e-qe-6.3/bin/ph.x -npool 2 < ph.in
8. python pp.py
9. cd ..
10. cd epw
11. mpirun -np 2 $HOME/q-e-qe-6.3/bin/pw.x -npool 2 < scf.in
12. mpirun -np 2 $HOME/q-e-qe-6.3/bin/pw.x -npool 2 < nscf.in
13. mpirun -np 2 $HOME/q-e-qe-6.3/bin/epw.x -npool 2 < epw.in
※ 他の例も同様に行えばよい。私のPCでは上記6)で計算は走っているがなかなか終わらない。そして、クラッシュした。そのため、ph.inとepw.inにおいて、nq1=2, nq2=2, nq3=2 としたら、2 CPU, 2 GBでepwの途中まで動作した。恐らく私のPCではメモリ数が足らない（CPU数も時間短縮のために必要）。

◇ Pb with SOC (spin orbit coupling)
1. cd $HOME/q-e-qe-6.3
2. cd EPW
3. cd examples
4. cd pb
5. cd wSOC
6. cd phonons
7. mpirun -np 4 $HOME/q-e-qe-6.3/bin/pw.x -npool 4 < scf.in
8. mpirun -np 4 $HOME/q-e-qe-6.3/bin/ph.x -npool 4 < ph.in
9. python pp.py
10. cd ..
11. cd epw
12. mpirun -np 4 $HOME/q-e-qe-6.3/bin/pw.x -npool 4 < scf.in
13. mpirun -np 4 $HOME/q-e-qe-6.3/bin/pw.x -npool 4 < nscf.in
14. mpirun -np 4 $HOME/q-e-qe-6.3/bin/epw.x -npool 4 < epw.in
※ Ziman resistivity formula で resistivity を計算できる。文献[2]に具体的な方法が記載されている。
※ 上記8) でクラッシュが起きた。私のPCスペックが悪くて生じているのか？
