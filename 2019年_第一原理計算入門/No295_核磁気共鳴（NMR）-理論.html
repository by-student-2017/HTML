◆ 理論的背景 [6]

◇ 共鳴現象
　ほとんどの原子核は角運動量p（核スピン）と、それにともなう磁気モーメントμ=γ*pをもっている。磁気回転比γ（γ<0なら磁気モーメントと核スピンは反対方向を向く）はそれぞれの核に固有の定数である。量子力学からそれぞれ次式で表される。
　p = {h/(2*π)}*√{I*(I+1)}
　μ = γ * {h/(2*π)}*√{I*(I+1)}
I値は原子核の核角運動量量子数または核スピン量子数で、整数または半整数をとる。
　I = 0, 1/2, 3/2, 5/2, ...,
　均一な静磁場B0中では、角運動量ベクトルPがB0ベクトルに対して特別な角度をとることができる（方向の量子化）。このpの成分は次式で与えられる。
　PB = m*{h/(2*π)}
方向あるいは磁気量子数mの値は
　m = +I, I-1, I-2, ..., -I+1, -I
となり、トータルで（2*I+1）個の、いわゆる核Zeeman準位に分裂して、さまざまなエネルギーを持つ。これは次式で与えられる。
　Em = -μB・B0 = -γ・PB・B0 = -γ・m・{h/(2*π)}・B0
　（m = +I, ..., -I）
例えば、水素核において、プロトンはI=1/2なので、m=±1/2となる。
-----------
・プロトンのエネルギー準位
+(1/2)・γ・{h/(2*π)}・B0 : m=-1/2,  μのベクトルは-B0方向を向いて歳差運動
 -(1/2)・γ・{h/(2*π)}・B0 : m=+1/2, μのベクトルは+B0方向を向いて歳差運動
-----------
　より低いエネルギー状態では、μはLarmor周波数ν0=|γ|・B0/(2*π)で、B0の周りを歳差運動する。より高いエネルギー状態では、逆に-B0の周りを歳差運動する（もしEmを正に定義するなら、上記の磁気量子数mは入れ替えなければならない）。
　熱平衡のもとで、1H核はBoltzmann分布に従う。次式で示されるエネルギー差
　ΔE = γ・{h/(2*π)}・B0 
は、きわめて小さく。占有数は、低いエネルギー状態のほうがごくわずかに過剰となる。占有数の比は次式で与えられる。
　N(m=-1/2) / N(m=+1/2) = exp{-ΔE/(k*T)}
ΔEのエネルギーの量子を照射すると、スピンの反転が誘起される。占有数に差があるために、吸収Aが優先する。共鳴条件は次式のように定義される。
　h*ν = ΔE =  γ・{h/(2*π)}・B0 
　1.4 Tの磁場の強さのとき、プロトンの共鳴周波数ν=f(B0)は約60 MHzになる。これはλ=5 mのラジオ波に相当する。共鳴条件が満たされると、二つの核Zeeman準位間の占有数の差は、吸収によって非常に速く消失する。逆のプロセス、すなわち緩和が十分に起こらないとき、系は飽和しているという。
　上から下のエネルギー準位へ核の遷移が起こって放出されたエネルギーは、熱エネルギーのかたちで周りに吸収される（スピン-格子緩和）。この緩和過程は速度定数1/T1で起こる。このプロセスでは、磁場を変化させる方向と磁場の方向とが同じなので、T1は縦緩和時間と呼ばれる。横向きの磁化も、磁気モーメント同士の相互作用によって時間とともに変化する（スピン-スピン緩和）。この速度を横緩和時間といい、T2の記号で表す。
　上に示したように、磁気モーメントがぜろでない（μ≠0）ことが、核磁気共鳴の必須条件である（磁気モーメントμ=0の核は、偶数の質量数と偶数の原子番号をもつ）。大きなスピン量子数の核は核四極子モーメントをもつので、スペクトルは複雑になる（シグナルが幅広くなる）。したがって、I=1/2の場合は非常に有利である。
　有機化学において最も重要な核は、1H, 7Li, 11B, 13C, 15N, 17O, 19F, 29Si, 31P, 77Seである。そのうち、13C, 15N, 29Si, 77Seは天然存在比が小さいということが不利な点となっている。
　シグナルの強度は磁気モーメントの3乗に比例するから、大きな磁気モーメントをもつ核のほうが研究しやすい。（省略）
　磁場が一定のもとでは、各種の核の共鳴周波数νは直接γ値と関連している。γ（1H）/γ（13C）=26.75/6.73=3.975なので、例えば、1Hの共鳴周波数が90 MHzのとき、13Cの共鳴周波数は90/3.975で約22.6 MHzとなる。

◇ NMRにおける検出感度 [11]
　核スピンをI、共鳴周波数をν0、試料中の核スピン濃度をNとすると、一定磁場の下で検出感度S（Sensitivity）は、
　S = I*(I+1)*ν03*N
に比例する。実際に観測されるNMR吸収線の面積強度が、このSに比例すると考えればよい。Iとν0は、核の磁気モーメントによって決まるので、測定対象核が決まれば一義的に決まってしまう。また、Nは試料溶液の濃度と対象核の天然存在比によって決まるので、検出感度を高めるために天然存在比の低い核は、特に濃縮（enrichment）をしない限り、試料濃度を高くするとか、観測周波数（磁場）を高くすることも必要となる。I=(1/2)である1Hや13C核等は、この考え方で対応すれば良い。
　多核種NMRの場合、感度がそれほど良くないことから大抵の場合、何度も掃引して積算したNMRスペクトルを測定する場合が多い。この時、どの程度積算すう必要があるかの目安があると都合が良い。そこで、13C核の感度を基準にして比較することが良くある。
　R = [{I*(I+1)}/{(1/2)*(1/2+1)}]*(ν0/ν13C)3
これは同濃度の核のときの比で、相対感度R（relative sensitivity）と呼ばれる。
　一方、天然存在比（α）の小さい核種の場合は、それをも考慮した総合相対感度（relative receptivity）と呼ばれるR'、すなわち
　R' =  [{I*(I+1)}/{(1/2)*(1/2+1)}]*(ν0/ν13C)3*(α/α13C)3
を用いることが多い。
　一方、I > (1/2)の核には、核四極子モーメント（Q）が存在し、速い核四極子緩和によって信号の線幅が広がり、見掛け上、検出感度が低下する。すなわち、吸収線の高さが減少する。これは、注目する核の周りの電場勾配の対称性が低くなることから起こるものである。したがって、質量数の大きい核で、核四極子モーメントが大きく、しかも、電場勾配の対称性の低い場合には信号の検出までに多数回の積算を要することになる。したがって、この線幅が広がる要因をも考慮した因子を線幅（広幅化）因子（line width factor: LW）として定義し、観測のしやすさの程度を知ると便利である。
　LW =[(2*I+3)*Q2]/[I2*(2*I+1)]
　このLWが小さいほど、線幅の狭いNMR吸収線を与えて観測しやすいことになる。LWは比較的線幅の狭いNMR吸収線を与える2H核のLWに対する比として与えた。これを用いるとR'が小さくLWがえ大きい核種の測定は極めて困難であると言える。例えば、197Au核などがこれにあたる。

◆ 化学シフト [6]
　ある核の正確な共鳴周波数は、核の置かれた環境により異なってくる。核における有効磁場強度Beffは、誘起磁場によってB0とは異なった値となる。
　Beff = B0 - σ*B0
共鳴条件は無次元の遮蔽定数σを用いると次式のように表される。
　ν = {γ/(2*π)}*B0*(1-σ)
　核が強く遮蔽される、すなわちσが大きいと、Beffは小さくなる。つまり周波数を一定にすると、核の共鳴のために外部磁場B0をより強くしなければならないことを意味する。同様にB0を一定にすれば、遮蔽が増すに従い共鳴周波数νは減少する。

　測定核の化学シフトが核の周りの環境に敏感であることは、有機化合物の構造を決定するうえで非常に重要である。遮蔽定数σは共鳴の位置を決定し、次の3つの項から成り立っている。
　σ = σdia + σpara + σ'
反磁性項σdiaは、核周辺の電子雲中で、外部磁場により誘起された磁場に対応している。核の近傍の電子は、遠方の電子に比べて強い遮蔽効果をもつ。常磁性項σparaは磁場中のp電子の励磁に対応する。これは反磁性遮蔽とは逆の効果を示す。水素のようにs軌道のみが存在する場合は、σdiaのみを考える。13Cのようにより大きな核になると、常磁性項の寄与が重要になってくる。σ'は近傍の基の影響のことである。これは場合によって核の感じる磁場を増加させたり、減少させたりする。σは分子間の影響も受け、このときはさらにσmedium項を付け加える必要がある。
・化学シフトの起源 [9]
　分子が磁場中に置かれると、分子内で反磁性（diamagnetic）および常磁性（paramagnetic）の2種類の電流が誘起される（反磁性物質とは、磁場中で外部磁場と反対方向の磁化を誘起する物質のことである。また常磁性物質とは、磁場中で外部磁場をさらに増強する磁化を誘起する物質のことである）。反磁性および常磁性電流は互いに反対方向に流れ、核に対してそれぞれ遮蔽および反遮蔽の効果をおよぼす。したがって、遮蔽定数は反磁性および常磁性の寄与の和として表す必要がある（上のσについての式）。σdia > 0, σpara < 0である。（省略）他の条件が同じである場合、エネルギーの低い利器状態の方がエネルギーの高い励起状態よりも常磁性反遮蔽に大きく寄与する。理論的には、常磁性寄与σdiaは平均的な励起エネルギーΔに近似的に反比例する（※ このΔは、px→pyだけでなく、配位子場による分裂幅Δでも成り立つ）。またσparaは核とその周囲の電子との距離（R）にも関係している。よく知られているように、電流が流れる小さな電線の輪の中心部の磁場は、輪の半径Rの3乗に反比例する。σparaについても同様な関係が成り立つと考えられるので、非常におおまかではあるが下の近似式が導かれる。
　σpara ∝ -(1/Δ)*<1/R3>
　ここで、<>は局所的な電子分布の平均値である。軸対称をもつ局所電子分布（例えば、アセチレン分子のπ電子）をもつ分子の場合、外部磁場が対称軸に平行であるときσparaの値はゼロとなる。同様に、原子の化学シフトに常磁性の寄与はない。常磁性および反磁性電流とそれらが作り出す化学シフトの理論は、AtkinsとFriedman（2011）を参照してほしい。

◇ よく使われる化学シフトの式-定性的な常磁性項σpの式 [11]
A. 理論式
　雑賀とSliochterは、19F NMRのシフトを解析するために、次の式を用い、それぞれの項を見積もった。
　σ = σD + σp + σ0
ここでのσDは、問題とするF原子による反磁性シフト項を表し、σpはその常磁性シフト項を、そしてσ0は他の原子からのシフトへの寄与を表すものである。重原子の場合、σ0は無視できるほど小さいので、σD+σpを考えればよい。彼らの評価によれば、19F NMRシフト範囲のわずか1%がσDの寄与である。結局、19F NMRシフトに支配的寄与をするのはσpであると結論した。その後、19Fだけでなく、1H以外の核の遮蔽定数にも、この常磁性シフト項σpの寄与が支配的であることが明らかにされた。
　JamesonとGutowskyは、LCAO MO法およびValence Bond法のそれぞれから、σpを表す式を導き出した。
　σp = {2*e2*h2/(2*π)2}/{3*ΔE*m2*c2}*[<r-3>p*Pu+<r-3>d*Du]
ここで、<r-3>の添字のp, dはp軌道, d軌道を、Pu, Duは観測核上のそれぞれのp, d軌道における価電子の配置の不均衡状態を表す。したがって、原子の結合状態によって変化する。例えば、Puの最大は2（2つのp軌道が満たされ、1つが空、もしくは、1つが満たされ、2つが空の場合）だある。ΔEは電子の平均励起エネルギーであり、<r-3>はpまたはd電子の波動関数について取ったr-3（r: 軌道半径）の平均値である。この式はs電子（重原子）核の化学シフトの解釈によく用いられる。この式から類似した化合物間の化学シフトの要因を次のようにまとめることができる
1) 励起状態
　電子の平均励起エネルギー（ΔE）が小さい場合（いかなる励起状態でもよいと言うわけではなく、磁気双極子に関して許容遷移となる励起状態）。1電子の遷移に関する励起状態ではn→π*、n→σ*、π→σ*、σ→π*などである。例えば、px→py, dxy→dx2-y2である。遷移金属錯体ではd6電子配置の八面体構造のものではdxy→dx2-y2、d8電子配置の平面型構造のものではdxy→dx2-y2やdyz,xz→dx2-y2などである。
2) 軌道の広がり
　<r-3>p,dに示されるpまたはd軌道の空間的広がりが小さい場合によく効いてくる。すなわち、電子の核の周りでの運動が核に近いほど効果がある。周期律表の同周期では原子番号が大きくなるほど増加し、同族でも原子番号の順に大きくなることが伺える。これは電子雲膨張効果（nephelauxetic effect）として分光学的に整理されている。
3) 電子配置
　原子価電子の原子の周りのアンバランスな配置の程度に影響される。すなわち、核近傍の電子分布の非対称性の程度に支配される。従って、完全に球対称分布したような原子では、上式のσpの[]内の値は0となる。電子が完全に詰まった内殻のp, d軌道の場合は全体として球対称となるのでこの項に効いてこない。
　金属核NMR化学シフトの場合にはd電子の寄与が極めて大きいことからd軌道項単独の式で、
　δ = -σp = k*<r-3>d*(ΔE-1)*Cd2
と表されることが多い。kは定数、ΔEは最も寄与するまたは平均的な励起状態への遷移エネルギー、CdはそのときのLCAO分子軌道のd軌道の係数である。こうして、
　1. 金属の周りの対称性による配位子場分裂エネルギーの大きさ（Δ）
　2. 電子雲膨張効果（<r-3>d）

◇ 化学シフトの応用例 [9]
　例として、アミノ酸であるヒスチジンの1H NMRスペクトルのpH依存性をみてみる。pHを変化させると、イミダゾール基のH2とH4の共鳴周波数は、酸性溶液で安定に存在するプロトン化型HisH+、および高いpH溶液で存在する電気的に中性である脱プロトン化型Hisとの間でなだらかな変化を見せる。ある特定のpHにおける化学シフトは、両極限型の化学シフト（δHisH+とδHis）の加重平均として観測される。

◇ 金属核NMRの特徴 [11]
　多核種のNMRは一般的に次のような特徴を持っている。
1. 有機化合物の分野で日常的に利用されている13C核を基準にすると、NMR信号の検出が容易である核が多く、測定に要する時間も短くなる
2. 対象となる核は、質量数が大きく、原子核の周りにpやd電子等を持つので、化学シフト範囲が広くなり、試料は必ずしも溶液状態でなくても測定ができる
3. 核スピンIが1/2より大きい（I>1/2）ものが多い
4. I > 1/2の核は核四極子モーメントと持ち、これによって緩和が短くなり、線幅の広い信号を与える
4. 緩和の短い核の測定には、繰り返しの速いパルスが使用できるので、測定時間も短くなる
　多核種のNMRは、それの観測装置を備えたNMR分光計を比較的容易に入手できるようになってきたこともあって、およそ化学と名の付く種々の分野に適用されている。また、この方法を駆使して得られた研究成果をまとめた総説もいくつか出版されている。原理的には、核スピンを持つ原子核すべてが、NMR法の対象となりうるため、周期律表全体の原子（元素）核がカバーされる。
　金属核のNMRは無機金属錯体、有機金属錯体、金属クラスター、金属タンパク質を対象とする。金属イオンに注目するところから、吸収線が1本ないし数本が観測されるだけの単純なNMRスペクトルを与える。したがって、帰属は比較的容易である。観測はこんなな場合が多い。遷移金属イオンの場合は様々な価数を取ることから電子状態が変わる。このとき奇数の電子を持つ場合、偶数でも電子の配置により電子対を取らない場合（高スピン配置）には常磁性となり、吸収線は著しく広がって観測できない。したがって、観測が容易な核種でも系の電子状態によって観測できないことがある。そして、何よりも59Coのような一部の核種を除いてNMR測定の歴史が浅いためデータの蓄積が十分ではなく、観測者が開発しつつ応用するという側面を持つ。3つの情報量（化学シフト、スピン-スピン結合定数、緩和時間）の中で比較的データが蓄積されている化学シフトを例に特徴を述べる。
　1,2族の金属イオンはs, p軌道に電子が完全に詰まった電子配置をとる。この場合、空間的に方向性を持たないため、化学シフトは極めて小さい（数ppmから数十ppm）。つまり、1H型の化学シフトを取ると考えられる。
　一方、遷移金属の化学シフトに関しては、これまで上記に示した式から、
　σ = σD + σp = σD -k*<r-3>d*(ΔE-1)*Cd2
　　〜 -k*<r-3>d*(ΔE-1)*Cd2
を用いて解釈しうることを述べた。すなわち、d軌道に電子を持つ金属（3から10族、dn: n<10）の場合、常磁性項（ΔE-1）が寄与する化学シフトを与えることが多い。この場合は、錯体の配位子場分裂エネルギーとの関連で極めて理解しやすい。一方、d軌道がすべて詰まった後周期遷移金属（11, 12族）の場合はどのような要因が大きく寄与しているのかを考えることは意味がある。中辻らは非経験的分子軌道法を用いてCu+, Ag+, Zn2+ , Cd2+の直接化学シフトの計算を試み、実験値と良い対応を示す満足すべき結果を得た（文献[11]には関連するデータについての表がある）。この内容をよくみると、さらに重要なことがいくつか理解される。
-----
1. CuからCdまで遮蔽定数の絶対値に寄与するのは値の大きい反磁性項σDであるが、錯体の構造に敏感に変化しているのは常磁性項σpである。
　我々は基準物質からの化学シフトに興味があり、絶対値にあるわけではない。しがたって、σpが重要な意味を持つことがこれより理解され、この項に注目すればよいことになる。
2. σpの内容をさらに調べると、大きな変化は価電子によるものであることが一目で理解される。すなわち、金属イオンの価電子に注目し、常磁性項の変化を研究すれば良いことになる。
-----
　さらに詳しく調べると、Cu+では錯体を作ると3d10の電子配置を取っているのではなく、配位子へのπ逆供与により3d軌道から配位子へ一部の電子が移動し、その結果3dn(n<10)配置となり、他の遷移金属同様、d軌道が寄与する化学シフトを示したものと考えられる。実際、完全に電子が詰まったときに予想される化学シフトに比べて、500 ppmもの化学シフト範囲を持つこともこれにより理解される。配位子の電子受容性が大きくなるほど低磁場シフトすることが実験的に観測されている。Zn2+, Cd2+では3d10, 4d10配置を持つ。この場合、π逆供与はないが空の4p, 5p軌道に配位子から一部電子が流れ込み、これによりp軌道支配による化学シフトが起こったものと解釈される。
　以上のように、金属核の化学シフトは金属特有の化学的性質を決めるd軌道が大きく関わっていること、また、非金属に近い12族（Zn, Cd）ではp軌道が効いてくることが理解され、NMRパラメータ（化学シフト）の「金属の化学」への貢献度が大きいことを示している。

　また、ハロゲンと希ガス元素を除くと、含まれている原子の数が少なく、よって化学シフトの範囲も比較的狭いものが多い。例えば、13C核が330 ppm、 14N, 15N核が1030 ppm、31P核が820 ppm、17O核が1650 ppm、33S核が1010 ppmとなる。
　非金属元素の共通の性質として、希ガス元素を除いて、反応性の高いものが多い点があげられる。例えば、酸素およびフッ素はヘリウム、ネオン、アルゴン以外の全ての元素と、直接の化合物を作るほど反応性が高い。そのために、一般的に言って、非金属元素は化合物中に含まれている割合が多い。特に、水素、炭素、窒素、酸素、硫黄を始めとして、フッ素、塩素、臭素などの元素は数多くの化合物中に構成元素として含まれている。それゆえに、これらの核種のNMRはその適用範囲が広い。
　非金属元素に属する核種のうちで、33S（天然存在比, 0.75%）、15N（同、0.366%）および17O（同、0.038%）は天然存在比が全て1%以下と低く、感度の点で不利であるが、これらの元素の重要性のために、測定例は比較的多い。水素および炭素も非金属元素に含まれるが、1Hおよび13C核のNMRに関しては、数多くの優れた成書が既にある。
　一般的に言えば、非金属同士の結合は共有結合によっているものが多い。共有結合からできている分子の中には、炭化水素などの炭素化合物をその良い例として、数多くの原子が連なった複雑な構造を持った化合物も多い。こう言った複雑な分子の化学構造決定には、非金属元素のNMRが大きな役割を果たす。その理由は、共有結合で結ばれた原子の間には、スピン-スピン結合相互作用（spin-spin coupling interaction）が存在しているためである。これはある核磁化の位置に、他の核磁化が作る磁場を感じるものである。その意味では磁気双極子-双極子相互作用（magnetic dipole-dipole interaction）と似ているが、磁気双極子-双極子相互作用は空間を通じて起こるのに対して、スピン-スピン結合相互作用は、結合を通じて起こる。すなわち、ある核磁化の分極が結合電子の分極を引き起こし、これが共有結合を通じて順に伝わっていくことによって起こる。デカップリング（decoupling）法や二次元相関（2D-correlation）NMRを用いて、どの原子とどの原子の間にこの相互作用が存在しているかを調べることによって、原子の結合順序を知ることができる。ただし、スピン-スピン結合相互作用による信号の分裂は比較的小さいため、核四極子モーメントを持った核種では、対称性のよい場合などを除いて、線幅に隠されてしまうことが多い。このため、I=1/2の核種について測定されることが多い。
　ケイ素化合物の化学シフトのd軌道まで入れたab initio計算によれば、シラン（SiH4）およびシリコンテトラフルオリド（SiF4）の反磁性シフトは、それぞれ873.7 ppmおよび863.4 ppmであり、また常磁性シフトはそれぞれ-355.7 ppmおよび-330.2 ppmであり、和としての総シフトはそれぞれ518.0 ppmおよび536.2 ppmである。金属元素の遮蔽定数がほとんど常磁性項によって決まることと比較すると、反磁性項の寄与が大きいことがわかる。

　Chesnutによれば、ファンデルワールス引力が電子軌道の拡大を招き、遮蔽効果を及ぼすことから逆に、ここで問題になっている分子内ファンデルワールス反発は電子軌道の収縮を招き、常磁性項σpの<r-3>を通して、反遮蔽効果を及ぼしていると考えられている。
　アリルケトン類、アロマティックアルデヒド類、アリルカルボン酸などの化合物の集団について、トーション角（torsion angle）と17O NMR核の化学シフトとの間に直線関係が見いだされている。Boykinらはトーション角による化学シフトの変化の少なくとも一部は、酸素原子上の電荷密度の変化により、常磁性遮蔽が変化することによると考えている。
　Kwang-Tingらによって、13種類のα,α,α-トリフルオロアセトフェノンについて、17O NMR化学シフトと有機化合物の置換基定数σ+との間に
　δ=24.6*σ+ + 556.3
という直線関係（相関係数0.998）が得られている。二重置換パラメータ法（dual substituent parameter method）を用いて、C(2)位を置換した9種類のチオキサントンなどの、17O NMR化学シフトの変化分について
　Δδ=9.63*σ1 + 4.09*σR(+)
という関係式が得られている。

◆ NMRスペクトルへの効果 [9]
　化学シフトだけの議論からすると、H13CO2-の1Hおよび13Cスペクトルにはそれぞれ1本のシグナルが現われると予想するかもしれない。しかし、実際に両スペクトルを測定すると、化学シフトの両脇に対称的に位置する2本線すなわちダブレットが観測される。分裂幅（この場合 195 Hz）は1H-13Cスピン-スピン相互作用の強さであり、両スペクトルで同じ値である。
　1Hの共鳴線が2本に分かれるのは、13Cの磁気モーメントが結合している1Hに対して小さな局所磁場を生み出すからである。すなわち、13Cがm=+1/2状態のとき（ここではC↑と表示）は、外部磁場に逆方向の局所磁場を発生するため、1Hの共鳴線は右側へシフトする。逆にm=-1/2状態のとき（C↓）は、局所磁場が外部磁場に加算され1Hの共鳴線は左側へシフトする。これを化学シフトの用語で表現すると、C↑炭素は1Hを遮蔽し、C↓炭素は反遮蔽する。それゆえ1Hのダブレット（doublet）のそれぞれの成分は、H13CO2-の2種類の（C↑とC↓を含む別個の）分子によるシグナルである。13Cの両スピン間のエネルギー差は熱エネルギーkBTと比較すると非常に小さいので、2種類の分子の存在比はほぼ等しい。したがって、ダブレットは等しい強度の2本の線となる。1Hとのカップリングによる13Cのダブレットも同様に説明できる。
　H13CO2-におけるスピン-スピン相互作用により二つのスピンが互いに逆平行であるH↑C↓とH↓C↑は安定化（※ 外部磁場のみにおけるHの分裂である低いエネルギー側｛安定側｝にあるH↑と高いエネルギー側｛不安定側｝にあるH↓に対して、エネルギー的に安定化する方｛低いエネルギー側｝へシフト）され、互いに平行であるH↑C↑とH↓C↓は不安定化（※ 外部磁場のみにおけるHの分裂であるH↑とH↓に対して、エネルギー的に不安定化する方｛高いエネルギー側｝へシフト）される（※ 観測されるスペクトルでは、選択則Δm=±1から矢印{↑や↓}の変化が1つだけが許される遷移、例えば、H↑C↓ → H↓C↓、H↑C↑ → H↓C↑では許容遷移となり、高い遷移確率でラジオ波を吸収する）。それゆえ、プロトンの二つのエネルギー準位（m=±1/2）はそれぞれ二つに分裂し、各状態のエネルギーは13Cと1Hスピン同士の相対的な配置によって決まる。すなわち、1H NMR遷移に注目すると、C↑をもつ分子における遷移（H↑C → H↓C↑）の方が遷移エネルギーが小さい。なぜならば、1Hと13Cのスピンが平行である不安定な（エネルギーが高い）基底状態（H↑C↑）から、互いのスピンが逆平行である安定な励起状態（H↓C↑）への遷移だからである。これとは逆に、C↓をもつ分子における1Hの遷移（H↑C↓ → H↓C↓）のエネルギーはより大きい（安定基底状態から不安定励起状態への遷移であるから）。

　上記の異種核（heteronuclear）間（1H-13C）相互作用の例は、J-カップリングの性質を示す為に便宜的に取り上げたものである。同種核（homonuclear）間、例えば、二つのプロトン間のカップリングもまったく同様な分裂を与える。
　H13CO2-で説明したスピン-スピンカップリングの特性は、二つの相互作用している核AとX（必ずしもスピン1/2核でなくてもよい）のエネルギー準位に関する次のような単純な式として一般化できる。
　E(mA, mX) = mAhν0A + mxhν0X + hJAXmAmX
ただし、mAとmXは二つの核の磁気量子数、ν0Aとν0Xはそれぞれのラーモア周波数である。JAXは相互作用の強さであり、スピン-スピンカップリング定数またはJ-カップリング定数と呼ばれる。カップリング定数は周波数単位（ヘルツ： Hz）で表され、正または負の値である。逆平行配向の核スピン対の方がエネルギー的に安定であれば（H13CO2-がその例）JAX>0であり、平行配向のスピンの方が低エネルギーであればJAX<0である。
　選択律ΔmA=±1と上の式を組み合わせると、AとXの相互作用はスピンAのラーモア周波数（ν0A）をJAXmXだけ変化させていることがわかる。より一般的には、共鳴周波数の方程式は以下のように表される。ただし右辺のΣは、Aに対して無視できない大きさのJ-カップリングを示すすべてのスピン（K）についての総和である。
　νA = ν0A + ΣK≠A JAXmK
　片方のスピンだけが磁気量子数を変える遷移（ΔmA=±1またはΔmX=±1）のみが許容されることに注意してほしい。mAとmXが同時に変化する遷移、すなわち、A↑X↑ ←→ A↓X↓とA↑X↓ ←→ A↓X↑は禁制である。
　カップリング定数の符号はスペクトルの形に影響を与えない。例えば、JAXを正から負に変えてもそれぞれのダブレット内で成分の右左が変わるだけである。
　この単純な考え方で、ほとんどの分子のNMRスペクトルにおけるスピン-スピンカップリングの様子を予想できる。（ただし、例外はある。例えば、下に示す「強いカップリングと等価なスピン」）

◆ スピン-スピンカップリング [6]
　NMRスペクトルで観測されるシグナルは、しばしば微細な構造を示す。それぞれのシグナルの数に対応して、一重線（シングレット）、二重線（ダブレット）、三重線（トリプレット）、四重線（カルテット）、多重線（マルチプレット）などと表現する。この微細構造の原因は、近傍の磁気モーメントを持つ核との相互作用による。スピン-スピンカップリングは同じ種類の核（同核）の間で生じたり、異なる種類の核（異核）の間で生じたりする。このことは、核Aのスピンの方向がカップリング相手の核Xの周辺の局部的な磁場に影響を与えることを意味する（また逆に、XからAに対しても同様である）。二つの核AとXにおいてどちらも核スピンが1/2の場合は、二つの核の組み合わせ可能な4つの配向に対応して、原理的には4つのエネルギー準位が存在する。スピン-スピン相互作用がないとき（J=0）、AとXはそれぞれ同じエネルギーの二つの吸収を示す。この縮退はカップリングJによる影響を受ける。両方の核が外部磁場B0に対して同じスピン方向であり、かつそのエネルギー準位が上昇するとき、Jを正と定義する（※ AとXのスピンの方向が同じ時、J>0ならエネルギー準位が増加する。逆も同様。J<0ならこれらの関係が逆になる）。逆方向スピンのスピン状態では、エネルギー準位の低下が起こる。反対のことが、Jが負の場合にもあてはまる。いずれの場合もAとXは二重線に分裂する。
　カップリングの大きさはカップリング定数Jで与えられる。この場合は直接にXあるいはAの線の分裂から測ることができる。プロトン-プロトンカップリングではだいたい-20 Hzから+20 Hzの間となる。他の核では、カップリング定数がもっと大きな値になることもある。例えば、アセチレンの13C-13Cのカップリング定数は171.5 Hzだり、C-Hのカップリングは 250 Hzである。
　特に重要なのは、カップリング定数は外部磁場B0と無関係であるということである。スペクトル中に2本線があるとき、2つの可能性が考えられる。
・第一に、異なる化学シフトをもつ2つのカップリングしていない核の一重線
・第二に、カップリングしている他方の核とA-X系を形成するもう一方の核から生じる二重線
である。これらの2つの可能性は、異なった周波数で2種類のスペクトルを測定すれば、容易に見分けることができる。分裂が同じように残ればカップリングによるものであり、より高い周波数での測定で分裂幅が増加すれば（Hz単位で）、2つの線は異なる化学シフトの2つの一重線である（δスケールにおいては、分裂は観測周波数には無関係である）。
　二つの核AとXの間でのカップリングは、溶液中では一般に分子内の結合を通じて起こる（スカラーカップリング）。
　一つの結合が間にあるとき、カップリングは直接カップリング1Jと表され、例えば1H - 19F, 1H - 13Cなどがある。この言葉を直接的双極子-双極子カップリングと混同してはならない。これは配向相（液晶状態や固体）で現われる空間を通じた効果である。等方性流体（液体）中では、これらのノンスカラーカップリングは分子の熱運動により平均化されてゼロになる。
　二つまたは三つの結合を間にはさむ場合のカップリングは、それぞれジェミナルカップリング2J、ビシナルカップリング3Jという。
　AとXの間の結合の数が増えると、JH-Hは一般に減少する。重い核とのカップリングでは、しばしば|J|値が結合の数に従っては減少せず、中間的最大値をもつ場合が多い。遠隔カップリングnJを検出するには、スペクトルの分解能がきわめて重要である。
　カップリングパターンは、カップリングする核の数の増加に従ってその複雑さが増す。分子中の全ての核iの化学シフトδiが、全ての可能な核の組み合わせのカップリング定数nJijとともにわかっていれば、NMRスペクトルを計算で求めることができる。逆に、単純なスペクトルからδとJ値を直接決めることもできる。
　上で述べたAX系よりも複雑な系のカップリング現象を取り扱うために、系統的なスピン系の命名をする必要がある。
　化学シフトが等価なn個の核は、偶然の一致または化学的等価性により、An系を形作る。さらにm個の核群が存在し、かつ化学シフトが等価ならば、スピン系はAnBm、AnMmまたはAnXmと記述する。これらは共鳴周波数νAからの離れ具合（わずかか、中程度か、大きくか）で決まる。この表記法には重要な制限があり、それはどのAiBj（AiMjまたはAiXj）（i=1, ..., n; j=1, ..., m）の組み合わせでもカップリングが同じであるということである。
　化学シフトが等価な核Aiが、近傍の基とただ一つのスピン-スピン相互作用をもつとき、それらは時期的に等価であるという。同じことが核B, M, Xにも適用できる（スピン-スピンカップリングは相互的であるので、AnBm系のAスピンが磁気的に等価であり、Bスピンが等価でないということはありえない）。二つ以上のスピンの組み合わせがある場合、例えば、AnBmX1系では、磁気的等価性の定義はただ一つのJAB, JAXおよびJBXをもつということになる。化学シフトが等価であることは磁気的等価性のための必要条件であって、十分条件ではない。一方、磁気的等価性は化学シフト等価の十分条件であって、必要条件ではない。分かりやすい二つの例をあげる。
　ジフルオロメタンは二つの化学シフト等価なH核とF核をもっている。それぞれのカップリング定数2J(H, F)は同じである。したがって、二つのH核と二つのF核は化学的にも磁気的にも等価である。CH2F2はA2X2スピン系である。1,1-ジフルオロエチレンでもH核とF核は化学的に等価である。しかし、2種類の異なるカップリング定数3J（H, F）が存在する（HがFと、ZおよびEの関係になるので）。（省略）
　カップリングパターンを理解するには、磁気的に等価な核間のスピン-スピンカップリングはスペクトルに影響を与えないことに注意する必要がある（しかし、実際はカップリングをもっている）。
　例えば、メタン、エタン、エチレン、アセチレン、ベンゼンなどは、1Hスペクトルで一重線の吸収を与える（1Hと13Cのカップリングは、13Cの天然存在比が1.1%と低いので、通常の測定では必ずしも観測されない）。これらは化学的等価性により、プロトンが化学シフト等価となっている。しかし、偶然同じ位置にシフトが現われたとしても分裂は観測されない。
　例を示すと、3-シアノプロピオン酸メチルは、化学的に非等価な二つのメチレンがδ=2.68に分裂せず、一重線を与える。
　二つの磁気的に等価な核の組をもつAnXmやAnMmのスピン系は、|νA - νM|がJA,Mの約10倍以上あると解析しやすくなる。線の数、すなわちシグナルの多重度は、次のようになる。
　A: m*2*IX+ 1
　B: n*2*IA+ 1
IXとIAは核XとAのスピンである。IX=IA=1/2のときは一次スペクトルが得られる。
　A部分には（m+1）本の線
　B部分には（n+1）本の線
が現われる。
　臭化エチル（H3C-CH2-Br）の1Hスペクトルを考えてみよう。これはA3X2スピン系である。3個の化学的にも磁気的にも等価なメチルプロトンの位置は、2個のメチレン基の核スピンによる影響を受ける。これらは外部磁場に対して、両方が平行、両方が逆平行、あるいは一方が平行で一方が逆平行となる場合がある。したがって4つの（このうち逆スピンをもつ二つは縮退するが）エネルギー準位ができる。それぞれのスピン状態における存在確率が等しいことから、メチレンプロトンによるカップリングはメチルプロトンに三重線を生み出す。その強度は1:2:1である。化学シフトはシグナルの重心で与えられ、この場合は1.67 ppmとなる。まったく同様に、メチレンプロトンの現われる位置はメチルプロトンのスピン-スピンカップリングにより影響を受ける。メチル基の3個のプロトンでは8種類のスピンの組み合わせが存在する。最も低いエネルギーをもつものはトータルスピンm=3/2で、最も高いものはm=-3/2である。その間に3個の縮退したm=+1/2と3個のm=-1/2がある。したがってメチレン基は、メチル基とのカップリングから強度比1:3:3:1の四重線を示す。重心はδ=3.43 ppmである。三重線と四重線の分裂の大きさはカップリング定数Jに等しい。

　ある分子がAnBm（AとBが同核）のスピン系をもち、そして、|νA-νB|/JABが10以下のとき、一次スペクトルの有効性は失われる。二つの三重線の線の強度が正確には1:2:1でないことがわかる。カップリング相手（β-CH2基）に近いほうの線が、遠いほうの線より強度が大きくなる。この現象はルーフ効果と呼ばれる。次式が成り立つので、
　|νA-νM|/JAM < |νA-νM|/JAM
ルーフ効果はメチレン基の三重線よりもメチル基の三重線のほうで際立っている。β-メチレン基ではルーフ効果が左右のα-メチレン基とメチル基にあるはずだが、同じ理由によりメチル三重線に対して観測される。複雑なスペクトルではルーフ効果がカップリングの相手を見つける手がかりとなる。このような二次スペクトルがΔν/J>10でも起こるのは、核群が化学的には等価だが、磁気的には非等価である場合である（たとえばAA'XX'）。一般的にスペクトルはゼロ次（一重線のみ）、一次、二次と分類される。
　最も単純な二次スペクトルはAB系から得られる。（省略）
　可能な3スピン系A3, A2X, A2M, A2B, AMX, ABX, ABCのなかで、A2B, ABX, ABCは一次解析の取り扱いができない。しばしば弱い遷移で通常のスペクトルには現われない部分もあるが、多重線のとりうる最大値はそれぞれ9, 14, 15である（スピン系AXX'またはABB'は、二つの化学的に非等価な核XまたはBが偶然等価なとき、すなわち化学シフトが同じ時にのみ生じる）。
　3つまたはそれ以上のスピン系を厳密に分析するには、NMRの専門書を参照されたい。二次の度合いが少ない場合にのみ、近似的に一次の規則に基づいて分析できる。その例は、フェニルオキシラン（スチレンオキシド）の1H NMRである。これはABM系である。これを仮にAMX系として取り扱うと、得られるパラメータ（化学シフトとカップリング定数）は、ABX系として正確に分析したものから得られるパラメータからわずかにずれるだけである。
　3つの異なるカップリング定数をもつAMX系は、それぞれの核A, M, Xに対して4本線を示す（理想的な場合、12本の線すべてが同じ強度となる）。カップリング定数は、それぞれの線の周波数から直接求めることができる。（省略）
　NMRシグナルの分裂は装置の分解能だけでなく、測定周波数（磁場強度に比例する）にも依存する。例えば、Δν/Jはスペクトルが一次であるか二次であるかを決定するが、この値は60 MHz（1.41 T）から360 MHz（8.45 T）になると6倍増加する。つまり360 MHzでは一次であったのが、60 MHzでは二次になる。これはスペクトルがかなり違った外観になることを意味している。化学的に等価で磁気的に非等価な核をもつスピン系は、磁場強度にかかわらず二次のスペクトルを与える。もちろん、このとき磁場が変わると、スペクトルの外観も変化する。

◆ スピン-スピン結合定数 [11]
　核間の磁気的相互作用は、直接空間および化学結合を通して起こる。この相互作用により核のスピンに関する情報が近接している核に伝わり、その核の共鳴周波数に変化をもたらす。この相互作用の大きさを表す量としてスピン-スピン結合定数Jが用いられ、単位はHz（ヘルツ）で表される。
◇ スピン-スピン結合定数の理論的背景
　核Nは静磁場H0との強い相互作用に加えて、隣接する他の核N'とも弱い相互作用をする。この相互作用の大きさを表す定数をスピン-スピン結合定数（nuclear spin coupling constant）と呼び、Jで表す。RamseyやMcConnellらによって核N, N'間のJNN'は、
　JNN' = JNNNN' + JNONN' + JNENN' + JFCNN'
と与えられる。ここで、
　JNNNN'は核スピンNおよびN'を磁石としたときに直接空間を通しての相互作用（核-核双極子； nuclear spin-nuclear spin dipolar）
　JNONN'は核スピンNと電子の軌道運動により生ずる磁場との相互作用（スピン-軌道； nuclear spin-elecron orbital motion）
　JNENN'は核スピンNおよび電子スピンSを磁石としたときに直接空間を通しての相互作用（核スピン-電子スピン双極子； nuclear spin-electron spin dipolar）
　JFCNN'は電子スピンが核スピンに直接接触することにより生ずるいわゆるフェルミコンタクト相互作用（Fermi contact）
である。（省略、文献[11]には模式図が描かれている）
　JNN'は化学シフトで述べたように、
1. 電子雲膨張効果（<r-3>p,d; JNONN', JNENN'）
2. 低いエネルギー（一重項励起1ΔE-1はJNONN'、三重項励起3ΔE-1はJNENN', JFCNN'）
3. 結合におけるs軌道の寄与（核上の電子スピン密度、<sA|δ|sA>; JFCNN'）
4. 電荷密度や結合次数（それぞれの項）
5. 核N, N'の空間的距離（r; JNNNN'）
に依存する。（簡単な近似式を表に示すと次のようになる）

◇ スピン-スピン結合定数に及ぼす各種効果 [11]
A. 結合の混成
　電子スピンが核スピンに直接接触することにより生ずるいわゆるフェルミコンタクト相互作用JFCが支配的な場合は、p, d軌道の混成の程度が増加するほどスピン-スピン結合定数の値は小さくなる。これはs軌道が関与する程度（s性の割合）が小さくなるからである。例えば、1JC=Cでは、C-C (sp3) < C=C (sp2) < C≡C (sp)となる。多核種の場合には、タングステン錯体(CO)5W=CRR', X(CO)4W≡CRでは、それぞれ1J(W=C)=87.9から102.5 Hz, 1J(W≡C)=168.5から197.5 Hzとなる。
　金属錯体では立体構造により混成の様子が変わるため結合定数が変わる。八面体（d2sp3）< 三角両錐（dsp3）< 四画平面（dsp2）となる。例えば、Rh錯体では八面体1J(Rh(V)-P) < 平面 1J(Rh(T)-P)である。
B. 置換基の電気陰性度
　M-X結合においてX（ドナー原子）の置換基（例えば、NR3で置換基はR）の電気陰性度（電荷を引っ張る力の程度）を増加させると1JMXは増加するという極めてよい相関関係がある。残念ながらMX系では行なわれていないがCH系では理論計算によりC-H結合のs性が増大することが示されている。一方、M-31P系で金属Mに配位する配位子の電気陰性度を変えてもP上の電気陰性度を変えたときほどの顕著な効果はない。
C. 配位数
　一般的に配位数が増えると1Jの値が減少する。橋かけ配位している原子との1Jは、末端で配位しているものとのそれより小さい。
D. 酸化状態
　一般的な法則を見出すまでには至っていない。高い酸化状態になると1Jは小さくなる傾向がある。
E. 立体構造
　1) アキシアル、エカトリアル配位の1Jについては、M-Xの結合距離が長くなると結合定数は当然小さくなる。
　2) cis, trans配位：強いトランス影響を持つものがXY結合のtrans位にあるときは1JXYは小さくなる。このcis, transの効果も結合距離の大小で解釈できる。

◇ 強いカップリングと等価なスピン（ルーフ効果）[9]
　ここでm=+(1/2)とm=-(1/2)の状態をそれぞれスピンα、βと命名すると、弱くカップリングする二つのプロトンに対する四つの状態は下記のように表される。
　αAαB, αAβB, βAαB, βAβB
四つの許容遷移（※ 片方のスピンだけが磁気量子数を変える遷移〔一般的な選択則であるΔmA=±1またはΔmX=±1が許容のため〕のみが許容される）は、
　αAαB ←→ αAβB, αAαB ←→ βAαB, αAβB ←→ βAβB, βAαB ←→ βAβB
である。これらは等しい強度の別個の周波数をもつので、よく見慣れた二組のダブレットを示すのである。一方、δνが小さくなるにつれて、上記の四つのうち、中央の二つの遷移間の周波数差がJと同程度の大きさになるため、両者の混合（mix）という現象が生じる。すなわち、純粋なαAβBとβAαBではなく、それらはαAβBとβAαBの線形結合（linear combination）となる。その結果として、遷移確率（transition probability）および遷移周波数が変化するので、内側の2本の線の許容度がより大きく（より強く）なり、外側の2本の線は許容度が小さく（より弱く）なる。この効果はδν/Jが小さくなるに従ってより顕著になる。最終的にδν=0の状態では、外側の線を与える遷移は禁制され、内側の2本は同じ周波数を持つ。
　これをより具体的に記述するために、量子力学的な取り扱いを以下のように要約する〔Hore, Jones, Wimperis (2015)〕。四つの状態ΨjとそれらのエネルギーEjは、
　Ψ1 = αAαB, E1/h = +ν + (1/4)*J
　Ψ2 = cosΧ*αAβB + sinΧ*βAαB, E2/h = +(1/2)*C - (1/4)*J
　Ψ3 = -sinΧ*αAβB + cosΧ*βAαB, E3/h = -(1/2)*C - (1/4)*J
　Ψ4 = βAβB, E4/h = -ν + (1/4)*J
として表される。ただし、
　ν = (1/2)*(ν0A + ν0B), C = √(J2+(δν)2), tan(2*Χ)=J/δν
である。
　上述のように、Ψ2とΨ3はαAβBとβAαBの線形結合であり、それらのエネルギーは、角度Χにより定められるカップリングの強さ（J/δν)に依存する。しかし、Ψ1とΨ4は互いのエネルギーが十分離れており、またΨ2とΨ3のエネルギーとも異なるのでカップリングの強さには影響されない。

強いカップリングに起因する線強度のゆがみは「ルーフ効果（roof effect）」と呼ばれることがある。強いカップリングの場合でも、ダブレットの分裂幅はJと等しいが、化学シフトはもはやダブレットの中心と一致しない。

　予測できると思うが、三つ以上のスピンが「強い」カップリングをすると、非常に複雑な結果になる。パターンは著しくひずみ、解析するのが困難になる。なぜならば、遷移確率が変化するため、本来禁制であるはずの遷移が起こり、よほど綿密な解析をしない限り化学シフトとカップリング定数を決めることすらできない。このような場合、高磁場分光計を使用すると問題を解決できることがある。すなわち、磁場強度B0を変えてもカップリング定数は変化しないが、δνはB0に比例するため、強いカップリングをしているスピン系が高磁場下では弱いカップリング系になることが多いからである。例えば、化学シフトが0.2 ppm離れている二つのプロトンがJ=6 Hzでカップリングしている場合、60 MHzの分光計で測定すると顕著な屋根型効果が見られるが（δν/J=2）、600 MHzの装置（δν/J=20）ではそのような変形は観測されない。

◇ 線幅 [6]
　半値幅である線幅は、Heisenbergの不確定性原理で予測される”固有値”よりもかなり大きくなる。線幅は磁場の不均一性だけによるものではなく、むしろ遠隔カップリングや核の緩和時間T1, T2に依存する。
　試料中に14Nのように電気四重極モーメントをもつ核、あるいは常磁性化合物があると、スピン-格子緩和時間T1は減少し、線は幅広くなる（常磁性化合物自身はこの理由からNMR測定には適していない）。同様に、スピン-スピン緩和時間T2が溶液の粘度の上昇によって減少すると、線幅は広がる。特殊な効果としては、交換現象により引き起こされる線の広がりがある。この場合、分子間プロセスと分子内プロセスのどちらであるかを区別しなければならない。前者にはカルボン酸、アルコール、アミンのプロトン移動がある。含水（5%以下の水濃度）メタノールの1H NMRを0℃で測定すると、二つのOHシグナルが観測される。水濃度が高くなると、プロトン移動は促進される。初めにOHシグナルの広がりが観測され、交換の速い水濃度範囲になると、OHシグナルが平均化した位置に観測される。線が幅広くなるので、OHプロトンとメチル基とのカップリングが消える。純粋なメタノールでは、温度を0℃から+10℃に上げることにより後者の効果は観測される（加温によりプロトン交換が促進される）。

◇ 緩和時間 [11]

A. 双極子-双極子緩和
　あたかも小さな棒磁石として振る舞う2つの核IとSが空間を通して直接相互作用（双極子相互作用）することにより緩和が促進される。この場合、空間的にIとSが近いほど、この効果は大きい（核間距離をrとするとr6に逆比例）。さらに分子の回転の速さ（時間の定数、相関時間τで表す）にも影響される。
1. 分子の回転（molecular tunbling, τr）は分子内（intramolecular）および分子間（intermolecular）での核I, S相互作用に影響する
2. 分子の並進（translational diffusion, τdif）運動は、分子間のI, S相互作用に影響する
　双方が寄与する場合は、速い相関時間が重要となる（τc-1=τr-1+τdif-1で与えられるτcを用いる）。例えば、13C-1Hの場合の13C核の緩和時間T1は、
　1/T1DD = c*Σ {({h/(2*π)}2γI2γS2)/rIS6}*τc
と表される。ここでcは定数、和はこの緩和に寄与する全ての1H核について取り、rISは核間距離、τcは分子の運動に関する相関時間、γは核の磁石としての性質を表す核磁気回転比である。例えば、炭化水素の13C緩和は炭素原子に結合している1H核からの寄与が圧倒的に大きいため、その個数をn、rCH=1.1*10-8 cmとすると、
　1/T1DD = n*2.0325*1010*τc
となる。これよりT1を測定するとτcの値が得られる。
　通常の天然存在比の大きい核の原子どうしが結合している場合は、この効果は大きい。例えば、1H, 31P核などである。共鳴周波数の小さい核種の場合は磁石が小さい（γ値が小さい）ため相互作用が弱く有効な緩和機構にならないときがある。
　この相互作用を利用して核オーバーハウザー効果（nuclear overhauser effect, NOE）などの測定が行なわれる。
B. 核四極子緩和

　1/T1Q = 1/T2Q = 4π2*{3(2I+3)/(40I2(2I-1))}*(1+η2/3)*(e2qQ/h)2*τc
が成り立つ。ここでηは電場勾配についての非対称パラメーター、qは電場勾配である。核四極子結合定数（nuclear quadruple coupling constant）と呼ばれるΧ
　Χ = (e2qQ)/h

1. 小さな分子であるほど（回転が速い）
2. 試料の粘度が低く、温度が高いほど
3. 観測核の周りの対称性が良いほど

狭くなる。もちろん、核スピン)依存性もある。例えば、(2I+3)/{I2(2I-1)}項の比では、I=7/2ではI=1に比べて、f(I=7/2)/f(I=1)=0.027も狭い線幅を与える。このことから線幅因子（line width factor）として、次式

C. スピン回転緩和
　この緩和は核スピンと分子全体の磁気モーメントの回転によって生じる磁場との相互作用により生じ、
　1/T1SR = {(2πIkT)/h2}*Ceff2*τj
と書ける。Iは分子の慣性モーメント、kはボルツマン定数、Tは絶対温度、Ceffは有効スピン-回転結合定数、τjは分子の衝突に関する相関時間。この機構は小さい分子中のスピンI=1/2の核や溶液中におけるスピンI=1/2を持つ重原子核の緩和に重要である。
D. 化学シフトの異方性による緩和
　遮蔽定数がテンソル量である。溶液中で観測される遮蔽定数（化学シフト）は、その対角項の平均値[σαν=(1/3)*(σx+σy+σz)]である。緩和の場合は、このテンソル成分が直接寄与する。そして分子運動（molecular tumbling）により緩和を有効に引き起こすことができる。
　1/T1CSA = (1/15)*γ2*H02*(σ⊥-σ//)2*2τc/(1+ω2τc2)
　1/T2CSA = (1/90)*γ2*H02*(σ⊥-σ//)2*{6τc/(1+ω2τc2) + 8τc}
ここでH0は外部静磁場、σ⊥とσ//は軸対称系の対称軸に垂直および平行な成分である。この緩和機構は大きな化学シフトの異方性（|σ⊥-σ//|が大きい）を持つ系や磁場H0の大きな装置での測定のときに聞いてくる。例えば、Me2TlNO3重水溶液の205Tl吸収線のΔν1/2は、H0=9.4 Tでは140 Hzに対して、1.41 Tで4 Hzになる（T=Tesla, 1 T = 1*104 Gauss）。
E. スカラー緩和
　観測核Iが他の核Sと結合している系で、IとSとの間の化学交換の寿命をτex、スピン-スピン結合定数をJISとすると、
　1/τex >> JIS
のとき、第一種のスカラー緩和が次式で示される。
　1/T1SC = (8π2/3)*JIS2*s(s+1)*[1/{1+(ωI-ωS)2*τex2}]
　1/T2SC = (4π2/3)*JIS2*s(s+1)*[τex + τex/{1+(ωI-ωS)2*τex2}]
ここで、sは核Sのスピン量子数、ωIとωSはラーモア周波数（rad）である。
　逆に、1/τex << JISのときは第二種のスカラー緩和と呼ばれている。このときはτexの代わりにこの緩和に有効なSの緩和時間TISが使われている。このスカラー緩和が支配的なケースは希である。（ωI-ωS）が極めて小さいか、JISが非常に大きいときに可能性がある。Sが電子スピンのときはIとSの超微細結合による極めて大きくなり、この機構が支配的である（常磁性化合物）。
　一般には緩和時間は、例えば1/T1では、
　1/T1 = 1/T1DD + 1/T1Q + 1/T1CSA + 1/T1SC
と言うように加成則が成り立つ。この中で特に大きい項があれば、その核の緩和にはその項が支配的であると言える。
　これらの緩和機構の中で1/T1DDによるものは応用範囲が広い。核オーバーハウザー効果（NOE）と呼ばれるものが関係している。今、IとSの2つのスピンの双極子-双極子相互作用が支配的であったとすると運動が速いときには（(2πν)2τc2 << 1）、核オーバーハウザー効果因子（NOEF）は、
　NOEF = η = [T1DD/T1Total]*[γS/2γI]
となる。緩和が完全にT1DDのみのときには、
　η = γS/2γI
となる。このηは実験的にシグナル面積より求められる。すなわち、
　NOEF+1 = NOEのときの面積/NOE無しの時の面積
このときに気をつける必要があるのは15N核のようにγN<0のケースである。γH/2γN=-4.9であるから、20%（=T1DD/T1Total）が双極子であればNOEF=-1となり、実際のシグナル強度に関与する量は0となる。

◇ 化学交換 [9]
□ 対称な二つのサイト間の交換
・遅い交換
　遅い交換とは、個々の共鳴線が交換による広幅化（broadening）を見せるが、まだそれぞれが周波数位置ν0aとν0bに観測される状態に対応する。この速度範囲内では、交換による線幅（Hz）の増大分は単に次式で表される。
　Δν = k / π = 1/ (π*τ)
　すなわち、（遅い交換の領域内では）交換が速くなるほど線の幅が広くなる。この効果の起源は寿命広幅化（lifetime broadening）または不確定性広幅化（uncertainty broadening）と呼ばれる。後者はハイゼンベルグの不確定性原理に多少関係があるからである。この原理によれば、有限の寿命をもつある状態のエネルギーを厳密に決めることはできない。したがって、ある状態の寿命が短ければ短いほど、そのエネルギーの不確定さが増大する。「ぼやけた」エネルギー状態間の遷移が、共鳴線の広幅線の広幅化の原因となる。電子スペクトル〔自然放射の速度が非常に速い（励起状態の寿命が短い）〕の自然線幅が、回転または振動スペクトルの線幅より大きい現象はこれが原因である。またマイクロ波スペクトルにおいて、高圧下で測定すると（励起状態が、速い衝突により素早く基底状態へ失活する）広幅化が観測される現象の原因でもある。
・中間速度の交換（intermediate exchange）
　二つの共鳴線がちょうど1本の幅広い線に融合するとき、すなわち2本のピークの間に存在する「谷」が消える時点では、次式が成り立つ。ただし、kmergeは2本の線が融合するときの速度定数である。
　kmerge = (π*δ*ν)/√2 ≒ 2.2*δ*ν
　k > kmergeのときは共鳴周波数の中間の位置に1本の線が現われ、k < kmergeのときは2本の分離した線が現われるであろう。しかし交換速度が中間である領域では、NMRシグナルが非常に幅広くなり、シグナル対雑音比（S/N比）が小さな、感度の悪いスペクトル中ではシグナルが観測できないこともある。
・速い交換
　もう一方の極限は、2本の線が融合し、両者の平均ラーモア周波数の位置に1本の幅広い線を示す場合で、これを「速い交換」と呼ぶ。この領域では、化学交換により追加される線幅は、
　Δν = {π*(δν)2}/(2*k) = {π*(δν)2*τ}/2
で表される。以前と同じくδν=ν0a-ν0bである。遅い交換の場合は二つのサイト間の行き交いにより2本の共鳴線が幅広くなった。それとは対照的に速い交換の場合は、速度定数が大きくなるにつれて、両サイトの環境がより効率的に平均化されるため、結果的に生じる1本の共鳴線の幅がより狭くなる。これは、両スピンが経験する実行局所磁場が、二つの交換サイトにおける局所磁場の平均値になることを意味している。
　二つのサイトに由来する信号を分離して検出するためには、両者の間に十分大きな位相差（phase difference）がなければならない。その位相差が、例えば180°である場合(1/2)*(δν)-1の時間が必要である。この時間内で交換が必要である。この時間内で交換が起こると、位相差の蓄積（位相の差が徐々に大きくなること）が中断される。同時に、スピンがそれぞれの周波数を交換するので、δνの符号が反転し、位相差は減衰しはじめる。その後再び、スピンがそれぞれの周波数へ跳躍して復帰するので、位相差は再び増大し始める。すなわち、位相差は周波数の反転と共にランダム歩行する。したがって、速い交換とは、(1/2)*(δν)-1時間経過後の位相差が180°よりはるかに小さくなるため、事実上、二つのシグナルが非常に接近した周波数を所有する状態のことである。言葉を変えると、二つのサイト間の行き来が、位相差の積算を形成するよりも遥かに速い場合、その交換は周波数差（δν）をなくしてしまう。それぞれのスピンが各サイトで50%ずつ時間を過ごしているため、速い交換の領域内では、共鳴線は両周波数の中間点(ν0a+ν0b)/2に現われる。
・NMR時間尺度
　位相差について上記の議論で示したとおり、交換が「遅い」「中間」「速い」の区別は、周波数差δνに対する交換速度定数kの相対的な大きさで決まる。すなわち、化学交換という現象のの時間尺度（timescale）は(δν)-1に支配される。したがって、NMRの時間尺度で遅い、速いと表現される過程は、交換しているサイト間のNMR周波数差と比較して遅いか速いかという観点から論じられるべきである。δνの値は〜10 kHzを上回ることはほとんどなく、通常はさらに小さな値であるので、NMRで研究できるのは比較的遅い（s 〜 μs）過程のみである。
　例をあげると、100 MHzの分光計では化学シフト差が1 ppmである二つのプロトンの共鳴線は、k=220 s-1のとき、すなわち二つのサイトの平均寿命が4.5 msのときに融合する。しかし、500 MHzの分光計では、同じ二つの共鳴線が融合するためには、速度定数として1100 s-1（910 μsの寿命）が必要である。
　13C NMRにおいては、共鳴周波数が非常に広範囲に渡っているので、比較的速い過程を研究できる。13Cの磁気回転比は1Hの(1/4)であるが、化学シフトは1H の約20倍のppm範囲、したがってラーモア周波数として約5倍以上の範囲をカバーしているからである。
□ 非対称な二つのサイト間の交換
　等しい占有数をもつ二つのサイト間の交換は明らかに特殊例であり、より一般的で普通に見られる例は、任意の異なる濃度をもつ二つのサイト間の交換である。（省略）
　二つのサイトの部分占有率をそれぞれpAとpBとし、（pA+pB=1）、行きと帰りの一次速度係数をkAおよびkBとする。
　A ―kA→ B
　B ←kB― A
　平衡状態にあるとき、pA*kA = pB*kBである。また二つのサイトの平均寿命はτA=1/kAおよびτB=1/kBである。
　遅い交換の領域では、寿命広幅化は下式で表される。
　ΔνA = kA / π = 1 / (π*τA)
　ΔνB = kB / π = 1 / (π*τB)
　ここで注意を要するのは、二つの周波数が寿命の差の違いにより、不均等な影響を受けることである（低濃度の化学種の方が短い寿命であるので、より大きな広幅化を示す）。一方、速い交換の場合は、両者の加重平均の周波数位置に1本の共鳴線を示す。
　νavg = pA*ν0a + pB*ν0b
　また線幅は
　Δν = (4*π*pA*pB*(Δν)2)/(kA+kB)
である。
　これら全ての式において、pA=pB=(1/2)およびkA=kB=kとすると、対称的な二つのサイト間の交換の式へ誘導されることはいうまでもない。（省略）結果として、νavgはAとBの相対量に依存するので、例えば温度を変えるとνavgも変化する。
 （省略）
　二つのサイト間の交換について確立された原理は、さらに多サイト間の交換過程にも応用できる。当然のことながら、スペクトルはかなり複雑となり、多くの場合、解析にはコンピュータが必要となる。典型的なやり方は、まず反応過程を想定して、各段階の速度定数を変化させて計算を行い、シミュレーションスペクトルを実験スペクトルと一致させる方法である。
　化学交換以外にも、共鳴周波数の平均化を引き起こす過程がある。最も重要な例は、溶液中の分子の回転に双極子相互作用および四極子相互作用による分裂が平均化される場合である。これらの平均化の条件は速い交換の場合と同じである。すなわち、共鳴周波数の交換速度は周波数差と比較して大きくなければならない。

◇ スピン緩和 [9]
　化学シフト、スピン-スピンカップリング、および化学交換は、溶液NMRスペクトルの形を決める主要な因子である。一方、やや捉えにくいが見過ごすことができない別の因子がある。それらは二つの緩和過程、すなわちスピン-格子緩和（spin-lattice relaxation）およびスピン-スピン緩和（spin-spin relaxation）である。これらはパルスによる励起などの撹乱を受けながら、核スピンが平衡状態へ戻る過程である。
・スピン-格子緩和
　この緩和機構により核スピン群は準位間を移動することが可能となり、最終的にボルツマン分布に従う各占有数に落ち着く。占有数が平衡状態に達する過程で放出されるエネルギーは、周囲の環境（伝統的に格子（lattice）と呼ばれる）へ発散する。
　このような占有数の変化は、時間T1（スピン-格子緩和時間）によって特徴付けられる。スピン(1/2)核の集団に対して、緩和過程が対数的（実際に多くの場合そうであるが）であると仮定すると、m=±(1/2)間のスピン数の差は次に従って増加する。
　Δn(t) = Δneq*[1 - exp(-t/T1)]
ただしΔneqは平衡状態での占有数の差、tは時間である。したがって厳密に言えば、資料を磁石に挿入してから、実験者はT1よりも長い時間待ってから測定を開始すべきなのである。しかし、実際はスピン(1/2)核のT1は通常数秒以内であるので、この件は問題にしなくてよい。（省略）
　平衡に達していないスピン系の場合でも、それらの占有率は常に同様の方式に従う。従って、これらの挙動を一般式で表すと式のようになる。
　dΔn/dt = - (1/T1)*(Δn - Δneq)
　緩和現象が平衡時の占有率を常に一定に保つ理由は、基本的に安定系へ向かうスピン遷移（β→α）速度が上向き不安定系へ向かうスピン遷移（α→β）の速度よりもexp{((h/(2*π))*ω0)/(kB*T)}倍だけ上回るからである。
・スピン-格子緩和の原因
　他の種々の分光法で重要な緩和機構は、通常NMRにおいては効果を発揮しない。例えば、自然放出（spontaneous emission）（蛍光）の速度は、遷移周波数の三乗に依存し、NMRにおいては無視できるほど遅い。核スピンは、それを取り巻く世界と非常に弱くしか相互作用しないので、その核を含む分子の運動にはまったく影響を受けない。分子があらゆる方向の回転をしても、スピンは磁場の方向を向いたままである。（省略）
　当然のことではあるが、核スピン緩和の機構は磁気的相互作用が関与し、その中でも最も主要な因子は、双極子相互作用（双極子カップリング）である。二つの核間の双極子カップリングは、両者の距離rとθ（核間ベクトルと磁場方向がなす角）に依存する。このカップリングは純粋に異方性であり、溶液NMRスペクトルでは（ランダムな速い分子運動による）平均化によりシグナルに分裂をもたらすことはほとんどないが、ある瞬間の相互作用は無視できないほど大きい。溶液中では分子が並進、回転、振動運動を行なうためrとθは複雑に変化し、相互作用は素早く変動する。しかし、分子運動により変調緒受けることの双極子カップリングは、周辺の核スピンに対して時間により変化する（時間依存性の）局所磁場を経験させる。この局所磁場の変化が核スピンのラーモア周波数と一致する成分をもっている場合、周辺の核スピン群に共鳴周波数をもつ電磁波を照射することになるため、スピン群は無放射反射遷移を起こし平衡状態へ復帰する。他のスピン緩和の機構も本質的に同じ原因による。例えば、ランダムに働く分子運動により時間依存的に変化する分子内または（例は多くないが）分子間の磁気相互作用（あるいは、I>(1/2)核における電気四極子相互作用）である。
・溶液中の分子の回転運動
　気相中、少なくとも低圧下では、一つの分子が自由に動ける平均距離は長いので、他の分子と衝突して回転状態を変えられることもなく、何回でもくるくると回転できる。しかし溶液中では分子同士の衝突がはるかに大きな頻度で起こる。そのため分子はあらゆる方向からつねにもみくちゃにされ、衝突が起こる度に回転運動は加速または減速し、さらに回転軸も頻繁に方向を変える。このようなランダムな運動は回転ではなく、タンブリング（tumbling）と呼ばれることが多い。
　（省略）
　大まかに言うと、回転相関時間（rotational correlation time）τcは、分子が二乗平均平方根偏差（root-mean-square deflection）で約1ラジアン（約57°）回転するのに必要な時間である。τcよりもずっと短い時間では、ほとんどの分子はもともとの位置の近くに存在するが、t>>τcの場合、分子の配向は完全にランダム化し、もとの配向についての「記憶」は完全に消失してしまう。室温の非粘性溶液中にある小さな分子のτcは、おおよそ100 ps（ピコ秒）程度である。
　（省略）
　要約すると、分子間磁気相互作用の周波数スペクトルは、分子のタンブリング運動により変調を受ける。（省略）スペクトル密度（spectrum density）J(ω)は、周波数ωをもつランダムな運動の成分が見つけられる確率に比例すると考えてよい。すなわち、小さな分子、粘性が低い溶媒中、または高温下では、いずれも平均として速いタンブリングが起こるので相関時間が短く、その結果、スペクトル密度はより高周波数に広がる。
　J(ω) = (2*τc) / (1 + ω2*τc2)
　この式は、分子の初期の配向に関する「記憶」が指数関数的に減衰する場合に適用できる。
・スピン-格子緩和（再考）
　双極子相互作用は緩和の最も主要な原因であるが、事はそれほど簡単ではない。双極子カップリングしている二つのスピンは、相互に相関した時間依存性磁場を経験し（それらは同じrおよびθをもっている）、必然的に協奏的に緩和する。（省略）
　議論を単純化するために、単独の核スピンがランダムな局所磁場により、独立に緩和する理想的な構造について議論する（slichter, 1990）。この近似的説明は、大抵のスピン緩和に関する見識を与え、若干の修正は必要であるが個々の緩和機構についての定量的な推察を可能にするだろう。
　スピン-格子緩和は、変動する局所磁場が核のスピンエネルギー準位間の無放射遷移を誘導することにより起こされる。この過程の速度定数T1-1は、核が経験する局所磁場が、ラーモア周波数ω0で振動する成分をどれだけもっているかという確率に依存する。言い換えると、T1-1はラーモア周波数におけるスペクトル密度J(ω0)に比例する。予想される緩和速度は、
　1 / T1 = γ2 * <Bloc2> * J(ω0)
で表される。ただし、<Bloc2>は局所変動磁場の二乗平均値、γは問題とする核の磁気回転比である。上式から明らかなように、他の条件が同じである場合、局所磁場の強度が大きいほど緩和は速くなる。
　これまでの議論により、スピン-格子緩和速度が回転相関時間によってどのように変化するのか予測できる情報が揃った。
----------
(1) τcは回転に関する時間
　τcが小 → 速い運動
　τcが大 → 遅い運動
(2) τcとの関係
　速い運動/高温/小さな分子 → τcが小
　遅い運動/低温/大きな分子 → τcが大
----------
　高速でタンブリングしているある分子ではω0*τc<<1で、J(ω0)≒2*τcかつT1∝(1/τc)であるので、τcが増加するにつれ（例えば、温度を下げることにより）緩和はより速くなる。逆にゆっくりとタンブリングしている分子では、ω0*τc>>1でJ(ω0)≒(2/(ω02*τc)かつT1∝τcであるので、τcが増加するにつれ緩和はより遅くなる。τcの変化により、T1は極小点（最大の緩和速度）に達するが、この地点では式[1 / T1 = γ2 * <Bloc2> * J(ω0)]の微分から導かれるようにτc=1/ω0である。
　問題とする分子が、T1極小点の左側（速いタンブリング）か右側（遅いタンブリング）に所属するかを決めるためには、どうしてもその分子のτcを知る必要がある。「大まかで便利な経験則」（Sanders, Hunter, 1993）は次のようである。「室温で水溶液中にある分子のτc（ピコ秒）は、大体、分子の相対分子量Mrに等しい」。
例えば、
　Mr=100 ならば τc≒100 ps
　Mr=10,000 ならば τc≒10 ns
である。400 MHzの分光計では、1H緩和速度が最高点に達するのはτc≒1/ω0≒400 psであるので、Mrが数百またはそれ以下である分子はT1極小点の左側（速いタンブリング）に所属する。また分子量が1000を超えるような分子は極小点の右側（遅いタンブリング）に所属する。粘性溶媒中や低温下では、分子はより長い相関時間をもつと予想される。 
　回転運動がスピン緩和機構に関してなぜ重要であるのかについては、次のように説明できる。振動運動はあまりにも速度が大きすぎて、相対的にずっと低いNMR周波数（ω0）の成分をもつことはない。並進運動による分子間の双極子相互作用もそれほど大きな影響を与えない。なぜならば、分子間相互作用によるカップリングは（スピン間の平衡距離が大きいので）、分子内相互作用の場合よりもきわめて小さいからである。これらとは異なり、回転はおおよそω0に近い周波数であり、分子内相互作用にも大きな影響を与えることができる。したがって、回転運動がスピン緩和のための支配的な運動なのである。
　双極子場（〜<Bloc2>1/2）はr-3に依存するので、双極子カップリングしているスピン対のスピン-格子緩和の速度はr-6に比例することがわかる。このように、スピン-格子緩和時間は、核間距離に影響を受け易く、したがって分子構造の推定に役に立つ関数である。
　以上で展開した「大まかで便利な」経験則は、より詳細な理論的扱いにより正当化されている。次の式は異なる磁気回転比をもち双極子-カップリングしている二つの核AとXが、いわゆる極限先鋭化条件（extreme narrowing limit）（ω0*τ0 << 1）下にある場合のスピン-格子緩和速度を示している。
　1 / T1 = (2*π*RAX)2*τc = {μ0/(4*π}*{h/(2*π)}2*γA2*γX2*(τc/r6)
　式中のτcはスペクトル密度に関係する式（これまで上記で説明した式）から導かれ、RAXは双極子カップリング定数である。プロトンの典型的な双極子緩和時間T1は0.1から10秒の範囲にある。核スピン緩和が遅いのは二つの理由による。局所磁場は通常微弱であるので、核スピンとわずかにしか作用しない。またω0*τc=1の場合でさえ、ラーモア周波数でのスペクトル密度が小さい。一方、上式が示すように、T1-1が磁気回転比に依存するということは、他の条件が等しければ、13C核はプロトンよりもゆっくりと緩和することを意味している。
　スピン-格子緩和についてはさらに二つの考慮すべき重要なポイントがある。
・第一に、なぜ局所的な時間依存性磁場により誘起される遷移がスピンを平衡状態へ戻すのか
・第二に、スピンが緩和するときには必ずエネルギーの放出または吸収が起こる。では、このエネルギーはどこへ行きまたはどこから来るのか
　スピン-格子緩和は、非常に弱くではあるがスピンと（そのスピンをもつ）分子の運動の橋渡しをしている。言い換えれば、スピン-格子緩和は、スピン系とその周囲との間にエネルギー交換が起こる経路を供給し、スピンが格子と熱的な接触をすることを可能にしている。その結果、スピンは外界と平衡になることができるのである。この過程により、下向きの（β→α）速度が上向き（α→β）よりexp{ΔE/(kBT)}倍だけ速いことによる平衡状態の占有数が復活する。
　スピン-格子緩和過程におけるエネルギーの吸収・放出は、分子の運動を介して伝達されるので、格子はほんのわずかだけ冷えたり温まったりする。しかしスピンのこのエネルギーは、溶液中の分子の回転、振動、並進運動のエネルギーと比較すると微小であり、核スピンの緩和による試料の温度変化は測定不能なほど小さい。これは大きな湖に小さな熱い物体を投げ入れても、湖全体の温度が変わらないのと同じである。
・核オーバーハウザー効果（NOE）
　二つの等価なプロトンAとXを含む分子について考えてみよう。両者はJ-カップリングしておらず、したがって1Hスペクトルはそれぞれの化学シフトにシングレットを示す。この分子のスペクトルを記録している最中に、Xの共鳴周波数をもつ強いラジオ波磁場を照射し、Xを飽和させる（すなわちαとβの占有数が等しい）実験を想定しよう（※ 飽和とは、ラジオ波をあて続け、αとβの占有数が等しくなり、それ以上遷移が起こらない状態のこと）。この照射によりXのNMRシグナルは消失するが、二つのスピンがある大きさの双極子相互作用をしている場合、Aの共鳴線の強度に変化が現われる。すなわち、ピークが強度を増したり、弱くなったり、ときには強度が反転したりする。この顕著な現象は核オーバーハウザー効果（nuclear overhauser effect）または核オーバーハウザー増強（nuclear overhauser enhancement）、略してNOEと呼ばれている。この現象は、スピン-格子緩和より、もっと直接的に核間距離についての情報を与えてくれる。
　NOEの起源を理解するためには、互いに双極子カップリングしている二つのプロトンについて、どのようなスピン-格子緩和経路が可能であるかを考える必要がある。4つのエネルギー準位、αAαX, αAβX, βAαX, βAβXを示す。混乱を避けるため、今後は下付きのAとXを省略することにする。化学シフト差を無視し、さらに{h/(2*π)}*ω0 << kBTを用いると、平衡状態における4つの状態の相対的占有数は次の式で表される。
　n(αα) = 1 + 2*Δ
　n(αβ) = n(βα) = 1
　n(ββ) = 1 - 2*Δ
ただし、Δ=(1/2)*{h/(2*π)}*ω0 /(kB*T)である。4つの許容NMR遷移（ΔmA=±1またはΔmX=±1）間の占有数の差は2*Δである。
　二つの対になったプロトンに対して可能な6つの遷移のうち、4つにおいては一つのスピンが反転するのみである（例えば、αA←→βA または αX←→βX）。これらはこれまで議論してきたスピン-格子緩和過程そのものである。それらの速度定数をW1AおよびW1Xとしよう。ただし、下付きの添字は磁気量子数の変化（±1）、上付きの添字は緩和するスピンを特定するためのものである。
　他の二つの緩和過程は交差緩和過程と呼ばれており、AとXの両者が同時に緩和する。すなわちαα←→ββ（両スピンが同方向へ反転：速度定数W2AX）およびαβ←→βα（AとXが逆方向へ反転：測定定数W0AX）である。交差緩和はランダムな分子運動によりAとXが経験する局所磁場同士が相互に影響しあうことにより起こる。(3*cos2θ-1)/r3の値は両スピンに対してどの瞬間でも常に同じであるからである。その結果として、両者のスピンが同時に反転することができるのである。W0AXとW2AXの過程は、単に占有数をもとの平衡状態に戻す特別な経路である。なお、スピン緩和は電磁放射の吸収にも放出にも関与しないので、通常の選択律Δm=±1の制限を受けないことに注意すること。
（ここからNOEが発生する機構についての議論に進む）スピンXに十分な強さをもつラジオ波を照射して、Xの二つの遷移（αα←→αβとβα←→ββ）を飽和させる、すなわちααとαβおよびβαとββの占有数を等しくすることを想定する。（Xの照射はαX←→βXのみを誘発する）。ここでは単純化のために、A-遷移（αα←→βαとαβ←→ββ）の占有率数の差は影響を受けず、2*Δのままであると想定している。
　ここでラジオ波のスイッチを切り、W2AX以外の緩和経路を無視する（非現実的ではあるが便宜的なつくり話である）。この緩和経路によりααとββの間で占有数が移動し、最終的にそれぞれの平衡占有数1+2*Δと1-2*Δに復帰する。その結果、A-遷移による占有数の差は3*Δとなる。すなわちA-ピークの強度が増加する（増加分50%）。交差緩和は、飽和したX-スピンから双極子カップリングの相手であるAへ磁化を移動させてしまったのである。
　これとは逆に、Xを飽和させた後、W0AXが支配的な過程である場合は、αβとβαの占有数はそれぞれの平衡値（両者とも1）と変わらないので、A-遷移による占有率の差はΔだけになってしまう。これはA-ピークの遷移が減少（減少分50%）することを意味している。
　NOEは、Xの照射により変化を受けたスピンAのNMR強度（i）ともとの強度（i0）で表されるパラメータηにより定量的に定義されている。
　η = (i - i0)/i0
　上述の「大まかなで便利な」議論からは-(1/2)=<η=<+(1/2)が示唆される。実際、厳密な処理により求めた同種核NOEの最大値は確かに+1/2である。ところが、最小値は-1である。現実にはW2AXもW0AXも他の家庭をしのぐ支配的な経路ではなく、ηは両極値の間の値をとる。上記の議論で明らかなように、ηはW2AX-W0AXと同じ符号をもつ。
　では、何がW2AX-W0AXの符号と大きさを決めるのであろうか。我々は既にスピン-格子緩和の速度は、ラーモア周波数ω0におけるスペクトル密度に比例することを学んだ（なぜならαとβのエネルギー準位は{h/(2*π)}*ω0だけ離れているからである）。ααとββの差は2*{h/(2*π)}*ω0であり、αβとβαはほぼ完全に縮退しているので、次式が成り立つことが予測される。
　W2AX ∝ J(2*ω0) および W0AX ∝ J(0)
　最初にω0*τc>>1である。ゆっくりとタンブリングしている分子を考えてみよう。J(0)>>J(2*ω0)であるので、W2AX-W0AX<0, η<0である。すなわち、大きな分子のプロトンを飽和させると、近接するプロトンのNMR強度は減少する。
　小さな分子については、事態はそれほど単純ではない。もしω0*τc<<1ならばJ(0)≒J(2*ω0)である。したがって、極限先鋭化条件においては二つの交差緩和速度間の差は、スペクトル密度からではなく、双極子相互作用がααとββ、αβとβαをそれぞれ結びつける効率によって生じるのである。ω0*τc<<1でああるとき、W2AX=6*W0AXであり、したがってη>0である（Neuhaus, Williamson (1989)には、極限先鋭化条件でW2がW0の6倍になると記されている）。その結果、小さな分子のプロトンを飽和させると、近接するプロトンのNMR強度が増加するわけである。
　1H-1H NOEについてまとめると、ηは速い運動に対しては正の、遅い動きに対しては負の値となる。符号の変化はW0AX=W2AXの場合に起こり、この時点では二つの交差緩和過程が相殺してしまう。これはω0*τc≒1の場合に起こる。
　これまで我々は、双極子緩和機構のみに注意を向けていたが、それはとりも直さずこの機構が交差緩和の最も一般的な要因だからである。実際そのほかの緩和機構はW1経路を利用することにより、NOEに最も重要である交差緩和を迂回して、占有数を平衡状態に戻す傾向があるため、NOEを減少させる。したがって、「非常に速い運動」と「非常に遅い運動」の範囲内で、同種核NOEの最大・最小値が期待されるのは、双極子相互作用が緩和の支配的要因である場合に限る。
　ところでNOEは異種核スピン対の愛団も観測できる。たとえば、小さな分子で、プロトン共鳴線を飽和させると、好都合なことに近傍の13C核のNMRシグナルに増大が起こる。極限先鋭化条件においてほかの緩和機構がないとき、シグナルの増大は次の式で表される。
　η = (1/2)*(γX/γA)
ただし、γXとγAは飽和した核（X）と観測核（A）の磁気回転比である。したがって、X=1H、A=13Cの場合 γH/γc≒4 であるので、最大のNOEは約2である。同各種の場合のη=(1/2)と比較すると非常に大きな値であることがわかる。
　NOEは分子構造研究に非常に役立つ重要な情報源である。交差緩和がr-6に依存するので、原理的には核間の距離を直接求められるはずだが、実際はもっと複雑な要因が関係してくる。ただし、関係する核同士が類似した運動をし、双極子ではない他の緩和機構が両者に等しく作用している場合に、核間の相対的な距離を見積もることが可能である。またNOEは、二つの核が近いか遠いかを判定する判定量的な手段としてしばしば利用されている。このような制限があるにしろ、NOEは構造研究にとって計り知れないほど貴重な情報源となっている。

・スピン-スピン緩和
　遅い化学交換はスピンの寿命を短くするため、線幅を拡大するという現象について議論した。スピン-格子緩和についても同じ現象が起こり、1/(π*T1)程度の広幅化が予想される。しかし、これだけがNMRの線幅に影響を及ぼす緩和過程ではないので、実用的な新しいパラメータ、すなわちスピン-スピン緩和時間T2を定義する。
　1/(π*T2) = Δν
　ただしΔνは緩和により誘起される線幅（半値幅）である。T2についてはもっと根本的な説明が必要であるが、ここではT2を単に線幅のパラメータとして扱う。
　T2は（化学交換に続く）線幅に関する第二の要因であり、以下その性質を理解するための議論を進める。まず相互に双極子-カップリングしている一対の核が、（単結晶ではないような）配列が整っていない分子の固体中で自由に動けず固定化されている状態を想定する。ばらばらに配向しているスピン対は、それぞれ異なるθの値をもつので、異なる双極子分裂〜RAX*(3*cos2θ-1)を示す。その結果得られるのが、ペイクパターン（Pake pattern）と呼ばれるシグナルである（ペイグダブレットともいう。半整数核スピン間の双極子カップリングや重水素などの四極子核の遷移によって生じる）。同じ分子が液体中にある場合、もし分子のタンブリングが極端に遅ければ、配列が整っていない固体と本質的に同じようなスペクトルを示すであろう。しかし、タンブリングの速度τc-1がRAXに近づくと、(3*cos2θ-1)の変調により双極子分裂は平均化されるようになるだろう。この振舞いは、化学交換の場合と比較することができる。もし分子が二つの配向間を素早く飛び回り、その速度が両配向における双極子カップリングの差よりも大きい場合、それらの差は平均化されるので、両者の中間の双極子カップリングが観測されるはずである。もちろん溶液中の分子はあらゆる可能な配向の間を飛び回っているので、このような速いタンブリング状態でのスペクトルの形を正確に決めるためには、双極子カップリングを全てのθにわたって平均化する必要がある。優先する分子配向がない場合、上記に対応する平均は次の式で表される。
　λ*RAX*∫π/20 (3*cos2θ-1)*sinθ dθ
　式中の体積要素sinθは、AX対の配向がθとθ+dθとの間に見つかる確率を考慮するためのものである。この積分は厳密にゼロである。タンブリングが十分速いとペイクパターンの傾向が弱まり、速度がさらに大きくなると「運動により先鋭化した」共鳴線（motionally narrowed line）へと変化する。その線幅は回転速度の増加に従って小さくなる。
　この運動により先鋭化した共鳴線の線幅、すなわちT2を見積もるために、化学交換について説明した理論を取り入れよう。式[Δν=π*(δν)2/(2*k)=π*(δν)2*τ/2]によれば、速い交換による線幅はほぼ(δν)2*τである。ただしδνは交換サイト間の共鳴周波数差、τは各サイトでの平均滞在時間である。そこからの類推によれば、溶液中の分子の双極子カップリングによる線幅は、おおよそRAX2*τcすなわち双極子カップリング定数の二乗に運動の回転相関時間を掛けたものと見なすことができる。したがって、タンブリングが速くなるにつれて双極子相互作用はより効果的に平均化され、線幅が小さくなりT2が増加する。
　ランダム磁場緩和機構（random fields relaxation mechanism）の理論式（Slichter, 1990）は次のように表される。
　1/T2 = (1/2)*<Bloc2>*J(ω0) + (1/2)*γ2*<Bloc2>*J(0)

・四極子緩和
　スピン量子数が(1/2)より大きい核は、磁気双極子モーメントと共に電気四極子モーメント（electric quadrupole moment）をもつ。電気四極子については、原子核内の楕円体状の正電荷分布をイメージするとよい。正電荷は北極と南極部分に多く集まり、その分乏しくなった正電荷が赤道部分に存在する。
　電気四極子は、例えば、HClのような電気四極子とは異なり、空間的に均一な電場には影響を受けず、電場勾配（electric field gradient）とのみ相互作用する。この性質は、核四極子モーメントを背中合わせに二つの電気双極子であるとみなすと理解される。
　核における電場勾配は、局所的電荷分布の不均一さを示す尺度である。非常に対称性が高い、例えば球体、立方体、八面体や四面体などの環境中では、周辺の電荷から生じる電場勾配が完全に打ち消し合ってしまうので、正味の四極子相互作用は存在しない。しかし、対称性が低い環境中の核は、NMR分光計の磁場中における分子配向に依存するゼロではない電場勾配を経験する。（省略）
　この相互作用は磁気双極子カップリングと同様に、単結晶のNMRスペクトルに分裂を起こし、粉体や分子配向がランダムな固体では幅広い共鳴線を与える。四極子相互作用もまた、分子のタンブリングによる変調を受けるとスピン緩和の原因にもなる。
　対称性が低い環境中にあるI>(1/2)核の多くは、大きな四極子相互作用となり、非常に効果的にスピン緩和を引き起こす。このことを物語る最も顕著な現象は14N, 17O, 35Cl, 37Clなどの核が示す非常に大きな線幅（すなわち小さなT2）である。例えば、ピラミッド型のNMe3の14N共鳴線の幅はほとんど100 Hzに達するが、14NMe4+、15NMe3, 15NMe4+は全て1 Hz以下の線幅を示す（14NMe4+は正四面体形で窒素原子の位置に電場勾配をもたない。また、15NはI=(1/2)核であるので四極子モーメントをもたない）。四極子による線広幅化は分解能を著しく低下させる。このことが、溶液NMRでI>(1/2)核があまり好まれない大きな理由である。
　四極子緩和によるもう一つの主要な影響は、四極子核とJ-カップリングしているスピンの多重線構造が消えてしまうことである。例えば、効果的なスピン-格子緩和の結果、I=1核は3つのスピン状態（m=+1, 0, -1）の間を激しく飛び回る。その結果、この核とカップリングしているスピン（ラーモア周波数ν0、カップリング定数J）は、共鳴周波数ν0+J、ν0、ν0-Jの間を素早く飛び回る。もしこの跳躍の速度が関係する周波数の差、すなわちJと2*Jと比べて大きい場合は、1:1:1のトリプレットではなく、単に1本の線が中間周波数ν0の位置に現われる。これは分子間での速いプロトン交換により、多重線の分裂が消えてしまう現象と本質的に同じである。（省略、※ 文献[9]には実例も記載されている）

◇ 強度 [6]
　NMRシグナルの吸収曲線の面積は遷移強度の尺度である。積分値は解析装置において階段状の曲線として描かれる。
　1H スペクトルにおいて階段の高さで示される積分強度は、測定している分子中の1H核の数に比例する。
　混合物の定量的な分析には、それぞれのシグナルに対応する化学的に等価なプロトンの数を考慮しなければならない。（省略）
　通常の13C スペクトルのシグナル強度から、正確な定量的結果を得ることは困難である。シグナルの強度は、遷移の起こるエネルギー準位間の分布の差に比例する。したがって、緩和時間に大きく依存しているともいえる。13Cの緩和時間T1は10-1から3*102 s の範囲である。13Cにプロトンが結合していると、T1の値が効果的に減少する。第4級炭素13Cは最も大きなT1を持っているので、シグナル強度は最も弱い。さらに分子の大きさと分子運動は、縦緩和にかなり大きな影響を及ぼす。
　T1の値（秒単位で示される）は、大きな分子と小さな分子の間で、あるいはプロトンのついた炭素と第4級炭素の間で際立った差が見られる。分子運動の異方性も重要な役割を果たす。すなわち、パラ位の炭素は回転の起こり易い軸上にあり、最も小さなT1値をもつ（ある特別な場合には、この効果を用いてシグナルの帰属を行なうこともある）。
　パルス間の時間間隔は一般に13C NMRにおいてきわめて短いので、スピン系が緩和して平衡状態まで達することができない。（省略）
　ある核への、またはその近くのNMR周波数への照射は、近傍の核の緩和時間の摂動を起こす。これにより、照射した核の近傍の核のシグナルの強度に変化をもたらす（核オーバーバウザー効果、NOR）。1H NMRでは、二重共鳴実験を行なったときのみこの効果が起こる。しかし13C NMRでは、通常よく用いられる手法が、1H広帯域デカップリングで測定されるスペクトルである。したがって、プロトンとのカップリングによる分裂は消滅し、それが一重線のピークとして現われる。デカップリングによる異核オーバーハウザー効果により、シグナルの強度は最高200%増加する。緩和時間の違いと核オーバーハウザー効果の違いが、13C NMRシグナルの理論的強度比と測定値とのずれの原因となる。さらに、測定条件がピーク強度に大きく影響することを考慮にいれなければならない。しかしながら、こうした効果を取り除くことにより、これらの13C NMRの欠点を避けることが可能である。必要ならば積分できるようなスペクトルを得ることも可能である。

　NMR実験で最も役に立つJ-カップリングは、核と核とが3本結合（three-bond）によって離れたカップリングであろう。その代表例がH-C-C-Hにおける3JHHである。これらのカップリング定数は「カープラス（Karplus）式」に従い、二つのH-C-C面が作る二面角θの変化と共にさまざまな値をとることが、経験的および理論的に確かめられている。
　3J ≒ A + B*cosθ + C*cos2θ
　上式におけるA, B, Cのおおよその値（置換基効果や他の効果を含む）を計算することは可能であるが、これらはコンホメーションが固定されている構造既知のモデル化合物を用いて、経験的に知られた係数として取り扱う方がより好ましい。典型的な値はA=2 Hz, B=-1 Hz, C=10 Hzであり、これらによりθとJの関係が得られる。ただし、これら3つのパラメーターの値は、炭素原子上の置換基により影響を受ける。
　3本結合カップリングは、主としてコンホメーション解析に用いられている。例えば、シクロヘキサン環のプロトンについては、それがアキシアルまたはエクアトリアルのどちらに配置するかにより3JHHの値が異なる。またC=C結合を経由する1H-1Hカップリング定数は、トランス体の方がシス体よりもほぼ2倍ほどの大きさである。
　カープラス式はタンパク質の構造研究においてさまざまに利用されている。とくにポリペプチド鎖中のアミドプロトン（NH）とCαプロトンとのカップリングから、タンパク質骨格のコンホメーションに関する情報を得ることができる。例えば、タンパク質の二つの主要な二次構造であるαヘリックスとβシートは特有の二面角H-N-Cα-Hをもち、それぞれ約120°および約180°である。したがって、3JHH値が6 Hzより小さければタンパク質のαヘリックス領域、おおむね7 Hzを超えていればβシートの領域であると推定できる。
　コンホメーションが固定されていない分子についての3本結合カップリングの解析はやや異なっている。例えば、アミノ酸のα-プロトンと二つのβ-プロトンとのカップリングの例を考えてみよう。3つのねじれ型コンホメーション（回転異性体）が非常に速く交換しているので、観測される二つの3Jαβは平均化されるが、その値は3つのエネルギー極小状態の占有数の加重平均である。
　Jαβ1 = P1*Jg + P2*Jg + P3*Jt
　Jαβ2 = P1*Jg + P2*Jt + P3*Jg
　ただし、P1+P2+P3=1, JtとJgはそれぞれトランス形（θ=180°）とゴーシュ形（θ=±60°）の3本結合カップリング定数である。ここでは簡単化のためにJtとJgに対する置換基効果は考慮していない。したがって、固定化されたモデル化合物の測定または計算からJtとJgの値を知ることができれば、これら3つの回転異性体の相対的な占有数を決定することができる。
・交換可能なプロトンとのビシナルカップリング
　OHプロトンと隣のCHプロトンとのカップリングは、水または酸の含まれていない非常に純粋なアルコールにおいてのみ観測される。ジメチルスルホキシド（DMSO）中では、室温でプロトン交換が非常に遅いので、このカップリングを見るための溶媒としてよく用いられる。
　同様のことがNHプロトンでもいえる。3J(CH-NH)のビシナルカップリング（塩基触媒による）は、プロトン移動が遅い場合にのみ観測される。C-NH-CHの構造をもつものに、このカップリングが見られる（芳香族アミン、エナミン、アミド）。
　トリフルオロ酢酸中では、アンモニウムカチオン上のプロトンの交換は遅く、3J(CH-N+H)のカップリングがC-Hシグナルの分裂として観測できる。
　アルコール、アミン、アミドの3Jカップリング定数が立体配座に依存するのは、ビシナルのCH-CHカップリングの場合と類似している。自由回転があるとき、
　3J(CH-OH) : 約 4 - 5 Hz
　3J(CH-NH) : 約 5 - 6 Hz
たとえば、アミドでNHプロトンが幅広いシグナルを示す場合でも、CHシグナルの分裂が現われることは注目される。
　最後に、14N, 1Hの直接カップリングに関するいくつかの注意点を述べる。NH4+イオンはカップリング定数が52.8 Hzである。14Nは核スピン量子数1であり、電子四極子モーメントをもつ。極めて対称性の高いNR4+イオン以外の化合物は、四極子のために緩和時間が長く、14N, 1Hのカップリングは観測されない。これはプロトン交換とともに、NHシグナルが幅広くなる理由である。しかし、14Nの共鳴周波数を照射すれば、この問題は解決できる。
・遠隔カップリング

　極端な例を挙げると、ビシクロ[1.1.1]ペンタンで橋頭位のプロトンは4J=18 Hzとなる。

　4Jや5Jの大きなカップリングは、アルキンやアレンで見られる。

◇ 構造的特徴と1H化学シフトの相関 [6]
・メチレンプロトン
　メチレンの二つのプロトンは、分子中で対称要素により関係づけられているときは化学的に等価である。これには分子内運動の影響も考慮に入れてある。二つのプロトンが化学的に等価であっても、磁気的に等価であるとはかぎらない。
・メチンプロトン
　多様な置換基の組み合わせが考えられるため、メチンの化学シフトの幅はメチル基やメチレン基よりも広い。
・二重結合および三重結合のプロトン
　アセチレンプロトンの化学シフトの幅はだいたい1.8 =< δ =< 3.2であり、オレフィンプロトンよりも高磁場にある
・芳香族または複素芳香環化合物のプロトン
　芳香環または複素芳香環上の炭素に結合したプロトンは、δ=6.0から9.5の間の化学シフトの値を示す。多くは7と8の間にある。この種の化合物は多様性の富むので、表にまとめにくい。
・OH-, SH-, NH-プロトン
　OやS, Nのようなヘテロ原子に結合したプロトンの化学シフトは、測定条件（濃度、温度、溶媒）に大きく依存する。シグナルはたいてい幅広く、カップリングがない。これは速い分子間プロトン交換による。分子内で二つの化学的に非等価なプロトンの間で速い交換がある場合は、一つの平均化されたシグナルだけが観測される。
　XH-プロトンのシグナルを帰属するときに、D2Oやトリフルオロ酢酸との交換が有効である場合が多い。
　XH-プロトンの化学シフトを見ると（δ値はCCl4またはCDCl4を溶媒としたときの値）、一般にアルコールのOHシグナルは、濃度が高くなるが、あるいは温度が低くなるに従い、低磁場にシフトする。どちらの効果もアルコール分子の会合の程度を増加させるので、水素結合の非遮蔽効果は増大する。エタノールのCCl4の超希薄溶液のOHシグナルは、δ=約0.9 Hzである。濃度を上げていくと、純粋なエタノールのシフト値δが約4.6に近づく。X-Hシグナルを帰属するときは、シフト値が、濃度、温度、溶媒に大きく依存することを考慮する必要がある。

◆ 固体高分解能NMR
◇ 溶液および固体のNMR
　1H核の磁気モーメントは双極子相互作用により、局所磁場を近くの13C核に及ぼす。固体ではこの1Hの磁気モーメントが外部静磁場に対して不規則に平行、または反平行に向いて固定されるために、それぞれの13C位置の周りの局所磁場が異なり、13Cの共鳴周波数が色々な値を持って分布することになり、幅広いスペクトルのようになる。固体ではこの双極子相互作用による線幅が1から100 kHzのオーダーを持つ。一方、溶液では速い等方的な分子運動により、この1Hによる局所磁場は時間的に平均化されて消失し、シャープな線幅を与える。
　一般的に、固体状態の核スピンは、外部静磁場との相互作用により生ずるゼーマン項HZ、化学シフトの異方性項HCSA、間接スピン-スピン相互作用項HJ、双極子相互作用項HD、核四極子相互作用項HQにより、その吸収線が影響を受ける。ハミルトン演算子Hはしたがって、
　H = HZ + HCSA + HJ + HD + HQ
と表される。特に、線幅はこれらの各項に大きく影響される。そのオーダーは最も大きいものでHCSA〜100 kHz、HJ〜100 kHz、HD〜100 kHz、HQ〜1 MHzもある。言うまでもないが、核四極子項はI>1/2の核のみに現われる。
