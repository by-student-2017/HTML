■ QSTEM (Ubuntu 18.04 LTS (64 bit)) 
fit tem image (test version)

□ Usage (case is any file name) (lammps data case)
1. ./lmp2tem_qsc case -kv 80 -lab6
  (80 mean 80 kV)
  e.g., 
  ・ 200 kV, LaB6: ./lmp_data2cfg_qsc case -kv 200 -lab6
  ・  80 kV, LaB6: ./lmp_data2cfg_qsc case -kv  80 -lab6
  ・ 200 kV, FE  : ./lmp_data2cfg_qsc case -kv 200 -fe -autoz
  ・  80 kV, FE  : ./lmp_data2cfg_qsc case -kv  80 -fe
  ・ 200 kV, Correct-Cc  : ./lmp_data2cfg_qsc case -kv 200 -arm -autoz
  ・  80 kV, Correct-Cc  : ./lmp_data2cfg_qsc case -kv  80 -arm
  ・  30 kV, Correct-Cc  : ./lmp_data2cfg_qsc case -kv  30 -arm
2. python fit_tem.py
3. python plot_tem.py

□ Usage (case is any file name) (cif file case)
0. case.cif > VESTA > (P1 symmetry) > case.cif
1. ./cif2cfg_qsc case 80
  (80 mean 80 kV)
  e.g., 
  ・ 200 kV, LaB6: ./cif2cfg_qsc case 200 -lab6
  ・  80 kV, LaB6: ./cif2cfg_qsc case  80 -lab6
  ・ 200 kV, FE  : ./cif2cfg_qsc case 200 -fe -autoz
  ・  80 kV, FE  : ./cif2cfg_qsc case  80 -fe
  ・ 200 kV, Correct-Cc  : ./cif2cfg_qsc case 200 -arm -autoz
  ・  80 kV, Correct-Cc  : ./cif2cfg_qsc case  80 -arm
  ・  30 kV, Correct-Cc  : ./cif2cfg_qsc case  30 -arm
2. python fit_tem.py
3. python plot_tem.py
