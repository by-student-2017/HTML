charmm (https://www.charmm.org/charmm/)
------------------------------------------------------------------------------
■ Install (ubuntu 18.04 LTS (64 bit) on windows 10 (64 bit))
(http://charmm.chemistry.harvard.edu/charmm_lite.php)

□ charmm
・charmm.tar.gz
0. sudo apt install gcc libopenmpi-dev
1. tar -xvzf charmm.tar.gz
2. cd charmm
3. ./install.com gnu M
  (wait for a long time until you can get "install.com> CHARMM Installation is completed."

□ charmm [1]
・charmm.tar.gz
0. sudo apt install gcc libopenmpi-dev
1. tar -xvzf charmm.tar.gz
2. cd charmm
3. export MAKE_COMMAND="make -j4"
  (4 = number of core. in this case 4 core.)
  for 6 core case,
  export MAKE_COMMAND="make -j6"
4. ./install.com gnu M
  (wait for a long time until you can get "install.com> CHARMM Installation is completed."

□ test
1. cd charmm
2. cd exec
3. cd gnu_M
4. ./charmm -h

※ gedit doc/install.doc
※ ar: `u' modifier ignored since `D' is the default (see `U') と表示されても
  install.com> CHARMM Installation is completed.
と表示されるまで待ち続ければよい。
