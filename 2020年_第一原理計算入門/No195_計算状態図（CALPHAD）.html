■ 理論的背景 [4]

□ 純元素の自由エネルギーと相変態
　物質の相安定性を議論するためには、エンタルピーとエントロピーおよびこれらの2つを内包する自由エネルギーについて理解する必要がある。

□ 定積熱容量（Cv）、定圧熱容量（Cp）およびエンタルピー（H）
　物質の温度を 1 K だけ上昇させるために必要な熱量（1モルあたり）を熱容量という。この熱容量には、定積下における定積熱容量Cv と定圧下における定圧熱容量Cp の2種類があり、次式により定義される。
　Cv = (ΔQ / ΔT)v = (∂U / ∂T)v
　Cp = (ΔQ / ΔT)p = (∂H / ∂T)p
上式を積分することにより、Cv と Cp の値から内部エネルギーUとエンタルピーHを求める式が得られる。
　U = U0 + ∫T0 Cv dT
　H = H0 + ∫T0 Cp dT
ここで、U0とH0 はT=0KにおけるUとHの値である。CvとCpは次式により相互に換算できる。
　Cp - Cv = [ (∂U / ∂T) + P] ・(∂V / ∂T)p = (α2V/ β) ・T

□ エントロピー(S)
　ボールが斜面を転げ落ちる現象はエネルギーの考え方で説明できるが、相変態がなぜ起こるかについては、エネルギーだけでは説明できず、エントロピーを考慮しなければならない。このエントロピーはクラジウスによって次式のように定義された。
　ΔS = ΔQ / T
ここで、ΔQは注目している系が準静的に受け入れた熱量である。熱力学第一法則がエネルギー保存則であるのに対して、熱力学第二法則は一般にエントロピー最大の原理と称される。熱浴（H）中に置かれた試片（P）について、熱浴と試片を併せた全体（T）のエントロピーが増大することから、
　ΔST = ΔSH  + ΔSp >= 0
　ボルツマンは、変数Wを系を構成している原子集団のミクロ状態の総数としたとき、エントロピーSが次式で表せられることを提示した。（kBはボルツマン定数）
　S = kB*ln (W)
純元素におけるミクロ状態の総数Wは、結晶の熱振動、空孔の配置、磁気スピンの配列などにより定まる。

□ ギブス自由エネルギー(G)
　G = H - T*S
により定義されるギブス自由エネルギーを用いると、定温・定圧の条件下での試片のギブス自由エネルギー変化ΔGpについて、
　ΔGp = ΔHp - T*ΔSp <= 0
の関係が導かれる。従って、定温・定圧では物質のギブス自由エネルギーGが低下するような変化しか起こらない。すなわち、熱平衡状態における安定条件は次式により表される。
　Gp = min. （定温・定圧）
定圧熱容量 Cp を用いると、エンタルピー H は(H = H0 + ∫T0 Cp dT)により表され、エントロピーSはクラジウスの式 ΔS = ΔQ / T = Cp*ΔT / T より、
　S = S0 + ∫T0 (Cp / T) dT
ここで、S0はT=0 Kにおけるエントロピーの値であり、純元素の場合、熱力学第三法則よりS0=0である。従って、GとCpとの関係式は次式で表される。
　G = H - T*S = H0 + ∫T0 Cp dT - T∫T0 (Cp / T) dT
上式を温度Tで微分すると、
　(∂G/∂T)p = -∫T0 (Cp / T) dT= -S < 0
　(∂2G/∂2T)p = - Cp / T < 0
となる。これより、ギブス自由エネルギーGと温度Tの関係は常に右下さがりで常に上にで凸の曲線となる。

□ Lattice Stability （純元素の自由エネルギー）
　純元素Aがα相の状態にあるとき、温度Tの関数として次式により近似されている。
　GαA(T) - HSER A = a + b*T + c*T*ln (T) + Σ dn*Tn
ここで、HSER A は純元素Aが 298.15 K において標準状態（SER: Standard Element Reference)にあるときのエンタルピーの値である。上式における係数 a、ｂ、ｃ、dnの値は、固相、液相、気相が安定に存在する温度範囲では、それぞれの定圧熱容量 Cp の実測データなどを利用し、決定することができる。しかし、状態図の計算においては、各相が準安定な温度範囲の自由エネルギーに加えて、本来純元素では安定には現れない結晶構造の固相、例えば、993 K の融点までfcc相のみが安定なAlにおけるbcc(A2)相、hcp(A3)相,、bct(A5)相などの自由エネルギーについても、係数 a、ｂ、ｃ、dnの値を適切に評価することにより、Al基多元系合金において現れるbcc相、hcp相、bct相などの自由エネルギーが任意の温度と組成の範囲で計算できるようになる。
-----------------------------------------------------------------------------
■ Lattice Stability の評価例
　Lattice Stability は入手できないことも少なくない。この場合、自らlattice Stability を評価する必要が生じる。Lattice Stability の簡便な評価法の一例として、Dinsdale によるZnのrhombohedral相のLattice Stabilityの推定法を紹介する。
　常温常圧で安定なZn(hcp)の融解エネルギーを、
　GLZn - GHCPZn ≒ 7322 - 10.5706 ・T (A1)
と近似する。常温常圧においてrhombohedral構造が安定なAs, BiおよびSbの融解エントロピーの値を参考にして、Zn(rhombohedral）の融解エントロピーを 22 J/mol・Kと仮定する。すなわち、
　GLZn - GRHOMBOZn = A - 22 ・T (A2)
Zn (hcp)が高圧下においても他の結晶構造に変態しないことから、任意の温度（0 > K )においてZn(rhombohedral)が不安定となるように、Zn (hcp) -> Zn (rhombohedral)の変態温度を-200 K に設定する。このとき、式(A1) - 式(A2) より、
　GRHOMBOZn - GHCPZn ≒ (A - 7322) + 11.4294・T = 0 at T = -200 K
が成り立ち、A=2285.88が導出できる。以上より、Dinsdale はZn (rhombohedral)のLattice Stabilityを
　GRHOMBOZn - GHCPZn ≒ 2285.88 + 11.4294・T ≒ 2300 + 11.5・T
と評価している。Lattice Stability は、必要に応じて圧力pに依存するエネルギーGpresAと磁気エネルギーGmagAを含む次式により近似されている。
　GαA(T,p) = GchemA(=GαA(T) - HSERA ) + GpresA + GmagA

□ 圧力誘起スイッチングの熱力学的メカニズム [5]
　圧力によりλ-Ti3O5がβ-Ti3O5にスイッチングする機構を熱力学的に理解するため、Slichter-Drickamerモデルと呼ばれる平均場理論モデルを用いて計算を行った。今、λ-Ti3O5を表す電荷非局在状態Ti(1)3.3+ - Ti(2)3.3+ - Ti(3)3.3+と、
β-Ti3O5を表す電荷局在状態Ti(1)3.0+ - Ti(2)3.7+ - Ti(3)3.3+の間で起こる金属-半導体相転移を考える。すると、ギブス自由エネルギー(G)は、このモデルでは次式で表すことができる。
　G = x*(ΔH) + γ*x*(1-x) + T{R*[x*ln(x) + (1-x)*ln(1-x)] - x*(ΔS)} + Gβ
ここで、xはλ-Ti3O5に対するTi(1)3.3+ - Ti(2)3.3+ - Ti(3)3.3+ 電荷非局在化ユニットの割合、ΔHはλ-Ti3O5とβ-Ti3O5の間の転移エンタルピー、ΔSは転移エントロピー、γはλ-Ti3O5とβ-Ti3O5の間に働く相互作用パラメータを表している。また、Gβはβ-Ti3O5のギブス自由エネルギーでエネルギー基準値としており、Rは気体定数である。
　ΔH値およびΔS値にはDSC測定より得られた実測値を用い、γをフィッティングパラメーターとして計算すると、各温度における G vs. x 曲線が得られる。
　室温の G vs. x 曲線を見ると、大気下（0.1 MPa）でλ-Ti3O5にエネルギー極小値が現れており、λ-Ti3O5が準安定相であることがわかる。一方、60 MPaの圧力を加えたときのG vs. x曲線では、圧力により G vs. x曲線が変化しており、λ-Ti3O5のエネルギー極小値が消失している。そのためλ-Ti3O5はβ-Ti3O5に転移する。圧力誘起相転移の起源は、ΔH(=ΔU + P*ΔV)におけるP*ΔVの項であると考えられる。ここで、ΔUは内部エネルギーの変化、ΔVは体積変化である。このような低い圧力では、圧力誘起によるΔUの変化は無視できるほど小さい。実際、圧力下でのフォノンモード計算では、ΔUの圧力変化は60 MPaで約1*10-3 kJ/mol程度と示されており、これはP*ΔV=0.19 kJ/molに比べて2桁小さい。また、ΔSの圧力変化も非常に小さく、本系の圧力誘起相転移には寄与しないと考えられる。

□ マックスウェルの関係式 [1]
　ある状態関数が二階微分可能な場合（熱力学で取り扱う系では相転移点以外ではこの条件は満たされている）、二階導関数は微分の順序によらないという性質を用いて状態変数間の関係を求めるものである。一般式としては、次式で表される。
　∂[∂f(x,y)/∂y]/∂x = ∂[∂f(x,y)/∂x]/∂y
この関係をU(S,V,N)に対して用いると、新たに状態変数間の関係式を得ることができる。これをマックスウェルの関係式と呼び、この例の他にも多くの有益な関係を得ることができる。
　∂[∂U(S,V,N)/∂V]/∂S = ∂[∂U(S,V,N)/∂S]/∂V = -∂P/∂S|V,N = ∂T/∂V|S,N
マックスウェルの関係式は、単相であれば物質の種類によらず一般に成立することに注意しよう。

□ 状態図の相境界のルール
・シュライネマーカースの束（Schreinemakers' bundle）
　不変点に集まる相境界（線の束）に関する法則
　上記のシュライネマーカース則
・ゼロフラクションライン（Zero fraction line）
　ある相の割合が0になり得る境界を結んだ線。固相線（液相量が0）や液相線（固相量が0）がその例になる。
　多元系になると相の関係が複雑になってくるために、例えば、ある化合物相の析出限界を知りたい、添加元素に伴う析出限界の変化を知りたいなどの場合には、ゼロフラクションラインを用いることになる。
　また、ゼロフラクションラインは多くの相が現われる合金の状態図の組織観察などの実験を元に描こうとするときに便利である。
※ 例えば、３元系合金の等温断面においてゼロフラクションを描いた後に、シュライネマーカス則を考慮すると、相境界を書き入れることができる。
・溶解度積（Solubility product）
・アルケマーデ則（Alkemade theorem）
　2つの異なる相が初晶として晶出する領域の境界線（三元系の液相線投影図における一変系反応線）と2つの相の化学量論組成（例えば、純物質A, B, C）を結んだ直線（アルケマーデ線）との交点における液相線勾配に関する法則である。
　アルケマーデ則を適応する場合、化合物相は調和融解（コングルーエント融解）するものに限る（包晶反応には適用できない）。
※ 状態図の相境界に関しては熱力学的に導かれるいくつかの規則ががあり、これらは計算により状態図を求める場合には必然的に満たされている。しかし実験で状態図を求める場合には、得られた平衡相境界をどのように結んで状態図を描いたらいいのかは上記の四つのルールが参考になる。

□ 合金状態図から合金の諸性質変化を予測する

◇ 全率固溶型合金の場合
　Cu-Ni系合金状態図のようにCuからNiまで100%固溶度でなく、一部の組成で固溶型を示す状態図でも、その部分に限ってはやはり全率固溶型の場合の性質変化曲線を推定してよい。
　全率固溶型合金の機械的性質は一般に次の傾向を持つ。引張り強さ、降伏点、かたさなどは組織軸に対して連続的な山型となり、伸び率、断面収縮率、展延性などは連続的な谷型となる。物理的性質の比電気抵抗曲線は山型となり、熱伝導度は逆に谷型となる。融点、凝固点は状態図に示されるとおりであり、密度はほぼ組織に対して直線的関係にある。
・合金を作った場合、原子半径の違いによる結晶格子のひずみが当然現れ、引張強さは高くなる。純Niと純Cuの引張強さの値が同じならば、その合金の最高の引張強さは50at%Niのところになるはずである。しかし、純Niが純Cuより引張強さが倍も高いので、最高引張強さはややNi側に寄っている。
・伸び率曲線は引張り強さの場合とは反対に純金属側が高く、合金になると減少して連続的な谷型曲線を示している。両金属原子の原子半径の差や、それに伴う転移密度の増加衝突などが、結晶面のすべりによる変形を阻害しているのである。一般に引張強さと伸び率は相反関係にあり、機械設計などにおいて、この調和がつねに要求されている。かたさは素材の塑性変形の難易に関係があり、大体引張強さ曲線と相似の曲線になっている。
・物理的性質の一つとして比電気抵抗値もあわせ示した。電気抵抗は結晶構成因子の周期性に関係があり、合金化すると、全率固溶体の場合はきれいな山型の曲線となり、組成50%のところで最大となっている。

◇共晶型の場合
　固溶体の部分は全率固溶型と同じ曲線傾向を示す。共晶区域では独自の曲線形態を示す。また、同一材料ならば結晶粒の微細な材料は粗大な材料より機械的性質がよい。したがって、共晶点付の引張り強さが最大値となるのは、共晶組織がα、β固溶体の微細粒の集合体のためである。
　比電気抵抗曲線は共晶組成区域でわずかに谷型でほぼ直線に近い曲線である。共晶はPb側固溶体とSn側固溶体の2相の結晶が混在しているため、各相の固溶体の抵抗を合計したものと考えられる。

◇金属間化合物の存在する場合
　標準となる純金属といろいろな2成分系合金との熱起電力を測ると、組成により自由電子の密度が変わるから、熱起電力は一般に組成とともに変化する。
　共晶型合金の場合は、熱起電力曲線は、Cu-Ni系合金のように組成に対し連続的に谷型（逆に山型となる合金もある）の曲線となる。しかし、金属間化合物の場合は、その両側の部分状態図が共晶型であっても、共晶型特有の性質であろうという予測ではない場合がある。例えば、半導体的性格を持つ場合が該当する。

□ 結晶粒の大きさによる強度の変化 - ホール・ペッチ（Hall-Petch）の関係
　金属材料を強化するしくみの探求は、歴史的にもそして今日もなお金属材料の主要な研究課題の一つである。一口に強度というが、実は内容は多様である。弾性変形への抵抗、降伏に対する抵抗、破壊に対する抵抗等々である。基本的な強度は降伏応力であり、多くの研究がこれの向上を目指して行われてきた。その成果はいくつかの基本原理に整理されている。その中の基本的なものが結晶粒の微細化である。
　軟鋼の下降伏点と結晶粒径の-1/2乗との関係を調べた結果を見ると、両者に直線関係があり、結晶粒の微細化が有効な手段であることがわかる。この関係をホール・ペッチの関係という。
　結晶粒径の制御は、加工した金属材料を加熱すると生じる「再結晶」現象を利用して行うことができる。加熱温度は絶対温度で表した融点の1/2程度が目安になる。合金の場合の融点は状態図から知ることができる。
*降伏応力（yield stress）：材料が永久変形（塑性変形）を開始する応力
**N. J. Petch: Phil. Mag., 3 (1958) 1089.
***金属材料固体を塑性変形したうえで加熱すると、固体状態のままで新しい結晶粒が生成される。これが再結晶（recrystallization）である。1時間で試料全体が再結晶する温度を再結晶温度という。この温度は加工量と材料の種類に依存して変化する。再結晶温度の目安が融点の半分程度なのである。
