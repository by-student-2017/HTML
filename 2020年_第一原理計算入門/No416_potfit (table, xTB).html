Å° potfit (need "intel MKL") (ubuntu 18.04 LTS on Windows 10)
(Edit: 29/Jun/2020)
(https://www.potfit.net/wiki/doku.php)

Åû Install (xtb v.6.3.0)
0. sudo apt update
  sudo apt install -y git
1. git clone https://github.com/by-student-2017/xtb-6.3.0_Ubuntu18.04LTS.git
2. mv xtb-6.3.0_Ubuntu18.04LTS xtb-6.3.0
3. cd xtb-6.3.0
4. chmod +x install.sh
5. ./install.sh

Åû Optimization + MD (xtb v.6.3.0)
1. git clone https://github.com/by-student-2017/csh.git
2. cd csh/potfit/xtb
3. chmod +x xtb2force
4. chmod +x run.csh
5. ./run.csh Al2Cu.poscar 50
  (you can get config_all)

Åû MD only (xtb v.6.3.0)
1. git clone https://github.com/by-student-2017/csh.git
2. cd csh/potfit/xtb
3. chmod +x xtb2force
4. chmod +x run_md.csh
5. ./run_md.csh Al2Cu.poscar 50
  (you can get config_all)

Åû Table (xtb v.6.3.0)
0. sudo apt update
  sudo apt install -y git csh
1. git clone https://github.com/by-student-2017/csh.git
2. cd csh/potfit/xtb_table
3. chmod +x xtb2force
  chmod +x run_md.csh
  chmod +x run_xtb_potfit_table
  chmod +x correct_config
4. cp correct_config ~/potfit/util
5. gedit table
  gedit xcontrol
  gedit xcontrol_restart
6. ./run_xtb_potfit_table Al-Cu adp
  (or ./run_xtb_potfit_table Al-Cu eam)
Å¶ Debian 10.0: sudo apt install -y bc
