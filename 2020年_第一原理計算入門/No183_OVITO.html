□ RDF (Radial distribution function, g(r))
1. OVITO > *.cfg
2. Add modification... (on right side) > Correlation analysis
3. Cutoff radius: 5
  Number of histgram bins: 400
  [check] conpute partial RDFs
4. File > Export File
  Data Series Test File (*) [rdf.dat] > OK
5. Data object: Radial distribution function > OK

◇ Excel (RDFのプロット）
1. ファイル > 開く
  すべてのファイル(*.*): rdf.dat > 開く
2. ● カンマやタブなどの区切り文字によってフィールドごとに区切られたデータ > 次へ
3. ? スペース > 次へ > 完了
4. 2列目を左に1マス移動させる（コピー&ペーストして左端にしてもよい）
5. Ctrl + A
6. 挿入 > グラフ > 散布図（直線）
※ 1列目を削除するとなお良い

□ Movie (making *.gif file)
1. OVITO > *.cfg (e.g. gcmc_run.200100.cfg)
2. Camera mark
  Rendering range
   [check] Range:
 Render output
   [check] Save to file
   Choose...
   GIF Animation (*.gif): movie.gif > Save
3. Render Active Viewport
※ complete animation" is OK
