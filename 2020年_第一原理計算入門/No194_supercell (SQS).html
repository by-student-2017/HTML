Å° Supercell (https://orex.github.io/supercell/)
(ubuntu 18.04 LTS (64 bit) on windows 10 (64 bit))
(Edit: 10/Jul/2020)

Å† Install
0. sudo apt update
  sudo apt install -y git wget openbabel
  cd ~
1. git clone https://github.com/orex/supercell.git
2. cd supercell
3. wget https://orex.github.io/supercell/exe/supercell-linux.tar.gz
4. tar zxvf supercell-linux.tar.gz

Å† Environment settings
1. echo ' ' >> ~/.bashrc
  echo '# supercell environment settings' >> ~/.bashrc
  echo 'export PATH=$PATH:$HOME/supercell' >> ~/.bashrc
  echo 'export BABEL_DATADIR=$HOME/supercell' >> ~/.bashrc
2. bash

Å† Usage
1. cd ~/supercell/data/examples/Ca2Al2SiO7
2. supercell -i Ca2Al2SiO7.cif -s 1x1x2 -m -o Ca2Al2SiO7_cell_1x1x2

Å† Options
-i: input file
-o: output file
-s: supercell size
-m: present in the command line

Å° cif2cell-lammps (https://github.com/andeplane/cif2cell-lammps)
(ubuntu 18.04 LTS (64 bit) on windows 10 (64 bit))

Å† Install (cif2cell-lammps)
0. sudo apt update
  sudo apt install -y git python
  cd ~
1. git clone https://github.com/andeplane/cif2cell-lammps.git
2. cd cif2cell-lammps
3. tar zxvf PyCifRW-3.3.tar.gz
4. cd PyCifRW-3.3
5. sudo python setup.py install
5. cd ~/cif2cell-lammps
6. sudo python setup.py install

Å† Test
1. cd ~/cif2cell-lammps/cifs
2. cif2cell BaTiO3_cubic.cif --no-reduce -p lammps
  (you can get BaTiO3_cubic.data)
3. cif2cell BaTiO3_cubic.cif --no-reduce -p cfg
  (you can get BaTiO3_cubic.cfg)
  (sed -i '/^#/d' BaTiO3_cubic.cfg) or (sed -i '/#\*/d' BaTiO3_cubic.cfg)
4. cif2cell BaTiO3_cubic.cif --no-reduce -p vasp --vasp-cartesian-positions --vasp-format=5 -o case.poscar
  (you can get case.poscar)
5. cif2cell BaTiO3_cubic.cif --no-reduce -p dftb
  (you can get BaTiO3_cubic.gen)
  (sed -i '/^#/d' BaTiO3_cubic.gen) or (sed -i '/#\*/d' BaTiO3_cubic.gen)

Å° Special Quasirandom Structures (SQS)
(Supercell + cif2cell-lammps + Lammps)

Å† Examples (FeCrW)
0. cd ~
1. git clone https://github.com/by-student-2017/supercell-examples.git
2. cp -r ~/supercell-examples/* ~/supercell/data/examples
3. cd ~/supercell/data/examples/FeCrW-SQS-OPT
4. chmod +x df_cfg_lmp.bash
5. ./df_cfg_lmp.bash 1x1x2
  (or ./df_cfg_lmp.bash 2x2x1, etc)
6. cat SQS-1x1x2
  (or cat SQS-2x2x1, etc)

(you can get Total energy [eV] for every *.cif)
(FeCrW-SQS-OPT is minimize version)

Note: Other systems can be calculated by changing the description of the potential in "in.lmp" so that it corresponds to the element in the ".cif" file.

(Edit: 15/Jul/2020)

Å† cif2cell-informal
0. sudo apt update
  sudo apt install -y git python python-setuptools python-dev
  cd ~
1. git clone https://github.com/by-student-2017/cif2cell-informal.git

Å† PWscf (v.6.4.1)
1. sudo apt update
  sudo apt install -y gcc g++ build-essential gfortran libopenblas-dev libfftw3-dev libopenmpi-dev wget
  cd ~
2. wget https://github.com/QEF/q-e/archive/qe-6.4.1.tar.gz
3. tar zxvf qe-6.4.1.tar.gz
4. cd q-e-qe-6.4.1
5. wget https://github.com/QEF/q-e/releases/download/qe-6.4.1/backports-6.4.1.diff
6. patch -p1 --merge < backports-6.4.1.diff
7. ./configure
8. make pw
9. sudo make install
(/usr/local/bin/pw.x)

Å† Examples (PbSnTe) (beta version)
0. sudo apt update
  sudo apt install -y git
  cd ~
1. git clone https://github.com/by-student-2017/supercell-examples.git
2. cp -r ~/supercell-examples/* ~/supercell/data/examples
3. cd ~/supercell/data/examples/PbSnTe-SQS-OPT
4. chmod +x df_cfg_qe.bash
5. export OMP_NUM_THREADS=1
6. ./df_cfg_qe.bash 1x1x2 4
  (last number is number of cpus)
7. cat SQS-1x1x2
(you can get Total energy [eV] for every *.cif)

Note1: Since "MSDA" (units: r2/Angstrom2) uses an initial structure, the initial structure must have an ideal atomic arrangement. For example, in the case of the FCC structure, the initial structure (cif file) of the FCC whose atomic positions are not displaced is required.
Note2: Beta version remains because we can't move to the final test because of PC spec restrictions.
