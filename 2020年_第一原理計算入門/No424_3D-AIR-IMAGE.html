□ Preparation *.xtl file from *.cif
1. Material Project > CIF > Conventional Standard
  (For example,)
・MoS2: https://materialsproject.org/materials/mp-2815/
・beta-Ga2O3 (C2/m): https://materialsproject.org/materials/mp-886/
・GaN: https://materialsproject.org/materials/mp-804/
・C (diamond): mp-66: C (cubic, Fd-3m, 227) (materialsproject.org)
2. VESTA.exe (you can get VESTA-win64.zip from https://jp-minerals.org/vesta/jp/download.html)
3. File > Open... > *.cif > Open
  (or drag&drop *.cif on VESTA)
4. File > Export Data... > Fractional Coordinate (*.xtl) > Save
 
□ Simulation of XFH (e.g., MoS2.xtl case)
1. 3dairimage.exe
2. Holography > TMSP cluster
3. Import unit cell > *.xtl
4. (Rename) "Name [Mo2 S4]" to "Name [MoS2]" on "Define Crystal and surface"(window)
  > Save as New Crystal > Yes > OK
5. (Select) MoS2 on "TMSP cluster"(window) > Create Cluster
6. (Check) Mo on "Candidate atoms for EMITTER"(column)
  > (increase) 5 to 30 on "Cluster Radius"(column) > Create Cluters
  > Save as xml_cl(TMSP) > MoS2.xml_cl > Save
7. XFH on "3D-AIR-IMAGE"(window)
  > Simulation for X-ray fluorescence hologram[fast]
8. (1) Load Cluster > MoS2.xml_cl > Open > OK on "Dialog"(window)
9. (2) Energy (keV) list of incident photon
  (For example, 7 data for every 0.5 (keV) step)
  -----
  10
  10.5
  11
  11.5
  12
  12.5
  13
  -----
10. Anmalous (skip in this case)
11. Vibration & Fluctuation
  (Input "K") > TD(K) ->> Vibration (A)
12. (3) Hologram Image
  > (Select projection, e.g., Equidistant cylindrical) 
  > (Set value of Degree/Pix (resolution)) 
  > calculate (and wait for a long time)
  > OK on "Finished"
(You can get *.xmlsrs and *.xml_air)
13. Holography on "3D-AIR-IMAGE"(window)
  > Sphere: Convert Map Projection
14. Holography on "3D-AIR-IMAGE"(window)
  > Printout image v2(Write Numerals) 
  > Atom > Load TMSP Cluster
 
◇ Barton method
1. (Drag&drop) *.xmlsrs (from above step 12) on "3D-AIR-IMAGE"(window)
2. XFH > Atom image reconstruction
3. (Change Accuracy (Spherical Mesh size):
  (63 = 1 degree)
  (The lower the accuracy, the faster the calculation speed.)
  (This value depends on the photon energy.)
3. CALC SLICE on "XFH RECONSTRUCTION 0.0.27"(window)
4. VOLUME Reconstruction (for a long time)
5. Iso-Surface on "3D-AIR-IMAGE"(window) > (set CONTOIR(%)) > OK
6. SMOOTH SHADING on "3D-AIR-IMAGE"(window)
7. XFH  on "3D-AIR-IMAGE"(window) > XFH 3DImage Radial Correction
8. VolumeImage on "3D-AIR-IMAGE"(window) > Slice image
  > (set Plane index) > Update
 
◇ SPEA-L1
1. (Drag&drop) *.xml_air (from above step 12) on "3D-AIR-IMAGE"(window)
2. XFH > Atom image reconstruction
3. TEST > SPEA-L1
4. 1) Create real space voxel > OK
5. 2) Parameter ologram, Scattering pattern matrix > OK
6. 3) Parameter for L1 method
  > L1 Threashold(%)= 50  (<- Empirically good value)
  > Threshold scaling  (<- Empirically good value)
  > OK
7. RUN
8. NSmooth=3 > DISPLAY 3D ATOMIC IMAGE
9. (Check) ERROR FUNC > (Change Distance(A)) > DISP CROSS SECTION
10. (Click) RMSE, L1 parameter or voxel max value
 
◇ Get information (experimental data)
1. (Drag&drop) *.xml_air
2. Tool > Sdit image metadata
 
◇ Treat background (experimental data)
1. (Drag&drop) *.xml_air
2. XFH > XFH Background
3. (Change two value) Poly. Terms: 
  (start with a low order)
4. (Check background)
  (Increase order of "Poly. Terms:" until background become flat)
5. NEW IMAGE to Mainwindow
  (If you can see Kossel line, it is good)
 
□ Simulation of PEH
1. Holography on "3D-AIR-IMAGE"(window)
  > TMSP 5/SPEA-L1(MEM)
2. Load Cluster > *.xml_cl > Open > OK
3. Calculate Potential > Calculate or Load potential > Close
4. New Emitter
  > (Set Kinetic energy/(Innter potential is inclded) 1000.00 > "<- Add"
  > Set emitted electron > Close
  > Photoelectron (Dipole transition) > Polarization of light [ Circular+1] > Set Eitted electron > Close
  ----------
  DIANA: Photoelectron, Auger
  (Unpolarized case) DA30 or RFA: Auger, Auger
  ----------
5. New phase shift > Calculate phase shift > (Check "Phase shifts (degree)") > View Inside function [Pr(r)*r] > Close
6. New Scattering Calculation > (Select "Type", e.g., [Fast-FF-Multipule_scattering]) > Start Calculation > *.xml_sf > Save  (and wait for a long time)
7. Hologram image
  > (Check "Use meam free path (A)")
  > (Set "Temperature (K)" [300]) > (Set "TD(K)") > [->Calc Vibration]
  > List of pattern (Usually, check all)
  > (Select projection) > (Set value of Degree/Pix (resolution)) > Calculate
8. OneImage > Operate by Image > New Image
  (Divide the pattern by "I0")
9. OneImage > Spherical Defocus > 
  > Angular resolution (deg)
  ----------
  DIANA: 3 (dgree)
  DA30, RFA: Not use
  ----------
  > New Image

□ Experimental data of PEH
1. (Drag&drop) *.xml_air on "3D-AIR-IMAGE"(window)
2. Holography on "3D-AIR-IMAGE"(window)

4. Calculate Potential > Calculate or Load potential > Close
5. New Emitter
  > (Set Kinetic energy/(Innter potential is inclded) 1000.00 > "<- Add"
  > Set emitted electron > Close

  > OK

13. Display real space

  > Display 3D
  > Graph RMSE

■ PEH
1. A sample is irradiated with soft x-ray.
2. Photoelectron is excited.
3. Electron wave is scattered by surrounding atoms.
4. Interference pattern between the scattered waves and emitted wave appears.  This pattern is photoelectron hotogram.
※ Forwad focusing peak indicates the direction of scatterer.
※ Interference pattern implies the atomic distance between emitter and scatterer.
