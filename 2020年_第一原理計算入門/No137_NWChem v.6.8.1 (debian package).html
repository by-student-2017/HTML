Å° NWChem v.6.8.1 (debian 10.0 on windows 10 (64 bit))
(Edit: 10/Jun/2020)

Å† get information on OS
1. grep -H "" /etc/*version ; grep -H "" /etc/*release
(In my case, OS is "buster")

Å† Install
1. sudo apt update
2. sudo apt install -y nwchem

Å† check
1. which nwchem

Å† example settings
1. cd $HOME
2. mkdir nwchem
3. cd nwchem
4. cp -r /usr/share/doc/nwchem/examples ./

Å† jmol
1. sudo apt install -y jmol

Å† XLaunch (WSL case)
ÅEubuntu side
1. sudo apt-get update
2. sudo apt-get install -y xfce4-terminal
3. sudo apt-get install -y xfce4

Å† N2 (MP2 optimization and CCSD(T))
1. cd ~/nwchem/examples
2. cat << EOF > n2-mp2-opt-to-ccsdt.nw
start n2 

geometry
  symmetry d2h
  n 0 0 0.542
end

basis spherical
  n library cc-pvtz
end

mp2
  freeze core
end

task mp2 optimize

ccsd
  freeze core
end

task ccsd(t)
EOF
3. cat n2-mp2-opt-to-ccsdt.nw
  (Ref: http://www.nwchem-sw.org/index.php/Sample)
4. set OMP_NUM_THREADS=1
5. mpirun -np 4 nwchem n2-mp2-opt-to-ccsdt.nw | tee n2-mp2-opt-to-ccsdt.out
