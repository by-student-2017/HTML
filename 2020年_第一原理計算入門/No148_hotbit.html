Å° hotbit
(debian 10.0 or ubuntu 18.04 LTS (64 bit) on windows 10 (64 bit))
(https://github.com/pekkosk/hotbit)
(Edit: 26/Aug/2020)

Å† libxc-4.3.4
1. sudo apt update
  sudo apt install -y cmake gcc g++ build-essential
  sudo apt install -y python-dev python-distutils python-setuptools
  cd ~
2. wget https://www.tddft.org/programs/libxc/down/4.3.4/libxc-4.3.4.tar.gz
3. tar zxvf libxc-4.3.4.tar.gz
4. cd ~/libxc-4.3.4
5. sudo python setup.py install

Å† Install (hotbit)
1. sudo apt update
  sudo apt install -y ase python-ase
  sudo apt install -y libblas-dev liblapack-dev libopenblas-dev
  sudo apt install -y python-numpy python-scipy python-matplotlib
  (sudo apt install -y libxc-dev)
  cd ~
2. git clone https://github.com/pekkosk/hotbit.git
3. cd hotbit
4. python setup.py install --home=~/hotbit

Å† environment settings
1. echo ' ' >> ~/.bashrc
  echo '# hotbit' >> ~/.bashrc
  echo 'export HOTBIT_DIR=$HOME/hotbit/lib/python' >> ~/.bashrc
  echo 'export PATH=$HOME/hotbit:$PATH' >> ~/.bashrc
  echo 'export HOTBIT_PARAMETERS=$HOME/hotbit/param' >> ~/.bashrc
  echo 'export PYTHONPATH=$PYTHONPATH:$HOTBIT_DIR' >> ~/.bashrc
2. bash
or
1. cd ~/hotbit
2. bash env_exports

Å† test
1. cd $HOTBIT_DIR/hotbit/test
2. python test.py

Å† examples
1. cd ~/hotbit/examples
2. cd AuAu_parametrization
3. python AuAu.py
(you can get AuAu_repulsion.pdf)
