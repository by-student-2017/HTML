------------------------------------------------------------------------------
■ NWChem v.7.0.0 (ubuntu 20.04 LTS (64 bit) on windows 10 (64 bit))
(Edit: 9/Jun/2020)

□ Install
1. sudo apt update
2. sudo apt install -y nwchem

□ check
1. which nwchem

□ example settings
1. cd $HOME
2. mkdir nwchem
3. cd nwchem
4. cp -r /usr/share/doc/nwchem/examples ./
or
□ examples (from github)
1. cd $HOME
2. mkdir nwchem
3. cd nwchem 
(3.5. sudo apt install -y git)
4. git clone https://github.com/by-student-2017/nwchem-examples-7.0.0.git
5. mv nwchem-examples-7.0.0 examples

□ jmol
1. sudo apt install -y jmol

□ XLaunch (WSL case)
・ubuntu side
1. sudo apt-get update
2. sudo apt-get install -y xfce4-terminal
3. sudo apt-get install -y xfce4
・windows side
1. get VcXsrv Windows X Server from
  https://sourceforge.net/projects/vcxsrv/

□ N2 (MP2 optimization and CCSD(T))
1. cd ~/nwchem/examples
2. set OMP_NUM_THREADS=1
3. mpirun -np 4 nwchem  N2-MP2-opt-to-CCSDT.nw | tee N2-MP2-opt-to-CCSDT.out
or
1. cd ~/nwchem/examples
2. cat << EOF > n2-mp2-opt-to-ccsdt.nw
start n2 

geometry
  symmetry d2h
  n 0 0 0.542
end

basis spherical
  n library cc-pvtz
end

mp2
  freeze core
end

task mp2 optimize

ccsd
  freeze core
end

task ccsd(t)
EOF
3. cat n2-mp2-opt-to-ccsdt.nw
  (Ref: http://www.nwchem-sw.org/index.php/Sample)
4. set OMP_NUM_THREADS=1
5. mpirun -np 4 nwchem n2-mp2-opt-to-ccsdt.nw | tee n2-mp2-opt-to-ccsdt.out
