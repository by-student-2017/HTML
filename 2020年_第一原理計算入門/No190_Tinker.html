■ Usage (FFE)

□ VESTAとOpenBableを用いた構造ファイルの作成
1. VESTA (remove symmetry -> P1) > cif
2. cif > Openbabel > txyz--tinker XYZ format
  > [check] Write atom types for the MM3 forcefield
  > Output file case.txyz
  > CONVERT
3. gedit *.key
4. ./'Force Field Explorer' > Open > txyz file 
5. Display > Ball & Stick
6. Modeling Commands > [Analyze] -> [Minimize], etc > GO

□ output file
[Minimize]: *.xyz (output file)
[Newton]: *.xyz (output file)
[Optimize]: *.xyz (output file)
