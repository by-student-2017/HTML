□ Tutorial -1- (hotcent)
(https://gitlab.com/mvdb/hotcent/-/blob/master/tutorials/tutorial_1.ipynb)

□ DFTB+ [I5]
-----
we plan to release the parameterisation tools in the autumn release 19.2 this year. As they were used mostly internally so far (or given away to colaboration partners), their user interface is not as robust as I would wish and I want to fix that before letting users to make parameters with it. Especially, they should be robust enough of preventing users from making trivial mistakes without noticing it.

For sure, Maximes way of using Hotbit to generate a parametrisation is a possible alternative, so you may give it a try, if you want to start immediately.
