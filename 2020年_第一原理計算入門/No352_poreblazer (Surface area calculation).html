■ PoreBlazer v4.0+ (Edit: Dec/09/2020)
   (ubuntu 18.04 LTS or Debian 10.0 on Windows 10)
 
□ Install
1. cd ~
2. git clone https://github.com/by-student-2017/PoreBlazer-v4.0.git
3. cd PoreBlazer-v4.0
4. chmod +x ./install.sh
5. ./install.sh

□ Examples (ZTC)
1. cd ~/PoreBlazer-v4.0
2. cd Examples
3. cd ZTC
4. chmod +x ./run.sh
5. ./run.sh

□ cif2cell installation
1. cd ~/PoreBlazer-v4.0
2. chmod +x ./install_cif2cell.sh
3. ./install_cif2cell.sh
 
□ Calculation
1. cd ~/PoreBlazer-v4.0
2. cd Examples/seq
3. chmod +x ./run.sh
4. ./run.sh
  (you can get results.txt and results.csv for machine learning)
  (Please, put cif files into cif directory.)

■ poreblazer v3.0.5 (Edit: Dec/07/2020)
  (https://github.com/richardjgowers/poreblazer)

□ Feature
- Accessible surface area (with Nitrogen atom as a probe)
- Geometric pore size distribution
- Structure density
- Helium pore volume
- Maximum pore size
- Pore limiting diameter

□ Install (ubuntu 18.04 LTS on windows10)
0. cd ~
1. git clone https://github.com/richardjgowers/poreblazer.git
2. cd poreblazer/src
3. make
(you can get poreblazer.exe)

□ Example (HKUST1)
1. cd ~/poreblazer/HKUST1
2. vi defaults.dat

◇ cif -> xyz file
・ATEN (good and free soft): https://www.projectaten.com/aten
  Aten-2.1.9.exe (Win32) (Windows installer)
  1. Open > Filter (cif) > HKUST1.cif > Open
  2. Save As > HKUST1.xyz > Save
・VESTA (need check carefully)
  HKUST1.cif
  1. Edit > Bonds ... > Do not search atoms beyond the boundary (of all atoms) (on Boundary mode)  > OK
  2. Objects > Boundary... > x(max) = y(max) = z(max) = 0.9999
  3. File > Export Data... > XYZ file (*.xyz)
  4. Do you want to save hidden atoms too? (no)
・cif2cell (GitHub - by-student-2017/cif2cell-informal)
  1. cif2cell -p xyz -f HKUST1.cif

◇ Attention
- The current upper pore diameter is set to 20 A. 
  Change it in the defaults.dat if you anticipate larger pores  in the system
