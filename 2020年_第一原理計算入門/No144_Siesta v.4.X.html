Å° Siesta v.4.0 (ubuntu 18.04 LTS (64 bit) on windows 10 (64 bit))
(https://departments.icmab.es/leem/siesta/)
(https://gitlab.com/siesta-project)

Å† Install (OpenMPI)
1. sudo apt update
  sudo apt install -y make g++ gcc build-essential gfortran libopenmpi-dev libopenblas-dev
2. git clone https://gitlab.com/siesta-project/siesta.git
3. cd siesta
4. cd Obj
5. sh ../Src/obj_setup.sh
6. cp gfortran.make arch.make
7. gedit arch.make
-----
FC = mpif90
COMP_LIBS = libsiestaLAPACK.a libsiestaBLAS.a -lopenblas
-----
8. make
Å¶ cpu(AVX) case: FFLAGS = -O2 -fPIC -ftree-vectorize -mavx

Å† Test
1. cd $HOME/siesta/Tests
2. make SIESTA="mpirun -np 4 ~/siesta/Obj/siesta"
