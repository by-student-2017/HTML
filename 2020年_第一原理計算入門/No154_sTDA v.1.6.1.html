Å† install (sTDA v.1.6.1)
0. cd ~
1. git clone https://github.com/grimme-lab/stda.git
2. cd stda
3. wget https://github.com/grimme-lab/stda/releases/download/v1.6.1/stda_v1.6.1
4. chmod +x stda_v1.6.1

Å† environment settings (sTDA v.1.6.1)
1. gedit ~/.bashrc
  export STDAHOME=~/stda
  export PATH=$PATH:$STDAHOME

Å† check (sTDA v.1.6.1)
1. stda_v1.6.1 -h

Å† install (xtb4stda v.1.0)
0. cd ~
1. git clone https://github.com/grimme-lab/xtb4stda.git
2. cd xtb4stda
3. mkdir exe
  cd exe
4. wget https://github.com/grimme-lab/stda/releases/download/v1.6.1/xtb4stda
5. chmod +x xtb4stda

Å† environment settings (xtb4stda v.1.0)
1. gedit ~/.bashrc
  export XTB4STDAHOME=~/xtb4stda

  export OMP_NUM_THREADS=4
  export MKL_NUM_THREADS=4

(2. ulimit -s unlimited)

Å† check (xtb4stda v.1.0)

Å† manual (sTDA)

(1.5. sudo apt install -y evince)

Å† Usage (sTDA-xTB Procedure)
1. xtb4stda *.coord
  (or xtb4stda *.xyz)
2. stda_v1.6.1 -xtb -e 10
  (or stda_v1.6.1 -xtb -e 10 -rpa) (sTD-xTB)
  (or stda_v1.6.1 -xtb -e 10 -sf) (SF-sTD-xTB)
Å¶ "-e 10" mean "compute all excited states up to 10eV"

Å† examples
0. sudo apt install -y git
1. cd ~/stda
2. git clone https://github.com/by-student-2017/sTDA-examples.git
3. mv sTDA-examples examples

Å† Feature
ÅEThe sTDA program for computing excited states and response functions via simplified TD-DFT methods (sTDA, sTD-DFT, and SF-sTD-DFT).
ÅEA simplified Tamm-Dancoff density functional approach for electronic excitation spectra.

Å† indigo (UV) (sTDA-xTB)
1. cd ~/stda/examples/UV
2. xtb4stda indigo.xyz
3. stda_v1.6.1 -xtb -e 11.5
  (or stda_v1.6.1 -xtb -e 11.5 -rpa) (sTD-xTB)
  (or stda_v1.6.1 -xtb -e 11.5 -sf) (SF-sTD-xTB)
4. gedit tda.dat
-----(before)
 NM
 VELO
-----
-----(after)
 UV
-----
5. g_spec < tda.dat
6. gnuplot
  set ylabel "{/Symbol e} / L mol^{-1} cm^{-1} "
  set xlabel "energy / eV"
  set xrange [1:12]
  plot "spec.dat" w l t "UV"
Å¶ option in tda.dat
  UV: Calculate absorption spectra (if not specified, CD is calculated)
  VELO: Use the velocity representations of R and f (the latter if UV is also specified).
  NM: Plot spectrum in nanometer scale instead of eV.
  WIDTH: Half width at 1/e maximum of Gaussian curve (in eV).
  SHIFT: Energy shift applied to whole spectrum (in eV).
  LFAKTOR: Scaling factor for the stick spectrum printed to rots.dat.
  RFAKTOR: Scaling factor for the broadended spectrum printed to spec.dat.

Å¶ Please, see head of tda.dat.
