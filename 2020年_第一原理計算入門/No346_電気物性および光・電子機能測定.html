■ 電気物性測定
　物質の電気的性質は、物質内部に存在する電荷の運動に起因する。電場により電荷が局所的に変位して分極を形成すると誘電性、長距離移動して電流を生ずると導電性となる。試料の電圧刺激に対する電荷応答から誘電緩和、誘電スペクトルが測定される。

■ 誘電性の基礎
◇ 誘電率
　誘電率は電気変位（電束密度）Dと電場Eの間の比例係数として定義される。
　D = εE
　電場を印加すると、誘電体の内部には、
1) 原子内の原子核に対する電子の相対的変位（電子分極）
2) 正負のイオン間距離の変化（イオン分極）
3) 永久双極子の回転（配向分極）
の3種類の機構により分極Pが形成される。通常、分極は電場と比例関係を示す。
　P = ε0ΧE
ここで、Χは電気感受率、ε0=8.854x10-12 F/mは真空の誘電率である。誘電体を電極に挟んでコンデンサーを作ると、電極には、これらの分極を打ち消す電荷と誘電体内に電場を作る電荷がたまる。電気変位はこれらの電荷の和の単位面積当たりの値に等しい。
　D = ε0E + P = ε0(1+Χ)E
これらの式より、誘電率εが決まるが、実際にはこれをε0で割った日誘電率
　εr = ε/ε0 = 1+Χ
を、単に誘電率と呼んで使うことが多い。

◇ 誘電緩和現象
　時間とともに変動する電場に対して、誘電応答は時間的に遅れて生ずる。これを誘電緩和現象という。ステップ状の電場Esを時間t=0において加えたときの電気変位の時間変化をDs(t)とすると、時間の関数としての誘電率が次の式と決まる。これを誘電余効関数と呼ぶ。
　εs(t) = Ds(t)/Es
角周波数ωで正弦的に変動する電場、例えば下の式に対して、
　E(t) = E0*cos(ωt)
電気変位は振幅がD0、位相がδ遅れた信号として観測される。これを次の式のように書き、
　D(t) = D'*cos(ωt) + D''*sin(ωt)
この余弦および正弦成分の振幅から複素誘電率を次の式と決めることができる。
　ε* = ε' + i*ε'' = D'/E0 - i*(D''/E0)
ここで実部ε'を貯蔵誘電率、虚部ε''を損失誘電率と呼び、両者の比を損失正接と呼ぶ。
　tan(δ) = ε''/ε'
　線形の緩和現象については、Boltzmannの重ね合わせの原理が成立し、複素誘電率と誘電余効関数は互いにフーリエ変換の関係にある。
　ε*(ω) = ∫∞0 {∂εs(t)/∂t}*exp(-iωt) dt
　誘電余効関数は、最も単純な場合、次の式のような一つの緩和時間τで変化する指数関数となる。
　εs(t) = ε∞ + Δε*{1-exp(-t/τ)}
　ここで、ε∞を瞬間誘電率、Δεを誘電緩和強度と呼ぶ。これに対応する複素誘電率は、[ε*(ω) = ∫∞0 {∂εs(t)/∂t}*exp(-iωt) dt]の式を用いると、次式のような周波数の関数となる。
　ε*(ω) = ε∞ + Δε/(1+iωτ)
　この式はデバイ関数と呼ばれ、ωτ=1において実部はΔεだけ減少し、虚部はピークを示す周波数依存性を示す。現実の物質では、一般に複数の緩和過程が存在し、緩和関数は時間や周波数に対して[εs(t) = ε∞ + Δε*{1-exp(-t/τ)}]や[ε*(ω) = ε∞ + Δε/(1+iωτ)]よりずっと穏やかに変化する。

◇ 誘電応答測定システム
　試料の電圧刺激に対する電荷応答から、種々の誘電応答関数を測定するシステム（を議論する）。電圧波形はデジタル的に任意に合成され、アナログ信号に変換して試料に印加する。電荷増幅器によって測定される試料の電荷応答は、印加電圧とともに一定の時間間隔でデジタル信号に変換し、コンピュータを用いたフーリエ変換を中心とする演算処理により誘電余効関数や複素誘電率などを求めることができる。
　印加電圧には、矩形波、三角波、正弦波、あるいはこれらを組み合わせたさまざまな波形が用いられる。矩形波電圧を印加し、電荷応答を一定の時間間隔ΔtでAD変換を行ってQ(I)とV(I)を得れば、t=IΔt（I=1,2,3,...）における時間領域の誘電測定になる。角周波数ωの正弦波を用い、1周期を正確に等分する時間でサンプリングしたデータに対し、デジタルフーリエ変換を行うと、電荷と電圧信号を複素数として決めることができ、両者の比から複素誘電率を決めることができる。
　本システムを誘電測定に利用した場合、周波数域1 mHz から 1 MHzにおいて精度0.1%は容易に実現できる。低周波数域では、電荷増幅器の安定性に注意を払う必要がある。高周波数域では、各種増幅器やAD変換器の周波数特性の補正、浮遊インダクタンスや抵抗の補正が必要になる。加算平均によるS/N比の向上と、標準試料による補正が最終的な精度を支配する。

◇ 広帯域誘電緩和スペクトル測定
　現在（2006年）、誘電緩和スペクトルは1 μHzから1 THzまで18桁にわたる測定が可能となっている。このような超広帯域測定を行なうには、いくつかの装置を組み合わせる必要がある。このような超広帯域測定を行なうには下の図に示すように、いくつかの装置を組み合わせる必要がある。


　誘電応答関数の直接測定は、1 MHz以下の周波数域をカバーすることができる。1 mHz以下の超低周波数では、正弦波測定は定常状態を実現するために膨大な時間を必要とするので、階段波電圧を用いた時間領域の測定が行なわれる。得られたステップ応答関数をフーリエ変換により周波数領域に変換すると、1周期分の時間で、それより1から2桁高い周波数までの複素誘電率を同時に測定できる。100 Hzから10 MHz域の測定は、容量や抵抗を直読できるインピーダンスアナライザーやLCRメーターと呼ばれる装置を使うことができる。
　1 MHz以上の高周波数域では、電場は回路中を伝搬する電磁波としての性格が強くなるので、誘電率の異なる物質の界面で起きる電磁波の反射を測定する反射法を利用した測定が行なわれる。時間領域で行う反射法はTime Domain Reflectometry（TDR）と呼ばれ、フーリエ変換により比較的簡便に高周波域のスペクトルを測定する手法として貢献した。反射法を周波数領域で行なう装置は、1 MHzから40 GHzをカバーし、高周波インピーダンスアナライザーあるいはネットワークアナライザーとして市販されている。
　10 GHzから100 GHz帯では空洞共振法も有力な手法である。これは試料を含めて構成される空洞共振器の共振周波数やQ値の変化から誘電率と損失を求めるものである。周波数を変えるのがやっかいであるが、反射法より高精度が実現し、高周波低損失誘電体の測定には不可欠の手法である。100 GHzから1 THz域は遠赤外域に相当し、ここでは光としての電磁波の吸収から誘電率を求める方法がとられる。

■ 誘電緩和スペクトルの測定例
　誘電分極の三つの機構のうち電子分極とイオン分極は通常の誘電測定では高周波極限誘電率として観測される。配向分極の寄与は双極子を担う分子の運動を伴って発現し、誘電緩和として観測される。
◇ 非晶性高分子
　非晶状態の高分子鎖は、ガラス転移温度以上で活発な熱運動を行い、顕著な誘電緩和をもたらす。分子量4000のポリプロピレンオキシド（PPO）について-60℃で測定した広帯域誘電スペクトル（を例として見る。左の縦軸はε'/ε0、右の縦軸はε''/ε0、横軸はf/Hz）。30 Hz近傍の最も強い緩和βは主緩和あるいはセグメントモード緩和と呼ばれ、非晶分子鎖のミクロブラウン運動に帰着されるものである。低周波数側の緩和αはノーマルモードと呼ばれ、分子鎖に平行な双極子モーメントの成分の揺らぎが分子鎖全体の運動として現われるものである。高周波数側の小さな緩和γは副緩和あるいはローカルモード緩和と呼ばれ、セグメント内部の局所的な運動によるもので、ガラス転移点以下の温度でも観測される。
　主緩和と副緩和の組は高分子だけではなく極性の低分子液体においても同様に観測される。
　個々の緩和のスペクトルは、式[ε*(ω) = ε∞ + Δε/(1+iωτ)]に示したデバイ（Debye）型の単一緩和関数より緩やかな周波数変化を示し、次の式により再現することができる。
　ε*(ω) = ε∞ + Σk{Δε/[(1+(iωτ)βk]αk}
ここで、右辺第二項はHavrliak-Negami式と呼ばれ、デバイ関数に緩和時間の分布を表すパラメータα、βを導入したもので、Cole-Cole式（α=1, 0<β<1）とDavidson-Cole式（0<α<1, β=1）を統合したものである。曲線当てはめを行うと、緩和時間、緩和強度など、緩和を規程する物理量を最適パラメータとして決めることができる。
　温度を上昇させるとスペクトルは高周波数側に移動する。これは昇温により分子運動が速くなり、τが小さくなるからである。温度を広く変化させて測定したスペクトルから決めた緩和時間の対数を、絶対温度の逆数に対してプロットする（縦軸にlog(τ)、横軸に1/T）と分散地図と呼ばれる図が得られる。主緩和の緩和時間は-70℃付近で急激に大きくなり、その温度依存性は次式で表される。
　τ = τ∞*exp{B/(T-T0)}
　この式は、VTF（Vogel-Tammann-Fulcher）式と呼ばれ、従来WLF（Williams-Landel-Ferry）式と呼ばれた式と等価である。ここでT0は緩和時間が無限大に発散し、分子運動が完全に凍結する温度に相当する。実際にはT0は観測されず、温度を下げていくと例えばτ=103 sとなる温度以下では平衡状態を実現できず、非平衡状態のまま運動は凍結してしまう。この温度をガラス転移点Tgと呼ぶ。PPOの場合、Tgはほぼ-70℃となる。多くの非晶性高分子で共通に、τ∞は1 ps程度になり、高温極限のセグメント運動の速さを与える。またBは1000 K程度の値をとり、RB=10 KJ/mol（R:気体定数）のオーダーの高温における活性化エネルギーを与える。ノーマルモード緩和はセグメントモード緩和と同様のVTF型の温度依存性を示し、Tgで両者は一致する。これに対しローカルモード緩和は1/Tに対して直線的に変化するArrhenius型（T0=0 K）となり、高温で主緩和と合流する。このような主緩和と副緩和の組み合わせは、非晶性高分子に共通に観測される。
◇ 結晶性高分子
　結晶中の高分子鎖の運動による緩和を、結晶緩和という。代表例としてポリフッ化ビニリデン（PVDF）のスペクトル（をとりあげる。誘電率の虚部を温度と周波数に対して3次元表示すると）。PVDFはさまざまな結晶多形をとることで知られており、ここで議論するスペクトルはTGTG'鎖が反平行パッキングした無極性Ⅱ型結晶を約50%含む半結晶性試料の結果である。低温側のβ過程（スペクトルではピークが見える）は非晶域のセグメント運動による主緩和である。高温低周波数側の緩和（α過程、スペクトルではピークや肩構造が見える）が結晶緩和で、結晶中のTGTG'鎖の持つ分子鎖に平行方向の双極子モーメントの揺らぎに起因すると考えられている。（この三次元における虚部のピークの位置をfT面に投影すると分散地図となる。β過程がVTF型であるのに対し、結晶緩和はArrhenius型であることがわかる）
◇ 界面分極と電極分極
　多くの誘電体は多少の不純物イオンを含んでおり、高温になるとその拡散的運動による直流伝導が、誘電スペクトル虚部の低周波側に急激な増大をもたらす。さらにイオン伝導は内部の不均一構造や電極との界面において電荷の蓄積をもたらし、見かけ上誘電分極を形成する。実例として、分子量400のポリエチレンオキシド（PEO）に過塩素酸リチウムを0.1%混ぜ（Li-PEO）、さらに直径3 μmのチタン酸バリウム（BaTiO3）の微粒子を体積分率で40%分散させた試料の誘電率および導電率の実部の周波数スペクトル（をここでは議論する）。複素誘電率ε*と複素導電率σ*は次の式の関係
　ε* = σ*/(iω)
があるので、導電率の実部は誘電率の虚部からσ'=ωε''により計算できる。
　BaTiO3を含まないLi-PEOは1 GHz近傍にPEOのセグメント運動による誘電緩和を示す。1 MHz以下になると、導電性が支配的となり、虚部が周波数に反比例して増大する。10 kHz以下になると実部も急激に増大する。これは電極分極と呼ばれ、イオンが電極近傍に蓄積して大きな分極を形成することによる。
　BaTiO3微粒子を入れると、その大きな誘電率（ε/ε0=1700）のため高周波の誘電率は増大する。さらに10 - 100 kHzにおいて緩和強度が1000に達する大きな誘電緩和が発現する。これはイオンがBaTiO3微粒子界面に蓄積して分極を形成する（界面分極）ために生ずる誘電緩和で、Maxwell-Wagner効果と呼ばれる不均一系に特有の機構である。界面分極や電極分極を含む誘電スペクトルは、式[ε*(ω) = ε∞ + Σk{Δε/[(1+(iωτ)βk]αk}]に次の式の形のスペクトルを加えることによりよく再現できる。
　ε* = {Δσdc/(iω)}*[1-1/{1+(iωτ)γ}]
　この関数形は複素導電率がCole-Cole型の緩和を示すことを意味している。分散体界面と電極界面におけるイオンの蓄積は、分極形成による誘電率の増大をもたらすとともに、イオンのトラップとして導電率を減少させる。

◇ おわりに
　誘電スペクトロスコピーは18桁に至る超広帯性を特徴として、現在はほとんどすべての緩和モードの測定が可能となっている。最近ではこれを非線形域に拡張することも行われるようになった。誘電応答測定システムは一般的なもので、正弦電場に対する高調波電気変位応答を検出すれば、非線形複素誘電率を得ることができる。強い非線形誘電体である強誘電体においては、電場と電気変位の関係を直接測定し、いわゆるDEヒステリシス曲線やスイッチング特性を求めることができる。誘電結合効果として知られている圧電性や焦電性については、電場の代わりに力学的あるいは熱的刺激を与え、線形応答測定システムを利用して電荷応答を測定すればよい。さらに圧電率の測定は、交流電場を加えたときに起こる試料の力学的共鳴を測定する圧電共鳴法を利用することができ、この測定法は実質的に複素誘電率の周波数スペクトルの測定として行うことができる。

■ 光・電子機能測定
　試料に光照射することによって電気伝導度が増加する現象が光伝導であり、電界をかけて発光する現象が電界発光すなわちEL（electroluminescence）である。また、光と物質との相互作用において光強度、もしくは電界強度が強い場合に発現する現象が非線形光学効果である。電気伝導度や発光強度の測定、光照射や電圧印加に伴う位相差の検出などが測定の基本となる。

■ 電気伝導度
◇ 電気伝導度とは
　試料に印加した電圧V(J C-1)は、物体中の電子・ホール。イオンなどの荷電粒子に対して、エネルギーの勾配を発生させる。荷電粒子にはこの勾配に従って力が生じ、その方向へと移動する。荷電粒子の輸送量を表す物理量に電流がある。電流I（C s-1）は単位時間当たりに移動した電荷の総量である。電気伝導度（導電率）σ（Ω-1 cm-1）は、電流の流れやすさを表す物理量であり、低効率の逆数に当たる。電気伝導度は、ちょうど単位体積の試料に、単位電圧を印加したときに流れる電流量に相当する。単位体積では、電圧Vは電界Eに一致し、電流Iは電流密度jに一致するので、電気伝導度は一般に電界E（V cm-1）と電流密度j（A cm-2）を用いて次の式のように書かれる。
　j = σE
いま、試料中に存在する荷電粒子（電荷q（C））の密度をnとし、これらの粒子が電圧によって速度vで移動したとする。単位体積の試料では単位時間にqnv（C）の電荷が移動し、これが電流密度jに相当するので、次の式が成り立つ。
　j = qnv
さらに、電荷の移動速度vは、電界Eと移動度μを用いて次のように表される。
　v = μE
　移動度μ（cm2 s-1 V-1）は単位電界当たりの荷電粒子の速度に相当する。密度nは電荷の総量として電流に寄与し、移動度μは荷電粒子の速度として電流に寄与する。移動度、電荷密度がともに大きいときに、大きな電流となることがわかる。電気伝導度はこれらの式を比較することで次の式のように書ける。
　σ = qnμ
　電気伝導度には、試料中に存在するすべての荷電粒子の速度が反映される。荷電粒子には、電子、ホール、陽イオン、陰イオンが存在するため、厳密には式[σ = qnμ]は各粒子の密度と、粒子固有の移動度によって構成される。電気伝導度による評価は、これらの荷電粒子の協同現象を見ることになる。

◇ 導体・半導体・絶縁体の電気伝導度
　金属に代表される導体（102 - 106 ?-1 cm-1）では、電荷密度を構成する粒子は自由電子である。自由電子は大きな移動度μを有し、その大きさは電圧に依存しない。よって式[σ = qnμ]における電気伝導度は電圧に依存せず一定となる。
　半導体では、ホールが荷電粒子になる場合もある。半導体の電荷密度は導体に比べて数桁小さくなる。また移動度は、電界に対して一定ではなく、一般に電界に対して増加する。このため電気伝導度は電圧に依存して変化し、オームの法則は成り立たなくなる。半導体から絶縁体にかけては、試料の伝導帯が電極金属のフェルミレベルから大きく離れる傾向にある。この場合、電気伝導度は試料内部の荷電粒子の移動よりも、金属から試料への電荷の注入特性に大きく依存する。絶縁体では、電気伝導度が電極の種類に依存し、物質固有の値とは言えなくなる。この場合、電気伝導度よりも、移動度やエネルギー準位による議論が必要となる。

◇ 試料作製方法
　電気伝導度の測定には、対象とする試料によってさまざまな測定方法が存在するが、ここでは薄膜、結晶などの固体の試料に対するものを紹介する。電機伝導度の測定に適した試料は、単結晶のような形状の整ったものである。このような試料では、電極を試料の両面に接着するだけで、電気伝導度の測定ができる。電極は試料に銀ペーストを接着させることで作製する。銀ペーストに配線もあわせて結合させることで電気伝導度測定用の試料が作製できる。
　一方、高分子フィルムのように十分に厚みのある試料を得るのが難しい場合には、電極の上に直接フィルムを作製することで、一方の電極に綺麗に密着した試料を得ることができる。電極は金属板かITOガラスを使用し、その上にバーコート法、スピンコート法を用いて作製する。他方の電極は銀ペーストもしくは蒸着法によって作製する。蒸着法では試料と電極の密着性に優れるほか、面積の正確な電極を作製できるという利点がある。しかし、蒸着時に試料を高温にさらすため、試料の熱分解が生じやすいという欠点があり、熱的に不安定な試料では避けた方が良い。

◇ 電気伝導度の測定
　電気伝導測定の最も基本的な方法は、作製した2端子の試料に、電源、電流計をつないだものである。試料に電圧を印加し、電圧-電流特性を測定することで、試料の電極面積Sおよび、電極間距離dから、次の式に代入して求めることができる。
　σ = (d/S)*(I/V)
A. 4端子測定法
　電気伝導度の大きな試料では、試料と電極との接点に生じる接触抵抗が、試料そのものの抵抗に比べ無視できなくなる。この場合、電極接点において大きな電圧降下が起こり、電源が供給する電圧と、試料に実際にかかっている電圧との間に誤差が生じる。
　4端子測定法では、電圧測定用の電極も作製し、電圧と電流を個別に測定する。電圧の測定端子には電極から試料へ電流がほとんど流れないため、接触抵抗による電圧降下が無視でき、正確な電圧を測定することができる。電流端子で電流を、電圧端子で電圧を測定し電気伝導度を得る。
　電圧端子を作製する際は、電極間距離dの精度を良くする為に、電極の面積は小さく作製するほうがよい。電極端子に電圧計を接続し、電流端子には電源、電流計を接続する。任意の電圧を印加し、電圧計の電圧値、電流計の電流値を測定する。電圧端子の電極間距離dおよび、電流端子の電極面積Sを用いて式[σ = (d/S)*(I/V)]から電気伝導度を計算する。
B. 異方性試料の測定
　異方性を有した試料では、試料に印加する電圧の方向によって電気伝導度が異なってくる。異方性を伴った電気伝導度は、式[j=σE]が電界ベクトル、電流ベクトルに対するテンソル表示によって表される。伝導度の異方性は試料に印加する電圧の方向を変えることで測定でき、通常くし型電極やインピーダンスブリッジが利用される。
　試料の形状によっては、表面を通って電流が流れたときと、試料内部を電流が流れたときで伝導度が異なり、正確な値が測定できない場合がある。このような場合は電極に対して補正因子を組み込むことで、試料の伝導度を見積もることができる。

■ 光伝導
◇ 光伝導とは
　光伝導とは、試料に光照射することによって電気伝導度が増加する現象であり、コピー機などの感光ドラムや広くは太陽電池、光センサーなどに利用されている。光伝導では光照射によってホールと電子から成るキャリヤー対が生成されるため、式[σ=qnμ]における荷電粒子の密度nが増加し、電気伝導度σの増加につながる。光伝導は主に、次の三つの過程からなる。
A. 光励起
　分子が光によって励起される過程である。最低励起状態ではHOMO軌道の電子が、LUMO軌道に遷移し、いわゆる分子が励起子と結合した状態となる。励起子が一つの分子内に留まっているときは、LUMOに電子が、HOMOにホールが存在するとみなすことができる。しかし、分子程度の狭い領域に留まっている電子とホールでは独立したキャリヤーとみなすことはできない。
B. キャリヤーの生成
　分子中の電子とホールが、近傍の分子もしくは、より遠くの分子まで拡散した状態で、励起子が複数の分子上に存在する状態である。電子-ホール間の距離が大きいためクーロン引力は弱くなり、再結合して消滅する可能性が小さくなる。この状態では、電子とホールを独立したキャリヤーとみなすことができる。
C. キャリアー移動
　電子-ホール間の距離が十分に大きくなると、これらの間に働くクーロン引力に対して、外部電界から受ける力が相対的に大きくなる。この状態では、電子とホールは再結合することなく、分子間を自由に移動することができる。電子とホールとでは、外部電界から受ける力は逆向きなので、移動する向きは逆になる。

◇ 光伝導の測定
　光伝導を確認するには光を照射しない状態と、光を照射した状態とで電気伝導度を測定し、比較すればよい。試料は光が当たるように電極を作製するとよい。また、透明電極を用いると、電極の上から光を照射することもできる。光照射には、水銀灯、キセノンランプなどの白色光を用いるとよい。試料固有の吸収波長を照射したい場合は、レーザー光を用いるか、白色灯に干渉フィルターもしくはモノクロメーターを用いて必要とする単色光を取り出せばよい。
　光照射による、理論的な電気伝導度の増加は次のように計算される。
　Δσ = e*Δn*μ=e*(τηTI0/l)*{1-exp(-kl)}*μ
ここで、e（C）は電気素量、I0（photon cm-2 s-1）は照射する光強度、l（cm）は試料の厚さ、Tは試料表面での透過率、k（cm-1）は試料の吸光係数、μ（cm2 s-1 V-1）はキャリヤーの移動度、τ(s)は生成したキャリヤーの寿命、ηはキャリヤー対の生成収率（吸収した光子に対する生成したキャリヤー対の比）である。
　光子数で表される照射光強度I0（photon cm-2 s-1）は、パワーメーターを用いて測定された光強度P（W cm-2）から、次の式に従って変換される。
　I0 = {λ/(hc)}*P
ここで、λは単色光の波長、cは光速、hはプランク定数。

◇ ブロッキング電極法
　キャリヤーの寿命τや、生成収率ηを定量的に求める方法に、ブロッキング電極法がある。ブロッキング電極法で用いる素子は、試料の両面を絶縁体で挟み、電極から試料への電荷の注入を完全にブロックするように作製する。

　電荷の注入をブロックした状態で、試料に電圧を与えると、試料はコンデンサー中の誘電体の役目をする。光伝導を示す物質では、生成した光キャリヤーの移動に伴って、試料が分極し、誘電率が変化するため両端の電極に誘起する電荷量も変化する。よって、電荷が蓄えられる過程を、電流値として測定することで、試料内部の光キャリヤーの生成や移動を観測することができる。

　吸収係数kが大きく、光のほとんどが試料表面で吸収される試料の場合、光照射後の電流-時間特性は、次に示す式に従って減衰する。この減衰曲線から寿命τ、キャリヤー生成の量子収率ηを求めることができる。
　I(t) = dQ(t)/dt = {eηTI0(1-exp(-kl)μE/l}*exp(-t/τ)
ここで、e（C）は電気素量、I0（photon cm-2 s-1）は照射する光強度、l（cm）は試料の厚さ、Tは試料表面での透過率、k（cm-1）は試料の吸光係数、μ（cm2 s-1 V-1）はキャリヤーの移動度、τ(s)は生成したキャリヤーの寿命、ηはキャリヤー対の生成収率（吸収した光子に対する生成したキャリヤー対の比）である。

■ 電界発光（EL）[1]

◇ 有機電界発光素子
　ELとは電界をかけて発光するという意味で、electroluminescenceの略語である。元々は無機材料において発見された現象だが、近年の研究で、有機材料においても超薄膜化することで、高輝度の発光が得られることがわかった。有機EL素子は次世代の平面薄型ディスプレーへの応用として期待が高まっている。
　ELの発光原理は、光伝導の逆の関係にある。両電極から電子、ホールを注入し、色素近傍で再結合させることにより、励起状態の分子を生成し発光させる。効率の良い素子を作製するには主に次の3点が重要となる。
1. キャリヤー対である電子と正孔を両電極からバランスよく注入する
2. 電子とホールを効率的に輸送する
3. 電子とホールを発光色素近傍で効率よく結合させる
　これらの特性を持つ分子を分離・積層することで効率の高い素子が実現できる。

◇ EL素子の作製方法
　有機EL素子の陽極には、発光した光を透過でき、かつ、ホールを注入しやすいITO電極が用いられる。試料となる有機層はその上に積層し、陰極には電子を注入しやすいMgやAlなどの金属で挟んだサンドイッチ構造を形成する。EL素子の作製は、一般にITO電極側から行う。有機層を積層する方法には、主に真空蒸着法とスピンコート法がある。真空蒸着法は、抵抗加熱を用いて試料を蒸発させて蒸着する方法で、分子量が小さく沸点の低い物質に対して有効である。スピンコート法では材料を溶媒に溶かし、高速回転によって溶媒のみを揮発させて積層するので、分子量が大きな材料や粘性の高い高分子に適している。多層膜の素子を作製する場合、蒸着法やスピンコート法を組み合わせ積層するが、スピンコート法で、積層した下の層を溶かさないような溶媒を選ばなければならない。有機層の厚さは、電子注入層から正孔注入層までを合わせても、数十から数百nmにする。陰極電極は金属を真空蒸着法で積層する。電極の形は蒸着時に使用するマスクの形に応じて返ることができる。
　でき上がった素子を電源に簡単に接続できるよう、電極の構造を文献[1]の図のようにするとよい。ITOはあらかじめ基盤の両端を塩酸でエッチングし、電極をはがしておく。有機層上部の金属電極はITOに対して垂直になるようにマスク作製し蒸着する。

◇ 輝度の測定
　上記のようにして作製したEL素子では、ITOに陽極を、金属電極に陰極を接続し、電圧をかけることで、ITOと金属電極が交差した部分が発光する。発光面に輝度計の焦点を合わせ、緯度を測定する。輝度とともに、電圧電源における電圧値、電流計による電流値を合わせて測定する。

◇ 発光効率
　一般に素子の特性を評価する発光効率η1(lm W-1)は、供給した電気エネルギー（W）に対する視感度エネルギー（lm）の比で表される。一方、輝度計で測定される輝度の単位は、単位面積当たりの光度（cd m-2）となっている。これはELの発光が、点光源が集まってできる面からの拡散発光とみなされるからである。面発光による測定輝度L（cd m-2）を、点光源である光度（cd）に変換するには、発光面積S（m2）をかければよい。さらに、光度（cd = lm/sr）は、点光源が放出する単位立体角1（sr）当たりのエネルギー（lm）であるため、放出された光に立体角（sr）をかけることで、エネルギー（lm）に変換できる。拡散発光するELの場合、発光強度が方向に依存して変化するため、強度を立体角に対して積分する必要がある。面からの発光が完全拡散である場合、光強度は次の式に従い、正面I0で最も強く、真横で0になる。
　I(θ) = I0*cos(θ)
　この場合、ELから放出される全エネルギーは次の式となり、
　Itotal = ∫2π0∫π/20 I0*cos(θ)*sin(θ) dθdφ = 2πI0∫π/20 cos(θ)sin(θ) dθ = πI0 = πLS
阻止から放出される全光強度は、正面で測定される光強度のπ倍になることがわかる。
　ただし、後面への発光は金属電極で反射するものとした。この光エネルギーを、素子に供給した電気エネルギー（電流I（A）、電位差V（V）の積）（W）で割ることで発光効率η1（lm W-1）が得られる。
　η1 = πLS/(IV)
　また、輝度L（cd m-2）を電流密度i（A m-2）で割った、発光効率η2（cd A-1）も用いられる。
　η2 = L/i
　これは、単に光エネルギーを電流で割った値となるので、注入したキャリヤー対に対する発生した光子の割合、すなわち内部量子効率に比例した値になる。

■ 非線形光学効果 [1]

◇ 非線形光学現象
　非線形光学効果とは、光と物質との相互作用のうち、光強度、もしくは電界強度が強い場合に発現する現象である。非線形現象は、元来、光と物質の相互作用として、理論による予測にのみ留まっていた。それが、近年のレーザーの発達により、強力な光を発生させることが可能になったため、実験レベルで観測され、数多くの研究を生み出すこととなった。
　物質に一定の電界をかけたときに生じる分極Pは、電界Eに依存して変化する。われわれが通常経験する現象は、分極を電界について展開したときの一次の応答のみを考えれば十分説明ができる。
　P(E) ≒ Χ*E
　しかし、厳密には分極は電界に非線形な応答を示し、電界の多項式によって記述される。
　PI = ΣJ ΧIJ(1)EJ + Σ<J,K> ΧIJK(2)EJEK + Σ<J,K,L> ΧIJKL(3)EJEKEL + ...
（I, J, K, ... = X, Y, Z, ...）（通常、X, Y, Z軸は誘電主軸に取る）
（Eが大きくなると非線形分極が大きくなり無視できなくなってくる。線形分極+非線形分極となる）
　この意味では、線形な領域のみを考慮することは現象を近似的に取り扱ったにすぎず、非線形の領域を考慮してはじめて、物質の真の姿を記述することができる。そして、この非線形な応答が、複雑ではあるが数々の興味深い現象をもたらしてくれる。その例として、第2高調波発生（SHG）および第3高調波発生（THG）があげられる。

◇ 第2高調波発生（SHG）と測定
　第2高調波発生（Second Harmonic Generation: SHG）とは、物質に強い光（角振動数: ω）を照射したときに、物質から角周波数2ωの光が発生する現象である。光である電磁場は電界の振動であるため、上で示した式PIにおける電界を交流電界に拡張することで、SHGを説明することができる。今、簡単のため1次元のみを考慮する。照射光（角振動数: ω）の電場の振動を次の式として、
　E = E0*cos(ωt)
式PIにおける分極の2次までに代入する。途中、2倍角の公式を用いると次の式となり、
　P ≒ Χ1*E + Χ2*E2 = Χ1E0*cos(ωt) + (Χ2*E02/2)*cos(2ωt) + Χ2*E02/2
cos(2ωt)で振動する分極成分が現われることになる。
　分子の分極振動に従い電磁波が放出されるので、放出される電磁場には2ωの振動成分の光、すなわちSHGが含まれていることがわかる。cos(ωt)で振動する分極から発生する電磁波は、物質中を透過する光である。
　大きなSHGを示す物質の条件は、次のように考えることができる。物質に引き起こされる分極は、ωと2ωの分極成分の和であるが、相対的に2ωの分極成分が大きくなると、全体の分極は非対称になる。分極の非対称は、分子内の電子振動の非対称性に由来するものであり、それは分子構造の非対称性から現われると推測できる。このため、ドナー基（D）とアクセプター基（A）をπ電子共役系で結んだ、非対称構造かつ分子内で電荷移動が起こりやすい分子が大きなSHGを示す。
　実際にSHGを観測するには、分子レベルだけではなく、試料状態におけるバルクレベルでも異方性が必要とされる。単結晶の試料では分子が配向しているため、異方性を有しているが、アモルファス状態の試料では何らかの方法で配向させる必要がある。幸い、SHGを示す分子は、分子レベルの非対称性から大きな双極子モーメントを有しているので、高電界によるポーリング処理により分子を配向させることができる。
　試料から発生したSH光は分光器を用いて識別し、その強度をフォトマル（光電子増倍管）、もしくはフォトダイオードを用いて測定する。（参考文献[1]に図が示されている）1/2波長板はレーザーからの偏光を変えるために、回転ステージは光入射角に対するSHGの強度変化を見るため、また、偏光板は発生したSHGの偏光性を観察するために用いる。
　分子をポーリングにより配向させたポールドポリマーでは、分子が1軸配向し（点群C∞v）、xy面内は等方的になるので、式PIはd定数を用いたテンソル表示で、次の式のように表せる。
　[p2ωx P2ωy p2ωz]t = [A] [Eω2x Eω2y Eω2z 2EωyEωz 2EωzEωx 2EωxEωy]t 
         |0   0    0   0   d15 0|
[A] = |0   0    0   d24 0   0|
         |d31 d32 d33 0   0   0|
　ただし、d32 = d31, d24=d15
このテンソル成分のうちd33は、分子の配向方向（z軸）に振動する電場Ezを照射したときの、z軸方向に振動するSH強度に比例し、分子の配向に伴って大きくなる。xy面内における対称性からd31, d32は同じ値を示し、その大きさはd33に対して相対的に小さくなる。ポーリングがうまくなされたポールドポリマーではd33とd31, d32の相対値により分子の配向の度合いを調べることができる。
　d33およびd31, d32のテンソル成分は次の方法で測定できる。まず、照射する基本波s, p偏光に対して、発生するSH光（角周波数2ω）のp偏光成分Pps, Pppをそれぞれ入射角θに対して測定する。光強度が電界強度の2乗に比例するので、SH強度は入射角θに対して次の式に従う。
　Pps(θ) ∝ αps(θ)*{sin(θ(s)2ω)*d31}2
　Ppp(θ) ∝ αpp(θ)*[{sin(θ(p)ω)*sin(θ(p)2ω)}*d33+{2cos(θ(p)ω)*sin(θ(p)2ω) + cos2(θ(p)ω)*sin(θ(p)2ω}*d31]2
　θω, θ2ωは基本波および発生したSH光が試料中を透過するときの屈折角である。これらは振動数ω、および2ωの各偏光に対する屈折率n(s,p)ω, n(s,p)2ωから、スネルの法則を用いて求められる。
　n(s,p)ω(2ω) = sin(θ(s,p))/sin(θ(s,p)ω(2ω))
　αps, αppは、位相不整合によって生じるSH強度の振幅を表す項である。屈折率n(s,p)ω、n(s,p)2ωが異なると、基本波とSH光との間に位相の不一致が生じSH強度が入射角に対して振動する。αps, αppは入射角θに対して次の式のようになる。
　αps(θ) = (512π/S)*t(s)g4t(s)ω4t(p)2ω4*{sin2Ψsp(θ)/(n(s)ω2-n(p)2ω2)}*Pω2
　Ψsp(θ) = (2πl/λ)*{n(s)ω*cos(θ(s)ω)-n(p)2ω*cos(θ(p)2ω)}
　αpp(θ) = (512π/S)*t(p)g4t(p)ω4t(p)2ω4*{sin2Ψpp(θ)/(n(p)ω2-n(p)2ω2)}*Pω2
　Ψpp(θ) = (2πl/λ)*{n(p)ω*cos(θ(p)ω)-n(p)2ω*cos(θ(p)2ω)}
　Sはレーザーの照射面積、tωは試料へのレーザー光の透過率、t2ωは試料からの空気中へのSH光の透過率、tgはレーザー光の基板への透過率、Pωは基本波の強度である。測定したSH強度の角度依存性とPps(θ)とPpp(θ)の式の比例係数を求めればよい。

◇ 第3高調波発生（THG）
　第3高調波発生（Third Harmonic Generation: THG）はより高次の非線形現象である。したがってSHGのときよりもさらに高強度な光を必要とする。電界の振動（式[E = E0*cos(ωt)]）を、分極の3次（PIについての式）に代入すれば、
　P(3) = Χ3*E3 = (Χ3*E03/4)*cos(ωt) - (3*Χ3*E03/4)*cos(3ωt)
　3ωで振動する分極成分が現われることがわかる。この分極振動に伴い発生する光がTH光である。SHGのときとは異なり、THGは対称な分極振動から発生する。THGのテンソル表記では、PIについての式において、計30個のテンソル成分を必要とする。通常、TH測定ではこれらの成分一つ一つを求めるのではなく、SHGの場合と同じようにメーカーフリンジ法によってTH強度を測定する。

◇ その他の非線形光学効果
　そのほかの非線形光学現象で代表的なものに、ポッケルス効果、カー効果がある。これらは試料に印加した静電界に依存して屈折率が変化する現象である。屈折率が電界に比例して変化する場合をポッケルス効果、電界の2乗に比例して変化する場合をカー効果と呼ぶ。静電場は、波長が無限大の光子に相当し、屈折率を感知する光子とともに物質に作用する。この意味で複数の光子が物質に関与する過程となり、非線形現象の一つに分類される。物質に照射する光強度が強い場合、静電界をかけなくても、光強度（電界の2乗）に比例する非線形効果として、屈折率を変化させる場合がある。この現象を特に光カー効果と呼んでいる。
　ポッケルス効果、カー効果の測定の基本的な測定法は、屈折率変化に伴って生じた位相差を検出するような、（参考文献[1]に示されている図の）測定系で測定することができる。印加する電解に対して、偏光子を45°の方向で挿入することで、屈折率の異方性を高感度で測定することができる。結晶状の試料に対してさまざまな方向から電圧を印加することで、屈折率変化の異方性を測定できる。
　小さな屈折率差を精度よく測定するには光の干渉効果を利用すればよい。光路を試料を通る経路と試料を通らない経路に分け、互いに干渉させることにより、屈折率変化に伴って生じる波長程度の光路差を感知することができる。また、屈折率を試料の吸収も考慮した複素屈折率の形で求めるには、パルス電圧を印加したときの光吸収の応答を調べることで測定することができる。
