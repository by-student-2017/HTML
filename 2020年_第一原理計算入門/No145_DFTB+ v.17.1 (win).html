■ DFTB+ (version 17.1) (Edit: 23/April/2020) (windows10 (64 bit))

□ DFTB+ 17.1 (Windows Binary)
(https://sites.google.com/view/djmolplatform/get-dftb-17-1-windows)
1.Get DFTB+17.1WindowsBinaries > ダウンロード > ダウンロード
2.DFTB17.1Windows.zip を解凍
3.DFTB17.1Windows-CygWin を Windows(C:)にドラッグ&ドロップ

□ Test
1.C:\DFTB17.1Windows\DFTB17.1Windows-CygWin
2.dftb_in.hsd をメモ帳などで開く
---------(before)
Prefix = "C:\Users\staff\Downloads\mio-1-1\mio-1-1\
---------
---------(after)
#Prefix = "C:\Users\staff\Downloads\mio-1-1\mio-1-1\
---------
3.dftb+.exeをクリック
"***  Geometry step:"の右に記載されている数が増加していけばテストは成功

□ Example (H2O)
1.C:\DFTB17.1Windows\DFTB17.1Windows-CygWin\H2O のファイルを開く
2.dftb_in.hsd をメモ帳などで開いて下記のように書き換える
---------
#Driver = Socket {
#    File = "dftb"
#    Verbosity = 0 # quiet
#    Protocol = i-PI {} # i-PI interface
#    MaxSteps = 1 # Terminate this instance fairly quickly
#}
Driver = ConjugateGradient {
  MovedAtoms = 1:-1
  MaxForceComponent = 1E-14
  MaxSteps = 1000
  OutputPrefix = "geom.out"
}
---------
3.[Shift] + 右クリック > PowerShellウィンドウをここで開く
4.C:\DFTB17.1Windows\DFTB17.1Windows-CygWin\dftb+.exe

□ MoCalc2012 (Version 4.2.0.3) (Windows10)
(https://sourceforge.net/projects/mocalc2012/) 
(MoCalc2012-setup4203-win64.exe)
1.Options > Settings...
2.DFTB+:[C:\DFTB17.1Windows\DFTB17.1Windows-CygWin] > Save

□ Example (H2O on MoCalc2012)
1. MoCalc2012 > Start > Open Structure File...
  [All Files (*.*)] > ice.gen > [開く]
  (C:\DFTB17.1Windows\DFTB17.1Windows-CygWin\H2O\ice.gen)
2. Job Type: [Optimize + Vibrational Frequency]
  Driver: [LBFGS] または [Conjugate Gradient] 
  Dispersion: [Slater Kirkwood] 
3. Start > Create Input File
4. [Run DFTB+]
5.[Structure]
6.[Energy level] > [Densit of]
※ *.skfはDFTB+:[C:\DFTB17.1Windows\DFTB17.1Windows-CygWin]で指定してたファイルが用いられる。他の*.skfを用いたい場合はファイルを入れ替える。

(https://sourceforge.net/projects/jmol/)
