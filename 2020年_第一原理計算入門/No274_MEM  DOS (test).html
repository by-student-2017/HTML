■ 電子エネルギー密度 （Electronic energy density *.ted）： he(r) = g(r) + v(r)
※ ( 2* g(r) + v(r) = (?2 / 4m) * ▽2ρ(r) )
*.grd: density (e/Ang^3) = ρ(r)
*e.grd: energy (E/Ang^3) = he(r) (*.ted > File > Export Data *.grd > mv *e.grd)
