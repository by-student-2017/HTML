■ X線イメージング
（Edit: Jun/3/2021）
（聞き間違いがありましたらお許しください）
 
◇ X線プローブの特徴
・波長が短い -> 顕微鏡において高い空間分解能の可能性
・物質との多様な相互作用 -> 多様な情報が取得可能
・高い透過能 -> 大気中での非破壊観察が可能
 
◇ X線の複素屈折率
・X線の複素屈折率 n = 1 - δ + iβ
・β << 1 (10-6から10-8)
・n?1は、レンズでは屈折しない
・R=(δ2+β2)/4?0は、直入射では反射しない
 
◇ X線レンズ（光学素子）
  屈折率 n?1であるが、様々な光学素子が開発されている
・フレネルゾーンプレート（FZP）：回折・干渉現象でX線を曲げる
  最も利用され高い性能を有している
・ブラッグ・フレネルレンズ：ブラッグ回折でX線が反射することを利用
・全反射鏡：斜入射にすれば全反射となることを利用
・その他
  キャピラリー：初めに実現した
  ウォルターミラー：天体で用いられる
  K-Bミラー：点を作るのに使う
 
◇ フレネルゾーンプレート（FZP）
・円形の透過型回折格子
・レンズの結像公式（1/z1 + 1/z2 = 1/f）を満たす
  z1 = 試料（Object）とFZPとの距離
  z2 = FZPと像(Image)との距離
・バイナリーゾーンプレート： 最大効率 10%
  I = 1/π2 ? 10.1%
・位相ゾーンプレート： 最大効率 41%
  I = (C/π)2 * (1 + exp(-2ηΦ) - 2*exp(-ηΦ)*cosΦ)
  n = 1 - δ - iβ
  η = β/δ
  Φ = 2πtδ/λ
 （41%は理想値、通常は20%程度のものを使うことになる）
・分解能δ = 1.22*ΔrN
  最外周最小線幅ΔrN
 
◇ NTT-ATN社製タンタルゾーンプレート
・X線に対して透明・不透明の輪帯を交互に繰り返した円形の透過回折格子
・電子ビームリソグラフィー法で作製される（支持膜は窒化シリコン）
・軟X線用： ニッケルやゲルマニウムなどの軽い金属
・硬X線用： タンタルや金などの重金属
・空間分解能は最外周最小線幅と同程度
・軟X線用： 15 nm
・硬X専用： 35 nm
 
□ X線顕微鏡の種類
----------
◇ 走査型X線顕微鏡
・取得情報がデジタル（定量分析に良い）
・試料からの様々な情報が一度に得られる（多様なスペクトロスコピー）
・試料を動かす
・試料の被ばく線量を最小限に抑えられる
・露光時間は長い
 
◇ 結像型X線顕微鏡
・直接の拡大像が得られる
・実時間観察（動的観察）が可能
・ゼルニケの位相差顕微鏡が可能
・露光時間は短い
----------
 
◇ 蛍光X線による元素分析
・元素固有のエネルギーの蛍光X線を発生
・生体内微量元素の空間分布測定、定量分析
  これにより生きたままの分析や生命現象と対応した微量元素の挙動
  （試料の前処理が簡便、非破壊的測定、他元素同時分析）
◇ 付録： 生体に含まれる元素（組織あたりの重量比（g/g））
・多量元素： O, C, H, N, Ca, P, K, Cl, S, Na, Mg
・必須性が認められる微量元素： Fe, F, Zn, Cu, V, Se, Mn, I, Ni, Mo, Cr, Co
・汚染元素： Sr, Pb, Cd, As, Hg, Sn

◇ CT（コンピュータートモグラフィー）
・断層映像法または断層撮影法と呼ばれる
・病院の検査、工業製品の非破壊検査などに使われている
・結像型X線顕微鏡での空間分解能はX線レンズで決まる（現状、約60 nm）
・試料を回転させながら顕微鏡像を取得
・FBP（Filtered Back-Projection）法
  気になる方は[橋本雄幸ら、「C言語による画像再構成の基礎」、医療科学社]を参照ください
 
□ 吸収コントラストと屈折コントラスト
----------
◇ 吸収コントラスト
・従来のレントゲン写真
・X線の吸収量（波の強さの減少）で構造を可視化
・観察する距離が近い
 
◇ 屈折コントラスト
・放射光による新しい方法（入射光の平行性が必須）
・X線の位相変化量（波の形の変化）で構造を可視化
・境界が強調される（鮮明な像が得られる）
・観察する距離が遠い（数秒程度（Δα）の広がりしかないため距離をとる）
----------
 
◇ 磁気円二色性（Magnetic Circular Dichroism: MCD）顕微鏡
・磁気円二色性：右回りと左回り円偏光との吸収の差
  J. Stohr et al., Science 259 (1993) 658.
・全電子収量法や吸収を画像化
 （PEEM、ZPマイクロビーム、結像型顕微鏡）
   世界初： ZPマイクロビーム (Y. Kagoshima et al., J. Appl. Phys. 80 (1996) 3124.)
 
◇ X線磁気トモグラフィー
・多数の角度でXMCDでの磁気イメージを測定
・再構成アルゴリズムを用いて、3次元の磁区構造得る
・空間分解能(Wx = 310 nm, Wz = 360 nm) (CdFeCo磁気ドット（直径6μm）[4])
 
◇ CXDI（コヒーレントX線回折イメージング）法
・マイクロからサブマイクロメートルサイズの非結晶試料の内部構造を高空間分解能・高コントラストで非浸襲観察
・位相回復アルゴリズムを用いて回折パターンから投影電子密度像を得る
※ 真空やHeで散乱を抑える
 
◇ 位相回復アルゴリズム
・孤立粒子で輪郭のある場合に位相回復できる
・結晶性があるとその輝度に逃げて上手くいかない
・非結晶性試料が良い
・奥行きは和算
