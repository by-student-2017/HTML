◇ Fitting low loss spectra EELSMODEL
1. File > Open DM3
  Examples\low_loss_diamond\diamond_no_bkg_deconv.dm3
2. (click) spectra window
3. Model > New model
4. Model > Component
  Dielectric Model Li... > Add
  > enter number of points [10] > OK
  > 5.8081 [Parameter value] (for Eps0) > OK
  > 0.1 [Parameter value] (for srength) > OK
  > 3E5 [Parameter value] (for E0) > OK
  > 5e-8 [Parameter value] (for absolute thickness [m]) > OK
  > 20e-3 [Parameter value] (for thetamax [rad]) > OK
  > OK (for Estop [eV])
5. Model > Fit > GSL: Least Squares > OK
6. Do Iterations (on Tolerance reached)
7. (right-click) Dielectric M... (on Model component window)
  [chack] Apply relativistic correction
  nr. of points [10] -> [40]
8. Do Iterations (on Tolerance reached)

※ Eps0: dielectric constant at zero energy loss
  Eps0 = n^2
  n: refractive index
  diamond, n = 2.41 -> 2.41^2 = 5.8081

※ In my case, "[unchack] Apply relativistic correction" is good fit.
