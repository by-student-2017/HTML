◇ Result (picene band strcuture) (XC potential: PBE)

Please, compare this result with Phys. Rev. B 83 (2011) 134508 (https://journals.aps.org/prb/pdf/10.1103/PhysRevB.83.134508)
　有機物半導体を研究されている方は化学の方も多くおられると思います。PRB（Phys. Rev. B）といえば物理系での高級誌です。この結果と同じ分散が[Γ-X]の部分で約0.25時間（実際は午前中の時間全てを使ったという感じ）で得られるというのは嬉しいですね。論文では大学で40万円、企業で700-800万円するVASPを用いています。一方、私の結果はPC以外は全て無償です（普通のLANケーブルとOSインストールのためのDVD-Rが1枚必要ですが数百円のもので大丈夫。PCも上記のスペックであれば10万円以下で購入できる（事務用のPCてテストしてみたので。しかも仮想マシン上で動かしているという計算の環境としては不利な状況））。
　論文と同じようなバンド図を描きたい場合は、picene.bands.in にて下記のように入力すればよいです。ただし、メモリは2GBでは大幅に足りない。メモリが多くある場合は下の"7"の部分の値をより増やしても良いです。CPUが多ければPBEだけでなくB3LYPやHSEを試してみるのもいいですね。

　論文と同じようにa, b, c軸の方向と軸の長さを揃えると、Γ = (0, 0, 0), X = (1/2, 0, 0), M = (1/2, 1/2, 0), L = (1/2, 0, 1/2) となる。このPRBの論文は追試し易くて本当にありがたかった。他の化学系の論文は同じ結果を得られるようにできず困り果てていた。偶然にも、google scholarでVASPを使って計算条件が丁寧に書かれているPRBの論文を見つけられたのは幸運だった。この高級誌の論文と同じ結果が出た時には心の中で飛び上がって喜んだ（2 GBの都合上、k点を減らしたので計算精度が心配でしたが、半導体であったということもあって、Γ点ではフェルミ準位(0 eV)よりも少し下にバンドがあることを再現できているのに私自身も驚いた。全体的にバンド構造が似ていればいいやと思っていた）。piceneより大きな分子では私のPCでは計算できないので本当に幸運だった。お金の無い方やADFで上手く計算できていない方は是非とも試して頂きたい。この結果を新規のPCやスパコンに取り組めるように教官や上司に相談するきっかけにして頂ければ幸いです。ADFの方が計算速度が速いという話を聞いた記憶がおぼろげながらあるので（数年も前のことなので間違っていたらごめんなさい）、ADFが使えるならそちらの方がいいかもしれません。化学の場合は伝統を重んじるので、論文がアクセプトし易いコードを使うのがいいでしょう。私は貧乏なのでPWscfで調べていくことになりそうです。
　bands.plotも書き換えが必要。gnuplotでのコマンドが書かれているだけなので、検索で調べれば書き換えるのはそれほど難しくない。
  bands.plotは(http://www.cmpt.phys.tohoku.ac.jp/~koretsune/SATL_qe_tutorial/)の中身を書き換えている。2019年4月から埼玉の和光市近辺で暮らす予定なので、何時の日か東工大の講習会に呼んで頂けたら嬉しい。

■ calculation command
1. get cif file from COD (Crystallography Open Database)
2. cif2cell -p pwscf  --pwscf-pseudo-PSLibrary-libdr='/home/student/psl/' --setup-all --no-reduce -f *.cif
3. cif2cell -p pwscf  --pwscf-pseudo-PSLibrary-libdr='/home/student/psl/' --setup-all --no-reduce -f *.cif --pwscf-run-type=bands
4. mpirun -np 2 $HOME/q-e-qe-6.3/bin/pw.x < picene.scf.in | tee picene.scf.out
  ----------
  4.5 gedit picene.scf.out
     nspin = 1 ,
     remove initial spin settings
     K_POINTS automatic
       2 3 1  1 1 1
  ----------
  ※ For semiconductor and insulator, low k point is OK (e.g. default k point/ 2). However, DOS calculation need default k point or more
5. mpirun -np 2 $HOME/q-e-qe-6.3/bin/pw.x < picene.bands.in | tee picene.bands.out
  ----------
  5.5 gedit picene.scf.out
     nspin = 1 ,
     remove initial spin settings
     K_POINTS crystal_b
       2
       0.5 0.0 0.0 5
       0.0 0.0 0.0 1
  ----------
  ※ My PC is 2 GB RAM memory. 2 GB is almost limit for this number of k point..
6. mpirun -np 2 $HOME/q-e-qe-6.3/bin/band.x < bands.in
7. gnuplot bands.plot
