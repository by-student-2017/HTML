◆ AESの基本的な情報
・原理的に水素（H）は観測できない
・表面から浅いところだけ見える（そのため下記のことが言える）
　EDSやWDSと同じように見えるとは限らない
　表面が汚れていると見たいものが見えなくなる
　EDSでは層での中間層や境界がぼやけていても、AESでははっきりコントラストが見えることがある
・試料の仕上げ面（プロの腕前でも下記程度になる）
　ダイヤモンド：表面から200 nmは歪み層が形成
　コロイダルシリカ：表面から30 nmは歪み層が形成（水を使用のため残渣も心配）
　クロスセクションポリッシャ：歪み層は確認されない
・AESはEPMAより真空度が高いため、電子ビームによる炭素の体積が少ない
・未知試料の分析：感度を高くして測定 > 元素を絞る > 分解能を高くして測定
・マッピング: 1万倍以上が良い。そうすると大体「7分/枚/元素」となる
・状態分析: 測定時間は約10分程度。高分解能ではその4倍の40分程度
・絶縁物は帯電に注意（負に帯電して電子反発で電子が加速されてしまう。酸化物は還元）
　帯電した場合はサブピークも同じ分だけシフトする
　導体や半導体では帯電は大丈夫

◆ AES測定の情報
・本格的なマッピングは一昼夜かかる
・低倍率(x100倍)程度であればチャージアップを緩和できる。
・測定対象：金属表面、薄いコーティングの膜厚、筒状の摺動面など
　半導体や触媒、薄膜試料、微小欠陥分析、薄膜構造、化学状態分析などにも用いられる
・微分したスペクトルの縦軸はdN(E)/dE
・原子数濃度は±2.5%程度（原子数濃度を算出するには、Intensity と RSF を用いる）
・10 keV, 1x10-8 A
・検出深さ：0から約6 mm
・検出範囲：Φ10-20 mm
・5-6回積算したスペクトルを用いることが多い。積算中のスペクトルの変化は通常ほとんどない（同じスペクトルが重なっていく感じになる）。
・積算中にピーク位置がずれていく場合はチャージアップしている
・ArイオンスパッタではSiO2基準とすればよい（それ以外の論文はほとんど見たことが無いのでこれで良い）。
・http://www.jisc.go.jp/ のK0154 「表面化学分析-分析試料の準備及び取付けに関する指針」を読んでおくとよい

・オージェスペクトルでは、7から9点の測定定数で微分する。ディフォルトは7点で計算。微分の計算方法はSavitzky-Golay法の2次多項式適合による平滑化微分の重み係数を用いている
・チャンネルトロンは電子の数え落としに個体差があるものの、200万cpsあたりを目安に使うと、S/N比が良く電子の数え落としが起こらない条件で測定を行なうことができる（※ ただし、文献によっては25万cps程度まで数え落としが起こらないとも記載されており詳細は分からない）

※ S/N ∝ ピーク強度*（ピーク強度/バックグラウンド）となる理由
　S ∝ (P - B)/B
　N ∝ 1/√(B)
　S/N ∝ (P - B)/(B*N) = (P - B)/√(B) = √{(P - B)*(P - B)/B} ∝ (P - B)*(P - B)/B
　　= ピーク強度*（ピーク強度/バックグラウンド）
ここで、Pはオージェピークのピーク、Bはバックグラウンド、ピーク強度は(P - B)である。
※ (P - B)は、加速電圧が束縛エネルギーの5倍まで急速に増加し、5倍以上でなだらかに減少する。
※ (P - B)*(P - B)/Bは、加速電圧が束縛エネルギーの10倍まで(P - B)の時よりも急速に増加し、10倍以上で(P - B)の時よりもなだらかに減少する。

◆ 定量分析
・AESにおける定量分析は、オージェ電子のピーク強度が試料中に含まれる元素の原子濃度に比例することを基本としている。定量計算には相対感度因子（RSF: Relative Sensitivity Factor）が広く用いられている
・定量計算では、微分形でなくいわゆるN(E)スペクトルを用いることもできるが、バックグラウンドが簡単に除去できて、強度が定義しやすいN(E)スペクトルを一次微分したスペクトルが主に使われている
・一次微分されたピークの最大値と最小値の差がピーク強度と定義され、この大きさはN(E)スペクトルの強度に比例する

◇ 相対感度因子（RSF）の計算方法（例としてSiO2）
   オージェスペクトルからピーク強度を得る。例えば、SiO2でのSiとOに対して下記が得られたとする）
　　Siのピーク強度：249.0
　　Oのピーク強度：431.5
　Agのピーク強度（1771）を基準とするとSiとOのRSFは下記のようになる。
　　RSF(Si) = (Siのピーク強度 / Siの原子濃度) / Agの強度 = (249.0 / (1/3) ) / 1771 = 0.422
　　RSF(O) = (Oのピーク強度 / Oの原子濃度) / Agの強度 = (431.5 / (2/3) ) / 1771 = 0.365
　ここで、SiO2でのSiの原子濃度は33.3%(=1/3)、Oの原子濃度は66.7%(=2/3)である。
※ 各元素の相対感度因子は、標準物質としてAgまたはCuを用いることが一般的

◇ 原子濃度
　元素iの濃度 Ci = [Ii/RSF(i)]/Σnk=1[Ik/RSF(k)]
　ここでnは測定対象の元素の数、IiとIkはそれぞれ元素iとkのピーク強度。

◇ 相対感度因子（RSF）の特徴
・100%で規格化されるため誤差が分からない
・RSFの測定には一般的に純元素が用いられているので、酸化物などの化合物の定量を行なう場合は類似した化合物を用いて感度因子を作り直すことで定量精度を向上させる
・RSF法ではオージェ電子のピーク強度が原子濃度に比例することが前提となっているが、実際には元素の組成が変わると試料の密度、電子の平均自由行程、反射電子強度が変化し、オージェ電子のピーク強度が濃度に比例する値からずれが生じる。このような場合、マトリックス補正を行なうことで定量精度を改善することができる
　文献：志水隆一ら『実用オージェ電子分光方』共立出版 (1989) p.122.
・RSFの値は一次電子線の加速電圧に対する依存性があるため、試料を測定したときの加速電圧のRSFを用いる必要がある。なお、各加速電圧でのRSFは実測したRSF値にイオン化断面積と背面散乱電子補正係数を使って計算することができる
　文献：T. Sato et al., Surf. Interf. Anal., 14 (1989) 787.

◆ 深さ方向分析
・イオンスパッタリング法は、希ガスイオン（主にAr+）を試料に照射して試料表面をエッチングしながらスペクトルを測定する方法であり、深さ方向分析で最も一般的な方法である
・表面から深さ方向の元素の強度分布をデプスプロファイルと呼び、縦軸はオージェスペクトルの微分ピーク強度、横軸はスパッタリング時間で表示することが多い
・スパッタリング速度の校正には、試料の入手のし易さと平坦性からシリコン熱酸膜がよく用いられる
・縦軸を深さで評価する場合、実試料のスパッタリング速度がわからないことが多いので、SiO2のスパッタリング速度での換算値で表示するのが一般的である
・深さ分解能を決める要因としては、オージェ電子の脱出深さとイオンスパッタリングによる表面荒れやアウトミックミキシング等が挙げられる（例えば文献：日本表面科学会編『オージェ電子分光法』丸善(2001) p.87）。
・深さ分解能を向上させるには一般に以下の条件で測定すると良い
a) 脱出深さの小さいオージェピークを用いる
b) イオンのエネルギーを低くする（1 keV以下）
c) イオンの入射角度を試料法線から約60° - 70°にする
d) 表面荒れが起き易い金属試料等では、試料を回転しながらスパッタリングを行なう

◆ 状態分析
・オージェスペクトルは、化学結合状態が変化すると、ピーク位置のシフト（化学シフト）やスペクトル形状の変化が生じる
・AlやSiのような軽元素の場合には比較的大きな化学シフトが起きるので、ピーク位置から金属か酸化物かを区別することができる
・TiやFeなどの遷移金属では化学シフトはほとんど起きないが、スペクトル形状が異なることが知られている。スペクトル形状を比較する場合、高エネルギー分解能のスペクトルを用いた方がその差異は明確になる
・高エネルギー分解能（0.1%）は、通常のエネルギー分解能（0.5%）で測定したときに比べて強度が約1/7倍に低くなる

◆ 金属学へのオージェ電子分光法の応用
　最近の進歩のモチーブフォースは高いエネルギーコストに由来している。例えば、タービン翼用材料の開発ができればもっと効率の良い航空機エンジンの開発につながり、これがさらにエンジンメーカーや、このエンジンを採用する航空機メーカーや、これらの航空機を運用する航空会社に相当な利益をもたらすであろう。第二の推進力は、材料資源の供給についての配慮から生じている。資源が限られているか、あるいは供給が不確定な元素の代替が可能かという問題や、材料をどの程度再利用や再生ができるかについての疑問がますます強まっている。
　金属学の最先端においてすべて、表面分析、特にAESによる表面分析が、この進歩に対して注目に価する貢献をしてきた。これらの領域には、腐食、酸化、浸炭と窒化、ロウ付け、焼結と粉末冶金、切削、摩耗と潤滑、高硬度金属と重金属の開発、そして、一般の界面破壊に関する広範な問題などを含んでいる。
　多くのものは、スパッタ―による深さ方向分析の使用を含んでいる。深さ方向分析を含むこれらの問題の解釈は、ある程度、電子マイクロプローブX線分析（EPMA）の延長で深さ分解能の優れたものと見られるかもしれない。しかし、界面破壊問題の解釈はAESの発展によって初めて可能となった個別の学問分野である。
　単原子層以下の元素が-古典吸着理論によれば-結晶粒界や相境界のような内部界面に吸着し、工業的材料の大惨事に至る破壊をどのようにして引き起こすかのモデル例として、界面の問題、特に破壊の問題に集中することとする。全体像を与えるには、ヒンクレーポイント（Hinkley Point）火力発電所のタービン発電機のローターの悲劇的粒界破壊という、よく知られている例を見てみるのは有益であろう。1969年、通例の過速度試験中にこのローターはこわれ、それが取り付けられていたタービンの大部分を破壊した。この破壊を引き起こしたローターは、原因を探るために割れた部分を残しながらも再度み立てられた。もとは外縁の周りにタービン翼があり、中心を通る軸に焼ばめされていた。破断表面の走査電子顕微鏡像を見ると、これは、破断が変形やエネルギーを消費する過程を伴わずに、もとのオーステナイト粒界に非常に近いところを通って生じたことを示している。このような破壊は脆性的といわれ、技術者にとっては鋼製のローターをガラスで置き換えたのとほとんど同じである。この挙動は、他のローターのいくつかが全く安全であったので、この特定の合金鋼やその熱処理に特有なものではない。この挙動は、鋼中の不純物が粒界サイトに偏析したことに起因している。これらの不純物のレベルは鋳造ロットごとに異なり、特別な注意を払わないと、大惨事が避けられない限界値を超えることになる。
　この問題をもっと詳細に理解するには、結晶粒界を原子的に見ることが必要である。Bragg-Nyeの泡モデルは、結晶を代表する二つの規則的配列の境界にある不整合の度合いを明瞭に図示している。この図から、マトリックス原子よりもわずかに大きい原子がマトリックス位置に（すなわち固溶して）存在するときはひずみエネルギーを持っているが、粒界にあるマトリックス原子と入れ替わり、同時にそこで前に存在していたひずみを減少させることができることが容易にわかる。溶質原子が粒内部から粒界、自由表面あるいは他の界面に濃化するこの移動は平衡偏析と呼ばれ、その度合いは偏析の自由エネルギーにより熱力学的に律せられる。
　固体結晶中における吸着位置を考える。ここでは自由表面と内部欠陥をもつ固体が、拡散過程が平衡に近づくような温度で、等温閉鎖系に保持されている。この場合、溶質原子の化学ポテンシャルは、系の中では一定であり、溶質原子は固溶状態、蒸気、あるいは内部欠陥に分配されて存在する。
　すでに1894年には、砒素、燐、および硫黄によって明らかに結晶粒界の破壊が起こることが注目されてはいたが、1957年にMcLeanが粒界と結晶格子の間で溶質原子が分配されることをはじめて予言した。McLeanは、英国国立物理学研究所において表面および粒界エネルギーの偏析への依存性の予想にもとづき、銅および鉄中の溶質偏析を測定する研究計画に着手した。この研究は非常な成功を収めたが、大変時間がかかり、退屈で、しかも、二元合金系に限定されていた。偏析した元素を直接的に同定するのにAESを使用できるようになって、これらの制約はなくなり、材料の問題に対してもっと適切、より低温での実験が可能となった。また、工業的破壊が起こった場合には偏析元素を同定するAESの能力は非常に重要である。粒界偏析の研究に表面分析を用いるにあたって唯一の欠点は、装置のなかで粒界にそって試料を破断する必要があることである。このことは、焼戻し脆化する材料では問題にならないが、同じ粒界破断でも、AES装置の中では模擬実験し難いような、繰り返し応力や繰り返し加熱のような特別な条件の下だけで生じるものもある。もし装置のなかで粒界破断が起こらなければ粒界偏析の測定は不可能である。
　 鋼中での粒界偏析の測定がすでになされている元素範囲を見ると、二元あるいは三元合金系においては、適当な条件の下で鉄中に固溶している他の多くの元素が偏析することはありえそうなことである。鋼の表面偏析の実験により、アルカリ金属のみならずホウ素と遷移金属がすべて観察されている。同じ結果は銅やニッケルでも見られ、工業的純度の材料においては偏析は例外ではなく、原則であるといえそうである。

◇結晶粒界の測定
（省略）もし、ある特定の温度での偏析を測定する場合は、偏析元素の移動を抑制するために、冷却速度を十分に速くしなければならない。もちろん、室温まで冷却した後は、ほとんどの試料はいつまでも保存できる。多くの系で、バルク中の拡散係数は50℃上昇するごとに10倍増加する。したがって、事前に機械加工した小さな試料を用い氷塩水中に冷却しても（金属研究者が用いる最も急速な冷却方法）、鉄中での偏析測定は900℃までの温度に限られる。このため鉄についての研究はα相に限定されている。
　このように準備された試料は、次にAES系に挿入され、エアロックが使用できないときには、ベークアウト（真空容器の焼出し）によって超高真空を達成する。200℃でのベークアウトでは、置換により拡散する溶質元素が完全に再分布される。ほとんどの試料は、つぎに液体窒素で冷却されたステージの上での衝撃により破壊される。衝撃による速いひずみ速度と低い温度で、すべての体心立方（b.c.c.）材料や一部の面心立方（f.c.c.）材料は脆くなる。次に、SEMと同じやり方で絞った電子ビームで破断表面を走査して、表面の適当な領域の像を見る。現在約1 μm(0.1 μmを達成できる高価な装置を除いて）の空間分解能を持つオージェ電子マップによって、関心のある場所を選び、その点の元素偏析の度合いを定量的に分析する。ある種の鋼は室温以上での延性破壊から、0から-40℃の範囲で粒界破壊になり、さらに劈開破壊になるので、破壊過程に注意しなければならない。このような試料はできるだけ低温であればよいというのではなく、適切な温度範囲で破断しなければならない。興味ある臨界温度範囲とは別に、試料を延性的に破断する必要のある溶接の問題に対しては、引張破断ステージを使用することができる。高温の問題に対処するために、高温引張ステージが設計されており、また、鋼の破断を助けるために水素雰囲気や試料の電極電解チャージも使用できる。後者の場合、チャージングの正確な条件は試料によって決まる。一般には、室温で、As2O3のような水素再結合ポイズンを0.03 g/l 含む0.5M H2SO4溶液中で2時間、白金の電極と試料との間に20 mA/cm2の電流密度で電流を流せば、チャージングができる。多くの場合、これらの試料は液体窒素温度で保存できるが、室温ではチャージした水素の寿命が非常に短いので、ベークアウト過程を受けさせてはならない。
　上記操作に対する一般的注意を付け加えておかなければならない。破断したあと鉄やタングステンのような金属の表面は非常に活性であるので、真空条件が特に良い場所以外は、炭素と酸素のピークが時間に比例して増加するのが見られる。これが、これらの重要な元素の正確な測定を非常に困難にしている。さらに、手指からの汚染による塩素や、空気漏れに起因する窒素が認められることもある。もし十分な分解能をもつイメージング（画像表示）機能が使える場合は、粒内劈開と粒界破壊の領域が明瞭に識別できる。汚染元素はそのすべてが装置によるものだけではなく、材料それ自体からの表面拡散に由来するものもある。多くの金属中で、アルカリ金属や、鉛やビスマスのような金属は非常に固溶度が低いので、非常に小さく純粋な析出物を形成する。破断後表面に現れた析出物の原子が、劈開および粒界領域を含む全表面に拡散し偏析の正確な測定を困難にする。この影響は、分析の間、試料を液体窒素温度に保てば取り除くことができる。表面拡散の問題は、また、加熱引張ステージ上での研究範囲を限定する。この場合、破断が緩慢で、試料が冷却されるまで1時間も高温にあるので、すべての測定値は細心の注意を払って検討する必要がある。

◇マイクロエレクトロニクスのオージェ電子分光法の応用
　シリコンを用いたデジタルならびにアナログ マイクロエレクトロニクスの分野は、高集積化、高速度化、低電力消費ならびに高効率への要請にこたえるべく、回路をより高密度にする方向に発展してきている。この小型化は、回路設計、素子ならびにその構造の変革、新材料の利用および構造の面内方向や垂直方向の寸法を減らすことにより実現されつつある。超LSI(VLIS)における小型化と並行して、トランスデューサーやあらゆるタイプのオプト エレクトニクス素子を生産するために、シリコンや化合物半導体を用いた新しい素子や構造が開発されている。
　回路の高密度化に伴い、その技術はあらゆる面での物理的限界ぎりぎりを目指すことになるので、分析装置はますます決定的な役割を果たすことになる。ひとたび非常に小さいサイズの構造をもったものが作られたとすると、（それに対応して）高い空間分解能や深さ分解能をもつ分析手法が求められることになる。走査オージェ電子顕微鏡はまさにそのような手法の一つである。オージェ電子の平均自由行程が僅かに5から20Åであることから、オージェ信号は試料表面の数原子層を分析対象とすることになる。現在市販されているオージェマイクロプローブは、入射電子を50 nm以下の直径をもつ電子にまで収束させることができるので、このような装置は優れた走査電子顕微鏡（SEM）の性能をもつと同時に元素分析を可能ならしめ、SEM観察による表面トポグラフに対応したサブミクロン領域の物質を同定し、かつ鮮明な像として与えることができる。
　オージェ電子分光法（AES）は、エネルギー分散型X線分光法（EDX）とはかなり異なったやり方でSEMに分析機能を与えている。すなわちAESは、EDXにおける深さ分解能が大体1 μm程度であるのに対し、それより数桁も薄い表面層を分析対象としている。薄い汚染層、表面組成、薄膜構造や界面薄膜の組成は、素子の動作特性にますます重大な影響をもたらすものになっている。回路素子の面内方向や垂直方向のサイズが小さくなるに伴い、表面-体積比が大きくなり、したがって表面や界面がますます重要になってきた。その結果、表面リーク電流や接触抵抗など、大きいサイズの場合には二次効果にしかすぎなかったものが、今や最新の素子の動作特性を左右するのである。したがってAESのような表面分析技術には、表面や界面の化学的性質が素子の動作特性にどのような影響を及ぼすかということを見極めることが求められる。AESはまた、軽元素に対して感度が高いということでもEDXとは異なっており、水素とヘリウムを除いた周期律表のすべての元素が独特のオージェスペクトルの形で検出され、ほぼ同じ感度でかつ容易に同定できる。
　AESの面内方向の分解能も、EDXが固体内の電子の散乱効果によってミクロンのオーダーであるのに比べて、はるかに高いものになっている。走査オージェ電子顕微鏡の面内方向の分解能は入射電子線の直径でほぼ決まり、現在では化学組成についての空間分解能は100 nmを切るに到っている。

◆ スペクトルの解釈のための知識
（省略）かくして電子の全角運動量は軌道角運動量とスピン角運動量の結合であり、実際のところ二つの運動量の単なるベクトル和で表される。しかし、これらベクトル和は、j-j結合とL-S結合（またはRusell-Saunders結合）の2種で実現される、ということに留意することがここでは最も重要である。

◇ L-S結合
　ベクトル和を求める他の方法は、まず個々の電子軌道角運動量の和を計算しておいて、次にスピン電子角運動量のベクトル和を求めることである。これら二つの全角運動量は、二つの量子数により特徴づけられる。すなわち、Σlに等しい全角運動量Lと、Σsに等しい全スピン角運動量Sの二つである。全角運動量を与える二つの運動量和の結合は量子数Jで表されるが、この場合は｜L±S｜に等しい。LとSは0, 1, 2, 3, ......なる値をとるので、Jは｜L+S｜と｜L-S｜との間の積分値となる。L-S結合という名前の由来はここにある。L-S結合はZ<20程度の低い原子番号の元素に適用できることが分かってきた。L-S結合では、終状態の電子分布を示す2S+1Lなる形式が用いられる。分光学的表記との類似から、L=0, 1, 2, 3, ......なる状態は大文字S, P, D, F, ....で表される。全スピン角運動量Sは添字(2S+1)として入る（L=0に対応する状態Sは全スピン量子数Sと取り間違えないように注意が必要である）。
（省略）
　L-S分類法と表記法は、理論との比較が目的で、高エネルギー分解能モードでオージェスペクトルを測定する場合にはよく用いられるが、日常的に使われることはない。L-S結合、j-j結合いずれも終状態を記述するには不適当な周期律表の領域で用いられる方式、すなわち混合あるいは中間結合方式についても事情は同じである。中間結合方式ではそれぞれL-S項が、異なるJの値をもつ多重項に分裂する。それでその記号は2S+1Ljなる形となる。

◇ j-j結合
　j-j結合においては、一つの孤立した電子の全角運動量は、スピン角運動量と軌道角運動量のベクトル和で示される。すなわちj=l+sなる全角運動量子数jで特定の電子の全角運動量が示される。明らかにjは1/2, 3/2, 5/2などの値をとる。原子全体の全角運動量を得るには、原子を構成する全電子に関し和をとることが必要である。これにより、J=Σjなる量子数の全角運動量が得られる。このようにしてベクトル和を得る方法がj-j結合として知られているものである。厳密にいうと、j-j結合はZ>75程度の高い原子番号をもつ元素における電子相互作用を説明するのに最も良い方法である。しかし、実際のところ、j-j結合に基づいた記述は、すべての元素のオージェおよび分光学的特徴を説明するのに用いられてきた。光電子生成により生じるスペクトルでは、終状態が1回電離したものであるため、このことは特に問題とならないのに対し、オージェ過程では、終状態が2回電離し、二つのホール間の相互作用が生じるため、j-j結合が適用できない。
　j-j結合では、記述は主量子数n, 軌道量子数l, および前述のjに基づいている。歴史的なX線の表記法（命名法、notation）ではn=1, 2, 3, 4, ......の状態はK, L, M, N, ......を用いて表され、l=0, 1, 2, 3, 4, ......とj=1/2, 3/2, 5/2, ......との種々の組み合わせで表される状態は、簡便な1, 2, 3, 4, ......の添字で表される。このX線的表記法は、ほとんど常にオージェ遷移に用いられている。（省略）
　分光学的記述は、X線的記述と全く同等であるだけでなく、種々の量子数と密接に結びついている。まず主量子数が表され、次いでl=0, 1, 2, 3, ......に対応してs, p, d, f, ......がこれに続き量子数jが添字として付与される。X線的表記法でL3と書かれる状態はn=2, l=1, j=3/2なる量子状態にあり、分光学的表記では2P3/2と表される。（省略）光電子を表すには、それが放出されたもとの原子単位の分光学的表記を用いるのが便利である。

◆ オージェ電子スペクトル
・イオン化断面積は一次電子エネルギーに依存するため、実際問題として十分なイオン化のためには一次電子エネルギーは束縛エネルギーの約5倍より大きいことが必要である。
・K殻に内殻空孔を生成した後のオージェ電子放出および特性X線放出による緩和の相対確率を見ると、2 KeV以下の比較的浅い準位では、オージェ遷移による緩和の確率が蛍光X線のそれよりも圧倒的に有利である。
　オージェ電子放出の確率 + 蛍光X線の発生確率 = 100%
　また、L, M, Nなどの電子準位についても同様である。
・オージェ電子のエネルギーが原子のエネルギー準位の関数であるということに留意すればおおむね十分である。
・周期律表のそれぞれの元素に関しては一義的に決まる一組のオージェエネルギーが存在し、同じ組のエネルギーを持つ二つの元素は存在しない。かくして、オージェ電子エネルギーの分析を行えば直ちに元素の同定ができることがわかる。オージェ過程に価電子帯の電子が一つあるいは二つ含まれる場合でも、支配的な項は常に最初にイオン化された内殻レベルの束縛エネルギーであるため、やはり同定が可能である。
　重元素に関しては、エネルギー準位の数が多いので、与えられた一次電子エネルギーでエネルギー的に可能なオージェ遷移の数はもちろん多くなる。しかし幸いなことにAESでは、その多くのなかでほんの一つ、二つの遷移の確率が大きいため重い元素でも扱いやすいものとなる。
・イオン化断面積は、一次電子エネルギーがイオン化エネルギーの約3倍のあたりで最大となることが理論式からも示されるが、オージェ分析においては背面散乱電子の寄与が効いてくるので、一次電子のエネルギーがイオン化エネルギーの約4-5倍のあたりで、オージェ信号強度が最大となる。

◇オージェエネルギー
　原子番号Zの原子におけるABCなるオージェ遷移のエネルギーEABCを求める近似は、ChungとJenkinsにより経験的に求められている。そしてこの近似は実用上、十分正確である。それは、
EABC(Z) = EA(Z) - (1/2)*[EB(Z) + EB(Z+1)] - (1/2)*[EC(Z) - EC(Z+1)]
　ここで、Ei(Z)は原子番号Zなる元素のi番目の準位の束縛エネルギーであり、Ei(Z+1)は周期律表で一つ原子番号の大きな元素の同じ準位の束縛エネルギーである。

◇オージェ信号強度
　一つのスペクトルを詳細にみたり、二つの近接元素のスペクトルを比較すると、同元素の異なるオージェ遷移あるいは異元素の同じオージェ遷移の相対強度がかなり異なることがただちにわかる。これらの相違の理由は一般的によく知られている。しかし、オージェエネルギーの計算の場合と異なり、オージェ強度（あるいは遷移確率）の計算の現状はかなり困難である。すなわち相対強度を求めるうえで使える簡単な半経験的な公式は一つもない。理論家が用いなければならないような複雑な公式でさえ、これまで実際に強度を正確に予測することには成功していない。固体では特にそうである。

◇情報の得られる深さ
　情報の得られる深さは固体中での電子の非弾性平均自由行程（IMFP, λ）によって支配される。（省略）観測される光電子の信号の95%が3*λ*sin(α)の厚さの層から由来する（平面の表面の場合）。ここでαは表面に対する電子の脱出角度である。α=90-Θである。Θは表面の法線からの角度である。
